## 一、六大拓扑类型总览

### 1.1 拓扑分类体系

```
微网拓扑
├── 并网型微网
│   ├── Type A: 交流微网 (AC Microgrid)
│   │   └── 所有DER通过AC母线耦合，传统拓扑
│   ├── Type B: 直流微网 (DC Microgrid)
│   │   └── 所有DER通过DC母线耦合，适合直流负荷为主场景
│   ├── Type C: 交直流混合微网 (AC/DC Hybrid)
│   │   └── AC子网+DC子网通过双向变流器(ILC)耦合
│   └── Type D: 光储充一体化 (PV-BESS-EV Integrated)
│       └── 光伏→储能→充电桩全直流链路+交流并网
│
├── 离网型微网
│   ├── Type E: 柴储混合 (Diesel-BESS Hybrid)
│   │   └── 柴油发电机+储能作为主电源
│   └── Type F: 纯离网/孤岛 (Off-Grid/Island)
│       └── 完全脱离电网运行，需黑启动能力
│
└── 特殊型
    └── Type G: 级联型多微网 (Multi-Microgrid)
        └── 多个微网通过柔性互联或常规开关连接
```

### 1.2 选型决策流程

```
输入项目基本参数
  │
  ├── 步骤1: 电网可用性判断
  │   ├── 有稳定电网 → 并网型(优先)
  │   └── 无电网/极不稳定 → 离网型(E/F)
  │
  ├── 步骤2: 负荷类型分析
  │   ├── 直流负荷>60% → 直流微网(B)或交直流混合(C)
  │   ├── 交流负荷>80% → 交流微网(A)
  │   └── 混合负荷 → 交直流混合(C)
  │
  ├── 步骤3: 光伏/储能容量匹配
  │   ├── 光储比例>2:1 → 优先直流(DC耦合)
  │   ├── 光储比例 1:2到2:1 → 灵活选择
  │   └── 储→充(有充电桩) → 光储充一体化(D)
  │
  ├── 步骤4: 供电可靠性要求
  │   ├── 关键负荷>30%且需不间断供电 → 需黑启动+柴发备援
  │   └── 一般负荷 → 标准拓扑即可
  │
  └── 步骤5: 经济性评估
      ├── 初投资约束紧 → 交流微网(设备成熟便宜)
      └── 全生命周期最优 → 交直流混合(效率高)
```

| 技术适用性 | 0.25 | 负荷匹配度、光伏消纳能力、电压等级适配 |
| 初投资(CAPEX) | 0.15 | 设备采购+安装工程费 |
| 运行效率 | 0.15 | 全链路转换效率(含待机损耗) |

### 3.3 决策树（详细分支）

```
输入: 接入电网？ ──否──→ 有柴发？ ──是──→ 【Type E: 柴储混合】
    │                      │
    │                      └──否──→ 纯储能离网？ ──是──→ 【Type F: 孤岛/黑启动】
    │
    ├── 是(并网型)
    │   │
    │   ├── 有充电桩？(≥5台, >300kW)
    │   │   ├── 是 ──→ 光伏+储能+充电桩？ ──是──→ 【Type D: 光储充一体化】
    │   │   └── 否 ──→ 继续判断
    │   │
    │   ├── 直流负荷占比 > 60%？
    │   │   ├── 是 ──→ 交流负荷<10%？ ──是──→ 【Type B: 直流微网】
    │   │   │         └──否──→ 【Type C: 交直流混合】
    │   │   └── 否 ──→ 继续判断
    │   │
    │   ├── 光伏DC/AC比 > 2.0？
    │   │   ├── 是 ──→ 建议DC侧直接耦合PCS ──→ 偏Type C或B
    │   │   └── 否 ──→ 继续判断
    │   │
    │   ├── 要求孤岛运行 + 黑启动？
    │   │   ├── 是 ──→ 需配置构网型PCS(GFM) ──→ 偏Type C(含GFM)
    │   │   └── 否 ──→ 跟网型PCS(GFL)即可
    │   │
    │   ├── 扩容预期高？(年增长>15%)
    │   │   ├── 是 ──→ 【Type C: 交直流混合】(扩展最灵活)
    │   │   └── 否 ──→ 继续判断
    │   │
    │   └── 预算极度紧张？
    │       ├── 是 ──→ 【Type A: 交流微网】(设备最成熟便宜)
    │       └── 否 ──→ 按综合评分最高选择 → 默认【Type C】
    │
    └── 特殊情况
        ├── 军用/海岛/极端偏远 → 考虑Type E或F
        ├── 多园区互联 → Type C级联 + 柔性互联
        └── 参与电力交易 → 需Type C(响应调度指令)
```

### 3.4 强制排除规则（一票否决）

| 条件 | 强制排除拓扑 | 原因 |
|------|------------|------|
| 无电网 > 6个月/年 | 排除A/B/C/D | 并网型无电网不可用 |
| 噪声要求 < 60dB且环境敏感 | 排除E | 柴油发电机噪声不达标 |
| 无柴油/燃气供应保障 | 排除E | 燃料不可得 |
| 海拔 > 3000m | E中柴发需降容 ≥ 30% | 高海拔需特殊选型 |
| 直流负荷 < 5% | 排除B(直流微网) | AC/DC转换不经济 |
| 预算 < 500万元 且 > 2MW | 排除C(先做A,后升级) | AC最经济 |
| 无LVRT/HVRT要求 | F可不考虑构网型 | 但功能受限 |
| 充电桩 < 3台且无光伏 | 排除D | 无光储充协同价值 |

## 四、交流微网拓扑详解

### 4.1 标准拓扑结构

```
                    ┌────────────────────────────┐
                    │     10kV/0.4kV 变压器      │
                    │     Dyn11, Uk%=6           │
                    └──────────┬─────────────────┘
                               │ 0.4kV AC Bus
              ┌────────────────┼────────────────┬──────────────┐
              │                │                │              │
         ┌────┴────┐     ┌────┴────┐     ┌────┴────┐    ┌────┴────┐
         │ 本地交流 │     │  PCS    │     │ 无功补偿 │    │  柴发   │
         │  负荷   │     │ 双向变流│     │  (SVG)  │    │ (备用)  │
         │ 400V    │     │ AC⇄DC  │     │  ±xkvar │    │  400V   │
         └─────────┘     └───┬──┬──┘     └─────────┘    └─────────┘
                             │  │
                          ┌──┘  └──┐
                     ┌────┴────┐  ┌┴────────┐
                     │ 光伏逆变 │  │  BESS   │
                     │ DC→AC   │  │ 电池舱  │
                     │ xxkWp   │  │ xxkWh   │
                     └─────────┘  └─────────┘

通信与控制:
  EMS ──Modbus TCP──→ 全部PCS/逆变器/电表
  EMS ──MQTT────────→ 云平台(可选)
  BMS ──CAN─────────→ PCS (充放电控制)
```

### 4.2 交流微网核心特性

| 特性 | 参数/描述 |
|------|----------|
| 耦合方式 | 共交流母线，所有设备并接到交流母线 |
| 光伏接入 | DC→AC通过组串/集中式逆变器并网 |
| 储能接入 | DC电池→PCS(AC/DC)→交流母线 |
| 变压器配置 | 10kV/0.4kV或10kV/0.69kV |
| 无功补偿 | 需单独配置SVG或电容柜 |
| 谐波治理 | 需配置APF或有源滤波 |
| 构网能力 | GFL(跟网型)为主，可配置GFM(构网型) |
| 标准化程度 | ★★★★★ (设备最成熟) |

### 4.3 交流微网优势与劣势

**优势**：
- 设备采购渠道最广，市场竞争充分，价格低
- 运维人员技术门槛最低，培训成本低
- 标准最成熟(GB/IEC/IEEE均有完整标准)
- 变压器保护/继保配置成熟
- 扩容简单：新增馈线柜即可
- 与既有配电系统兼容性最佳

**劣势**：
- 多级AC/DC转换损耗大(光伏DC→AC, 储能需DC→AC→DC→AC)
- 直流负荷需独立AC/DC变换器(如充电桩)
- 无功/谐波问题需额外治理
- 频率/电压耦合控制复杂(交流系统有功-频率、无功-电压)
- 同步要求高(所有逆变器需锁相环同步电网)

### 4.4 典型设备清单

| 设备 | 规格示例 | 数量 | 备注 |
|------|---------|------|------|
| 升压变压器 | SCB13-1250/10, Dyn11 | 1 | 含温控 |
| 10kV开关柜 | KYN28-12 | 1面 | 含进线+计量+出线 |
| 低压开关柜 | GCS | 3面 | 含进线+馈线+电容补偿 |
| PCS | 630kW, 400V, 三电平 | N台 | 含隔离变(如需) |
| 光伏逆变器 | 100kW, 组串式 | N台 | MPPT路数 |
| SVG | ±200kvar, 400V | 1台 | 无功补偿 |
| APF | 100A, 400V | 1台 | 有源滤波 |
| BESS电池舱 | 5.016MWh, LFP液冷 | N台 | 含高压箱 |
| 交流汇流柜 | 4进1出, 630A | 1台 | 含CT |
| EMS工控机 | 边缘计算，Modbus+MQTT | 1台 | 含HMI触屏 |

### 4.5 应用场景

| 场景 | 适合度 | 说明 |
|------|-------|------|
| 工业园区 | ★★★★★ | 传统配电，交流负荷为主 |
| 商业楼宇 | ★★★★☆ | 暖通/照明以交流为主 |
| 数据中心 | ★★★☆☆ | 有大量直流负荷(UPS)，不如直流微网 |
| 充电站(纯充电) | ★★☆☆☆ | 直流负荷为主，AC多级变换不经济 |
| 光储充一体化 | ★★★☆☆ | 可做，但效率不如直流耦合 |

## 五、直流微网拓扑详解

### 5.1 标准拓扑结构

```
                    ┌────────────────────────────┐
                    │     10kV/0.4kV 变压器      │
                    └──────────┬─────────────────┘
                               │
                    ┌──────────┴─────────────────┐
                    │  AC/DC双向变流器(前台PCS)   │
                    │  GFM模式, 建立直流母线      │
                    │  额定功率 xxxkW             │
                    └──────────┬─────────────────┘
                               │ DC Bus (±375V 或 ±750V)
         ┌─────────────────────┼──────────────────────┐
         │                     │                      │
    ┌────┴────┐          ┌────┴────┐           ┌─────┴─────┐
    │ 光伏    │          │  BESS   │           │  直流负荷  │
    │ DC/DC   │          │ DC/DC   │           │  (充电桩/  │
    │ 优化器  │          │ 变换器  │           │   LED/     │
    │ MPPT    │          │ 充放电  │           │   变频器)  │
    └─────────┘          └─────────┘           └───────────┘
         │                     │
    ┌────┴────┐          ┌────┴────┐
    │ 光伏阵列│          │  电池模组│
    │ xxkWp  │          │  xxkWh  │
    └─────────┘          └─────────┘

    ┌─────────────────────────────────┐
    │  交流负荷(少)                    │
    │  DC/AC逆变器(小功率)             │
    │  从DC母线取电                   │
    └─────────────────────────────────┘
```

### 5.2 直流微网核心特性

| 特性 | 参数/描述 |
|------|----------|
| 耦合方式 | 共直流母线，DC/DC变换器并接 |
| 直流母线电压 | 常见 ±375V(750V) 或 ±190V(380V) |
| 光伏接入 | DC/DC MPPT优化器 → 直流母线 |
| 储能接入 | DC/DC充放电变换器 → 直流母线 |
| 前台PCS功能 | 构网型(GFM), 建立稳定直流母线电压 |
| 无功问题 | **不存在**，直流系统无无功概念 |
| 频率问题 | **不存在**，直流系统无频率概念 |

### 5.3 直流微网优势与劣势

**优势**：
- 光伏→储能→直流负荷全链路DC，仅一次AC/DC转换(并网点)
- 系统效率高(比交流高3~8%)
- 无无功补偿设备，无谐波治理设备
- 无频率/功角稳定问题
- 光伏和储能DC/DC变换器成本低于DC/AC逆变器
- 控制算法简单(仅需维持直流母线电压)
- 适合光伏直驱负荷

**劣势**：
- 直流设备门槛高(DC断路器、DC接触器、DC/DC)
- 直流保护技术相对不成熟(无过零点灭弧)
- 交流负荷需额外DC/AC逆变器
- 直流母线电压标准化程度低(±375V/±190V/750V/1500V)
- 运维人员技术要求高
- 直流侧接地/绝缘监测要求更严格
- 直流开关设备供应商少

### 5.4 直流母线电压等级选择

| 电压等级 | 适用功率范围 | 典型场景 | 绝缘要求 |
|---------|------------|---------|---------|
| DC 48V | <5kW | 户用/通信基站 | 安全电压 |
| DC 220V | 5-30kW | 小型商业 | 基础绝缘 |
| DC ±190V (380V) | 30-200kW | 中型商业/充电站 | 加强绝缘 |
| DC ±375V (750V) | 100-500kW | 工业园区 | 加强绝缘 |
| DC 1000V | 200-1000kW | 大工业 | 高绝缘 |
| DC 1500V | >1MW | 大储/集中式 | 高绝缘 |

### 5.5 应用场景

| 场景 | 适合度 | 说明 |
|------|-------|------|
| 数据中心 | ★★★★★ | 服务器供电原就是直流，天然匹配 |
| 充电站 | ★★★★★ | 充电桩内部就是DC/DC,免去AC/DC级 |
| LED照明系统 | ★★★★☆ | 直流LED驱动效率更高 |
| 通信基站 | ★★★★☆ | 48V直流系统成熟 |
| 工业园区(交流多) | ★★☆☆☆ | 大量交流负荷不经济 |
| 居民小区 | ★★☆☆☆ | 家电全交流,直流无优势 |

## 六、交直流混合微网拓扑详解

### 6.1 标准拓扑结构

```
                         ┌──────────────┐
                         │  10kV电网    │
                         └──────┬───────┘
                                │
                         ┌──────┴───────┐
                         │  变压器      │
                         │  10/0.4kV   │
                         └──────┬───────┘
                                │
                    ┌───────────┴───────────┐
                    │  0.4kV AC Bus (母线段1)│
                    ├──────┬────────┬───────┤
                    │      │        │       │
               ┌────┴─┐ ┌──┴──┐ ┌──┴──┐  ┌─┴─────┐
               │交流  │ │柴发 │ │交流 │  │ ILC   │
               │负荷  │ │备用 │ │光伏 │  │ 双向  │
               └──────┘ └─────┘ └─────┘  │变流器 │
                                         │AC⇄DC  │
                                         └───┬───┘
                                             │
                    ┌────────────────────────┴───────┐
                    │   ±375V DC Bus (直流母线段)    │
                    ├─────────┬──────────┬───────────┤
                    │         │          │           │
               ┌────┴────┐ ┌──┴───┐ ┌───┴────┐ ┌───┴────┐
               │ 光伏    │ │ BESS │ │ 直流   │ │其他DC  │
               │ DC/DC   │ │ DC/DC│ │ 充电桩 │ │ 负荷   │
               └─────────┘ └──────┘ └────────┘ └────────┘

ILC (InterLinking Converter): 连接AC和DC子网的桥梁
  - 正常运行: 控制直流母线电压，功率双向流动
  - 离网模式: 做构网型逆变器，给交流侧关键负荷供电
```

### 6.2 ILC（互联变流器）核心参数

| 参数项 | 典型配置 | 说明 |
|-------|---------|------|
| 额定功率 | 等于DC子网总功率 | 必须能承载直流侧全部功率 |
| AC侧电压 | 400V ±10% | 三相四线 |
| DC侧电压 | ±375V (750V bipolar) | 三电平NPC |
| 工作模式 | 整流(AC→DC) + 逆变(DC→AC) | 四象限运行 |
| 电流THD | <3% | 并网电流谐波 |
| 效率 | >98.5% | 双向满载效率 |
| 控制策略 | Vdc-Q, P-Q | 直流侧电压稳定优先 |
| 构网能力 | Grid-Forming支持 | 离网时建立AC电压 |
| 切换时间 | <20ms | 并网↔离网模式切换 |
| 通信接口 | Modbus TCP + CAN | 与EMS/BMS交互 |

### 6.3 AC侧与DC侧的功率分配策略

```
正常运行(并网模式)：
  ILC负责维持DC母线电压稳定，等效于Vdc-Q控制
  AC侧PCS(如有)可做交流侧储能控制
  DC侧光伏优先供直流负荷 → 多余充BESS → 进一步多余经ILC→AC侧
  AC侧光伏优先供交流负荷 → 多余经ILC→DC侧充BESS

离网运行(孤岛模式)：
  ILC切换为GFM模式，从DC侧取电供给AC关键负荷
  DC侧BESS作为主电源 → ILC → AC关键负荷
  DC侧光伏 → 给DC负荷 + 充BESS
  柴发(如有) → 直接到AC侧给非关键负荷
```

### 6.4 交直流混合微网优势与劣势

**优势**：
- 同时服务AC和DC负荷，适用性最广
- 直流侧效率高(光伏→储能→负荷全DC)
- 直流侧无无功/谐波问题
- ILC可作为黑启动电源
- 扩展最灵活：新增DC设备接DC母线，新增AC设备接AC母线
- 整体效率比纯交流高5~8%
- 可实现多层级能量管理

**劣势**：
- 系统最复杂，控制策略难度最高
- ILC是关键设备，单点故障影响大(需N+1冗余)
- 初投资相对较高(ILC+DC开关设备)
- 运维需同时掌握AC和DC技术
- 保护配合复杂(AC侧和DC侧保护需协调)
- 标准化程度低(无成熟标准)

### 6.5 适用场景

| 场景 | 适合度 | 说明 |
|------|-------|------|
| 大型工业园区 | ★★★★★ | AC+DC负荷混合 |
| 光储充一体化 | ★★★★★ | PV/BESS在DC侧,充电在DC,其他AC |
| 商业综合体 | ★★★★☆ | 冷暖用AC,LED+充电用DC |
| 港口/机场 | ★★★★☆ | 大量变频驱动+LED+充电 |
| 纯交流负荷园 | ★★★☆☆ | AC即可,无需DC |
| 小型项目<200kW | ★★☆☆☆ | ILC成本占比太高 |

## 七、光储充一体化拓扑详解

### 7.1 标准拓扑结构

```
                         ┌──────────────┐
                         │  10kV/0.4kV  │
                         │  变压器      │
                         │  10/0.4kV    │
                         └──────┬───────┘
                                │
                    ┌───────────┴───────────┐
                    │     AC母线            │
                    ├───────────────────────┤
                    │                       │
               ┌────┴────┐            ┌────┴────┐
               │ 交流负荷 │            │  PCS    │
               │ (办公/  │            │ 双向    │
               │  暖通)  │            │ AC⇄DC  │
               └─────────┘            └───┬────┘
                                          │
                    ┌─────────────────────┴──────────┐
                    │    DC Bus (750V/1000V)          │
                    ├─────┬────────┬────────┬─────────┤
                    │     │        │        │         │
               ┌────┴─┐ ┌─┴────┐ ┌┴──────┐┌┴──────────┐
               │ 光伏 │ │ BESS │ │充电堆 ││充电终端×N  │
               │DC/DC │ │DC/DC │ │(群充) ││CCS2/GB/T   │
               │xxkWp │ │xxkWh │ │xxkW   ││60-360kW    │
               └──────┘ └──────┘ └───────┘└────────────┘

能量流动优先级：
  1. 光伏 → 充电终端 (直驱，效率最高)
  2. 光伏 → BESS (充储能，消纳余电)
  3. BESS → 充电终端 (放电，补充光伏不足)
  4. 电网 ↔ PCS (辅助，平衡供需)
  5. BESS → 电网 (峰谷套利，如需)
```

### 7.2 光储充一体化的三种子拓扑

| 子拓扑 | 光伏接入 | 储能接入 | 充电接入 | 效率 | 复杂度 |
|-------|---------|---------|---------|------|-------|
| AC耦合 | 逆变器→AC | PCS→AC | AC输入充电桩 | 低(多级) | 低 |
| DC耦合(集中) | DC/DC→DC Bus | DC/DC→DC Bus | DC Bus→分功率模块 | 高 | 中 |
| DC耦合(分布式) | DC/DC→直流子母线 | DC/DC→直流子母线 | 每桩独立DC/DC | 最高 | 高 |

**推荐**：工商业场景推荐DC耦合集中式，效率最高、成本可控。

### 7.3 充电功率分配算法框架

```
输入参数：
  - P_pv: 当前光伏功率(kW)
  - P_load_ac: 交流负荷功率(kW)
  - SOC: 储能荷电状态(%)
  - P_ev_request: 充电桩请求总功率(kW)
  - P_grid_limit: 电网限功率(kW) [防反送/需量]
  - Time_slot: 当前时段(峰/平/谷)

输出：
  - P_pcs: PCS功率指令(正=从DC→AC, 负=AC→DC)
  - P_ev_output: 实际分配给充电桩的功率
  - P_bess: 储能充放电功率(正=充电, 负=放电)

策略决策表：
┌─────────┬──────┬───────────┬─────────────────────────────┐
│ 时段    │ SOC  │ P_pv vs   │ 策略                        │
│         │      │ P_ev      │                             │
├─────────┼──────┼───────────┼─────────────────────────────┤
│ 峰电    │ >50% │ P_pv>P_ev │ 余电+BESS放电→反送电网(套利)│
│ 峰电    │ >50% │ P_pv<P_ev │ BESS放电补充充电缺口         │
│ 峰电    │ <20% │ 任意      │ 仅光伏直驱充电,不放电        │
│ 平电    │ 任意 │ P_pv>P_ev │ 余电充BESS,超出再反送        │
│ 平电    │ 任意 │ P_pv<P_ev │ 电网补充,不调用BESS          │
│ 谷电    │ <90% │ 任意      │ 电网充BESS + 光伏直驱充电    │
│ 谷电    │ >90% │ P_pv<P_ev │ 电网补足充电,不再充BESS      │
└─────────┴──────┴───────────┴─────────────────────────────┘
```

### 7.4 光储充一体化优势与劣势

**优势**：
- 光伏→储能→充电桩全直流链路，系统效率最高(可达95%+)
- 光伏消纳率最高(储能平滑+充电桩作为灵活性负荷)
- 对充电用户来说电费最低(光伏+谷电成本)
- 降低电网需量(储能削峰+充电功率可调)
- 多收益来源：充电服务费+峰谷套利+光伏绿电

**劣势**：
- 专用设备(光储充一体机)供应商少
- 直流充电桩与交流充电桩接口不同(需明确选择)
- 充电桩负荷波动大，对储能控制要求高
- 需预测充电需求(不如负荷稳定)
- 商业模式依赖电价政策

### 7.5 关键参数匹配

| 参数关系 | 推荐比例 | 说明 |
|---------|---------|------|
| PV:BESS | 1:1 ~ 2:1 (kWp:kWh) | 光伏充电时段消纳储能 |
| BESS:充电总功率 | 1:1 ~ 1.5:1 | 储能须足够支持充电高峰 |
| PV:充电峰值功率 | 0.5:1 ~ 1:1 | 光伏不够的要靠储能+电网 |
| 变压器容量:充电总功率 | >1.2:1 | 考虑交流负荷同时率 |

### 8.3 柴发启停控制策略

```yaml
diesel_control:
  start_conditions:
    - bess_soc_below: 20          # SOC低于20%启动柴发
    - load_exceeds: "bess_power * 1.2" # 负荷超储能功率120%
    - bess_fault: true            # 储能故障立即启动
    - manual_start: true          # 人工启动

  stop_conditions:
    - bess_soc_above: 80          # SOC恢复到80%
    - load_below: "bess_power * 0.6" # 负荷低于储能功率60%
    - pv_generation_exceeds_load: true  # 光伏足够供负荷
    - min_run_time_minutes: 30    # 最短运行时间(防频繁启停)

  synchronization:
    method: "droop_control"       # 柴发与PCS下垂控制并联
    frequency_droop: 0.03         # 频率-有功下垂系数(Hz/pu)
    voltage_droop: 0.04           # 电压-无功下垂系数(pu/pu)

  soft_start:
    ramp_rate_kw_per_s: 10        # 柴发加载速率(kW/s)
    sync_delay_s: 60              # 并机后热身延时(s)

  priority:
    - solar_pv                    # 光伏优先
    - battery_discharge           # 储能次之
    - diesel_generator            # 柴发最后(降油耗)
```

### 8.4 柴储混合优势与劣势

**优势**：
- 无电网条件下最可靠的供电方案
- 储能大幅降低柴发运行时间和油耗(40-70%)
- 储能提供黑启动能力
- 柴发与储能互补(储能响应快+柴发持续供电)
- 适应恶劣环境(高低温/高海拔/沙尘)
- 技术成熟，设备可获取性强

**劣势**：
- 柴发有噪声/排放(环保问题)
- 燃料运输和储存成本(偏远地区)
- 柴发维护成本高(定期保养/大修)
- 全生命周期度电成本高(>1.5元/kWh)
- 绿色低碳评级低
- 不适合环保严格区域

### 8.5 适用场景

| 场景 | 适合度 | 说明 |
|------|-------|------|
| 海岛 | ★★★★★ | 无电网，柴发+光伏+储能 |
| 矿山/油田 | ★★★★★ | 偏远地区，电网不可达 |
| 军用营地 | ★★★★★ | 可靠性第一，移动式部署 |
| 偏远村落 | ★★★★☆ | 大电网未覆盖 |
| 临时施工 | ★★★★☆ | 短期使用，移动式 |
| 有电网区域 | ★★☆☆☆ | 优先用纯储能方案 |

## 九、孤岛/黑启动拓扑详解

### 9.1 黑启动定义与要求

```
黑启动(Black Start): 在外部电网完全失电(全黑)的条件下，
依靠系统内部电源自启动，逐步恢复微网供电的能力。

核心要求：
  ├── 1. 必须有内置独立电源(储能/柴发)，不依赖电网
  ├── 2. 必须有构网型电源(GFM PCS或柴发)，能建立电压频率
  ├── 3. 必须有自动控制序列，无需人工干预
  ├── 4. 负荷必须分级投入(避免冲击)
  └── 5. 关键负荷恢复时间 ≤ 规定值(通常 ≤ 5min)
```

### 9.2 构网型(GFM)与跟网型(GFL)比较

| 特性 | GFL (Grid-Following) | GFM (Grid-Forming) |
|------|---------------------|-------------------|
| 电压源 | 外部电网 | 自身建立 |
| 同步方式 | PLL锁相环锁电网 | 自主惯性响应 |
| 孤岛运行 | 需要外部参考 | **可独立运行** |
| 黑启动 | **不具备** | **具备** |
| 惯量支持 | 无惯量 | 虚拟同步机惯量 |
| 短路容量贡献 | 1.2-1.5 pu | 3-5 pu (与同步机相当) |
| 控制器 | 电流控制(CSI) | 电压控制(VSI) |
| 市场成熟度 | ★★★★★ | ★★★☆☆ |
| 成本 | 基准 | +20-30% |
| 多机并联 | 需同步信号 | 下垂控制即可并联 |

**注意**: GFM PCS是构网型微网(离网/孤岛/黑启动)的必选设备。

### 9.3 黑启动执行序列

```
Phase 0: 电网全黑确认
  ├── 检测：PCC电压=0, 频率异常, 持续时间>10s
  ├── 确认：判断为电网全黑而非瞬时故障
  └── 动作：断开PCC断路器(QF-PCC分闸)

Phase 1: 自检与初始化 (T+0 ~ T+30s)
  ├── BESS自检：SOC>20%? 无故障? 绝缘正常?
  ├── PCS自检：通信正常? 准备就绪?
  └── 柴发自检(如有)：油量充足? 预热完成?

Phase 2: 储能独立启动建立母线 (T+30 ~ T+60s)
  ├── BMS闭合直流侧断路器
  ├── PCS启动GFM模式，建立AC母线电压(400V/50Hz)
  ├── PCS以电压源模式运行(V-f控制)
  └── AC母线电压建立完成确认

Phase 3: 关键负荷分级投入 (T+60 ~ T+180s)
  ├── 第1级：BMS辅助电源→控制电源→通信设备 (P<10kW)
  ├── 第2级：冷却系统(液冷机组)→消防系统 (P<50kW)
  ├── 第3级：关键生产/照明/安防 (P<额定功率50%)
  └── 第4级：其他重要负荷 (逐步加载)

Phase 4: 光伏/柴发投入 (T+180 ~ T+300s)
  ├── 光伏逆变器检测到电网恢复 → 重新并网
  ├── 柴发启动(如储能SOC不足) → 同步并机
  └── 系统转入孤岛稳定运行

Phase 5: 电网恢复后重新并网 (手动触发)
  ├── 检测电网电压/频率恢复正常
  ├── 电网同步(Synchro-check): ΔV<5%, Δf<0.1Hz, Δφ<5°
  ├── 闭合PCC断路器
  └── 转入并网运行模式
```

### 9.4 孤岛模式下的频率/电压控制

```yaml
island_mode_control:
  vf_control:                      # 构网型V-f控制(主从式)
    master: "GFM_PCS_1"            # 主构网单元
    slaves: ["GFM_PCS_2", "GFL_PCS_1", "PV_inverter_1"]
    voltage_setpoint: 400V
    frequency_setpoint: 50Hz

  droop_control:                   # 多机下垂控制(对等式)
    freq_droop_kw_per_hz: 100      # 频率-有功下垂系数
    volt_droop_kvar_per_v: 20      # 电压-无功下垂系数
    # P = P0 - Kf * (f - f0)
    # Q = Q0 - Kv * (V - V0)

  load_shedding:                   # 低频减载
    - stage_1: { freq: 49.0, delay: 0.5, shed_ratio: 0.15 }
    - stage_2: { freq: 48.5, delay: 0.3, shed_ratio: 0.20 }
    - stage_3: { freq: 48.0, delay: 0.2, shed_ratio: 0.25 }

  recovery:                        # 恢复后自动加载
    freq_restore: 49.8             # 频率恢复到49.8Hz
    delay_minutes: 5               # 稳定5分钟后逐步恢复负荷
```

### 9.5 孤岛/黑启动拓扑优势与劣势

**优势**：
- 完全脱离电网运行，最高自主性
- 黑启动能力确保关键负荷供电
- 适合无电网或电网极度不可靠场景
- 可与柴发/光伏组合使用

**劣势**：
- GFM PCS成本高(比GFL贵20-30%)
- 系统设计复杂，需多种运行模式切换
- 能量来源有限(储能容量决定孤岛时长)
- 孤岛时频率/电压稳定性差于并网
- 需要更完善的保护配置(防孤岛、低频减载等)
- GFM多机并联稳定性需详细仿真验证

|------|---------|---------|------|---------|

### 12.4 ILC(互联变流器)在模式切换中的角色

```
正常运行(并网):
  - ILC作为前端变流器, 维持DC母线电压
  - AC侧功率盈余/缺额→ILC自动调节
  - 交流侧保护装置独立运行

孤岛检测触发:
  - ILC检测PCC异常 → 发信号给EMS
  - EMS判断真孤岛 → 执行切换

模式切换瞬间:
  - 断开PCC断路器(如有,防孤岛保护动作)
  - ILC从Vdc-Q控制立即切换为V-f控制(GFM模式)
  - 切换时间 ≤ 20ms
  - AC侧电压频率由ILC接管

孤岛运行:
  - ILC作为主构网单元, 维持AC母线电压/频率
  - DC侧BESS提供能量来源
  - 如有多台ILC: 主从模式或下垂控制均分功率

重新并网:
  - ILC检测电网恢复 → 通知EMS
  - EMS决定何时并网(人工/自动)
  - ILC执行同步操作(调节自身输出匹配电网)
  - 同步完成后闭合PCC断路器
  - ILC切换回Vdc-Q控制(跟网模式)
```

## 十四、特殊场景决策指南

### 14.1 高海拔场景(>2000m)

```
海拔影响:
  - 空气密度降低 → 散热效率下降
  - 空气绝缘强度降低 → 电气间隙需增大
  - 柴油发电机降容 → 每1000m约降10-12%

推荐修正:
  - 柴发功率 × (1 + (altitude-1000)/1000 × 0.12) 向上取
  - PCS需降容使用或选高海拔型号
  - 开关柜电气间隙增加20-30%
  - 优先推荐Type A或Type D(无柴发)
```

### 14.2 沿海/高湿/盐雾场景

```
环境影响:
  - 金属腐蚀加速
  - 绝缘表面爬电(污秽等级提高)
  - 电子元件可靠性下降

推荐措施:
  - 设备选C5-M防腐等级
  - 变压器选用干式(SCB)而非油变
  - 所有连接器镀镍/镀金
  - 集装箱/预制舱做防腐涂层(C4/C5)
  - 优先推荐集装箱式(密封性好)
```

### 14.3 数据中心/通信基站

```
特殊要求:
  - Tier III/Tier IV级别 → N+1或2N冗余
  - 必须UPS不间断供电
  - 储能替代铅酸电池 → LFP直流微网Type B最合适
  - 黑启动必须 < 60s(关键IT负荷)

推荐拓扑: Type B (直流微网) 或 Type C (交直流混合)
  - DC 48V/240V给服务器/UPS
  - AC 380V给制冷/照明
```

### 14.4 多园区互联微网群

```
推荐拓扑: Type C 级联扩展

  Microgrid 1 (光伏为主)  ←→  Microgrid 2 (储能为主)
       │ ILC-1                    │ ILC-2
  DC Bus 1                    DC Bus 2
       │                          │
  光伏+BESS                   BESS+充电桩
       │                          │
       └───── 柔性互联装置 ────────┘
           (Power Electronics based)

特点:
  - 各子网可独立运行(孤岛)
  - 互联时可功率互济
  - 柔性互联比常规开关互联更安全(无冲击电流)
```

## 十五、常见选型错误

### 15.1 错误清单

| 错误 | 后果 | 纠正 |
|------|------|------|
| 有大量直流负荷选交流微网 | 多级AC/DC转换, 效率低10%+ | 直流负荷>60%应选B或C |
| 无构网型PCS却要求黑启动 | 无法黑启动, 系统要求无法实现 | 必须配置GFM PCS |
| ILC功率选型不足 | DC侧满功率时ILC过热/限功率 | ILC额定功率≥DC母线总功率 |
| 光储充项目用AC耦合 | 电能转换路径长(PV-DC→AC→DC→充电) | 光储充优先DC耦合 |
| 柴储项目储能容量太小 | 柴发无法停机, 油耗不减 | BESS容量≥日平均负荷×4h |
| 高海拔未留降容裕量 | 设备过载跳闸/烧毁 | 参照14.1修正 |
| 未考虑未来扩容 | 后期改造成本高 | Type C扩展性最佳 |
| 预算过低选交直流混合 | 项目超支, 导致停建 | 先做Type A, 预留DC母线位置 |
| 并网切换未做仿真验证 | 模式切换时系统崩溃 | 必须做HIL/PHIL仿真 |
| 忽视通信可靠性 | 模式切换依赖通信→失败 | FSS/ESD回路必须为硬接线 |

### 15.2 选型决策快速自检清单 (20项)

```
□ 1.  电网是否可用? 可用率多少? (基本输入)
□ 2.  是否有不间断供电的需求? (关键负荷比例)
□ 3.  负荷类型: AC占多少? DC占多少? (核心判断依据)
□ 4.  有无充电桩? 总功率和台数? (光照Type D)
□ 5.  光伏装机容量 > 储能容量×2? (建议DC耦合)
□ 6.  是否要求黑启动? (必须配置GFM PCS)
□ 7.  是否要求孤岛运行? (需GFM + PCC切换)
□ 8.  直流母线电压是否与光伏/BESS匹配? (电压选择)
□ 9.  ILC容量是否充足? (≥DC母线总功率)
□ 10. 柴发是否需要? (离网场景必备)
□ 11. 初投资预算与所选拓扑是否匹配?
□ 12. 运维团队技术水平与所选拓扑复杂度匹配?
□ 13. 是否有未来扩容规划? 扩容方式?
□ 14. 项目海拔高度是否需要设备降容?
□ 15. 环境条件(盐雾/高湿/低温)是否需特殊选型?
□ 16. 消防安全距离和规范是否满足?
□ 17. 通信方案是否可靠? (冗余通信? 硬接线?)
□ 18. 保护配合方案是否已初步设计?
□ 19. 是否需仿真验证(电磁暂态/稳定性)?
□ 20. 项目工期和设备供货周期是否可行?
```
