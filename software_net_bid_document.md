```
评分维度               │ 权重  │ 得分关键
──────────────────────┼───────┼──────────────────
投标报价               │ 30~35%│ 不低于最低价(否则超30%扣分)
技术方案               │ 35~40%│ ★最关键★ 完整的技术论证

环保/安全/创新          │ 0~5%  │ 加分项
──────────────────────┼───────┼──────────────────
总计                   │ 100%  │

### 3.4 安全与保护设计规范

```
【安全设计必须包含的防御层级】

  - 电芯过温/低温保护
  - 电芯短路保护（熔断器+电芯内部CID）
  - 电芯压差/温差监控和报警

第2层：模组/簇级保护
  - 模块级熔断器
  - 簇级过流保护
  - 簇间均衡保护

第3层：高压箱(BCP)保护
  - 主正/主负接触器+熔断器双重保护
  - 预充回路防止合闸冲击
  - 绝缘监测（正对地/负对地双检测）
  - 簇级紧急分断（BMS指令触发）

第4层：PCS保护
  - 直流过压/欠压、交流过压/欠压
  - 过流/过载/短路保护
  - IGBT模块过温保护
  - 防孤岛保护
  - HVRT/LVRT穿越（如需）

第5层：并网点保护
  - 进线断路器（短路/过流）
  - 防孤岛保护（独立于PCS的）
  - 低/过频率保护
  - 反功率流保护
  - 电能质量问题保护（谐波/闪变）

第6层：消防与应急救援
  - 烟感+温感+可燃气体（H2/CO）三级检测
  - 自动灭火系统（气溶胶/七氟丙烷/全氟己酮）
  - 全系统硬接线紧急停机联动
  - 手动急停按钮（主控室+各舱+各PCS柜）
  - 消防与电力系统隔离（独立物理断开）

### 4.2 成本核算公式

```
【投资总价核算】

设备总价 = Σ(设备单价 × 数量)
安装施工费 = 设备总价 × (10% ~ 25%)  [按变压器/开关柜等难度调整]
调试测试费 = 设备总价 × (3% ~ 5%)
运输保险费 = 设备总价 × (1.5% ~ 3%) [考虑集装箱尺寸和运输距离]
项目管理费 = (设备总价 + 安装费) × (2% ~ 5%)
设计/可研费 = 设备总价 × (1.5% ~ 3%)
培训费 = XX 元/次 × X 次
质保费 = 设备总价 × (1% ~ 2%) × 质保年限
备件 = 设备总价 × (1% ~ 2.5%)

总投资 = 设备总价 + 安装费 + 调试费 + 运输费 + 管理费 + 设计费 + 其他

每kWh单位投资 = 总投资 / 总装电量(kWh) = XX 元/kWh
每kW单位投资 = 总投资 / 总功率(kW) = XX 元/kW

报价总价 = 总投资 × (1 + 毛利%)
  毛利率参考：8%~20% (根据市场竞争激烈程度和关系)
```

## 九、质量保证计划

### 9.1 质量保证体系框架

```
【质量保证计划】

本项目的质量管理体系依据ISO 9001:2015标准建立和执行，
确保项目从设计到交付全过程有质量保证。

9.1 设计质量控制
  - 设计评审（Design Review）：◇ 方案设计阶段，◇ 详细设计阶段
  - 设计方案需通过内部技术评审+业主确认
  - 关键计算书（电缆/结构/热仿真/短路计算）必须独立复核
  - 采用PDM/PLM系统管理设计版本和变更

9.2 采购质量控制
  - 供应商准入：须通过合格供应商评定(含质量管理/技术/服务/安全)
  - 外购设备要求有出厂报告/第三方检测报告
  - 设备出厂前执行FAT(本合同要求项⻅第X节)
  - 设备运输过程中全程监控(温度/振动/防潮)
  
9.3 制造/装配质量控制
  - 关键工序设置停检点(Hold Point)：
    □ 主母线装配后耐压测试
    □ PCS满功率老化前测试
    □ BMS全链路通信测试
    □ 电池舱出厂前全部测试
  - 物料标识和可追溯性（RFID/条码）

9.4 现场施工质量控制
  - 施工方案在开工前获批
  - 所有焊接/电缆操作人员持证上岗
  - 隐蔽工程隐蔽前须经监理/业主/相关方验收(影音记录)
  - 线缆敷设前做100%绝缘/连续性试验
  - 保护/接地系统做完整性和功能测试

9.5 调试与试运行质量控制
  - 单机调试→系统联调→并网验检(顺序不可跳跃)
  - 所有测试数据由第三方测试/见证人员签证
  - 168h试运行记录(全时段的运行数据存储在EMS中进行)
  - 不合格项NCR追踪直至整改完毕

9.6 质量文件交付

  □ 设计图纸/计算书

```

### 12.2 差异化竞争策略

```
【如何让投标方案脱颖而出】

技术方案差异化点：
  1. 与本项目地点相关的本地化设计
     如：高温地区→强调液冷系统优势
     如：高海拔→说明设备已做高原适应性测试
     如：沿海→强调防腐蚀涂装/密封设计

  2. 数据支撑的可靠性论证
     不只是说"可靠"，而是引用MTBF数据或实际项目数据

  3. 详细的控制策略说明
    不只是说"有EMS"，而是展示EMS优化算法和真实优化案例

  4. 带二维码/URL链接的原证扫描件
    所有证书可以扫码验证真伪

商务方案差异化点：
  1. 提供质保期外的延长服务套餐
  2. 提供容量保证方案(如第8、10、15年可用容量保证)
  3. 提供灵活的付款计划
  4. 包含当地的运维人员支持
  5. 可提供融资方案或商业模式的推荐
```
