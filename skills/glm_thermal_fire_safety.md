### 1.3 消防设计基本原则

```
储能消防设计原则:
  1. 预防为主: 极早期预警(VESDA) + BMS热失控监测
  2. 多重防护: 探测→告警→联跳→灭火→排烟
  3. 分舱隔离: 电池舱/电气舱/变压器独立消防
  4. 联跳联切: 消防动作→PCS停机→断路器跳闸
  5. 人员安全: 疏散通道+应急照明+声光报警
  6. 环境友好: 灭火介质无残留/低毒性
```

### 2.4 热负荷计算

```python
def thermal_load_calc(
    total_energy_kwh: float,
    c_rate: float = 0.5,
    round_trip_efficiency: float = 0.95,
    ambient_temp_c: float = 40,
    target_temp_c: float = 25,
    clusters: int = 16,
    containers: int = 1
):
    """电池舱热负荷计算"""
    
    # 充放电损耗热
    p_loss_total = total_energy_kwh * c_rate * (1 - round_trip_efficiency)
    
    # 均衡热(约2%的损耗)
    p_balance = p_loss_total * 0.02
    
    # 辐射热(简化计算)
    # Q_rad = σ × ε × A × (T⁴_amb - T⁴_target) (忽略, 较小)
    p_radiation = 0  # 简化
    
    # 总热负荷
    q_total_kw = p_loss_total + p_balance
    
    # 每集装箱热负荷
    q_per_container = q_total_kw / containers
    
    # 空调/Chiller选型
    cop_air = 3.0
    cop_liquid = 3.5
    cooling_air = q_per_container * 1.2 / cop_air
    cooling_liquid = q_per_container * 1.2 / cop_liquid
    
    # 液冷流量计算
    rho = 1.07  # kg/L
    cp = 3.5  # kJ/(kg·K)
    delta_t = 6  # K (进出水温差)
    flow_rate_lpm = q_per_container * 1.2 * 1000 / (rho * cp * delta_t) / 60 * 60
    
    return {
        "总能量(kWh)": total_energy_kwh,
        "充放电倍率(C)": c_rate,
        "充放电损耗(kW)": round(p_loss_total, 1),
        "均衡热(kW)": round(p_balance, 2),
        "总热负荷(kW)": round(q_total_kw, 1),
        "每集装箱热负荷(kW)": round(q_per_container, 1),
        "风冷空调选型(kW制冷)": round(q_per_container * 1.2, 1),
        "风冷空调耗电(kW)": round(cooling_air, 1),
        "液冷Chiller选型(kW制冷)": round(q_per_container * 1.2, 1),
        "液冷Chiller耗电(kW)": round(cooling_liquid, 1),
        "液冷流量(L/min)": round(flow_rate_lpm, 1),
        "温控方式推荐": "液冷" if total_energy_kwh >= 5000 else "风冷"
    }

# 示例: 5MWh系统
result = thermal_load_calc(total_energy_kwh=5000, c_rate=0.5, containers=4)
import json
print(json.dumps(result, ensure_ascii=False, indent=2))
```

## 三、消防系统设计

### 3.1 气溶胶灭火系统

```
气溶胶灭火系统设计(GB 50067):
  灭火机理: K₂O/MgO固体微粒化学抑制+物理窒息
  设计浓度: 100 g/m³ (电池舱全淹没)
  喷放时间: ≤30s
  灭火时间: ≤60s (明火), ≤120s (深层火)
  残留: 有(需清洁), 不导电(对电气设备安全)
  
  设计计算:
    W = C₂ × K₁ × V  (设计用量)
    W: 灭火剂用量(kg)
    C₂: 设计浓度(g/m³) = 100
    K₁: 安全系数 = 1.3
    V: 防护区净体积(m³)
    
  示例(40尺集装箱, 内净体积56m³):
    W = 100/1000 × 1.3 × 56 = 7.28 kg → 选用8kg装置×1台
    
  安装要求:
    - 喷放口朝向防护区上部(气溶胶比空气轻)
    - 安装高度: 顶部或侧壁上部
    - 距离障碍物≥0.5m
    - 每个防护区≥1台
  
  品牌推荐: 首安/赋安/北大青鸟
```

### 3.2 七氟丙烷(FM200)灭火系统

```
七氟丙烷灭火系统设计(GB 50370):
  灭火机理: 化学抑制(捕捉自由基) + 物理冷却
  分子式: CF₃CHFCF₃
  设计浓度: 7% (V/V, 电气舱)
  喷放时间: ≤10s
  灭火时间: ≤30s
  残留: 无(气态挥发)
  ODP: 0 (不破坏臭氧层)
  GWP: 3220 (温室效应, 需关注)
  NOAEL: 9% (无可见不良影响浓度)
  
  设计计算:
    W = K × (C/(100-C)) × (V/S)
    W: 灭火剂用量(kg)
    K: 安全系数 = 1.3
    C: 设计浓度(%) = 7
    V: 防护区净体积(m³)
    S: 七氟丙烷过热蒸气比容(m³/kg)
    S = 0.1269 + 0.000513 × T (T为防护区温度℃)
    
  示例(电气舱, V=15m³, T=25℃):
    S = 0.1269 + 0.000513 × 25 = 0.1397 m³/kg
    W = 1.3 × (7/93) × (15/0.1397) = 1.3 × 0.0753 × 107.4 = 10.51 kg
    → 选用12kg瓶组×1
    
  系统组成:
    - 储瓶(含七氟丙烷+氮气驱动)
    - 瓶组架
    - 集流管
    - 选择阀(多区)
    - 喷嘴
    - 管道(镀锌钢管)
    - 控制盘(自动/手动/机械应急)
```

## 四、联跳逻辑设计

### 4.1 消防联跳全链逻辑

```
消防联跳逻辑(全链路<200ms):
  
  探测阶段(0~10s):
    VESDA Level3 ──→ 消防主机告警
    烟感探测器 ───→ 消防主机火警
    温感探测器 ───→ 消防主机火警
    BMS热失控预警 ─→ EMS告警
  
  告警阶段:
    消防主机火警确认 ─→ 
    ┌─ 声光报警器启动
    ├─ 消防主机上报EMS
    └─ 启动联跳计时(T1=5s可整定)
  
  联跳阶段(T1后, <200ms):
    ┌─ EMS下发PCS停机指令
    ├─ PCS停机(功率归零+断开直流接触器)
    ├─ BMS断开簇接触器
    ├─ PCC断路器跳闸
    ├─ 储能侧断路器跳闸
    └─ 消防灭火系统启动
  
  灭火阶段:
    ┌─ 气溶胶/七氟丙烷喷放(电启动)
    ├─ 排烟风机启动(喷放完毕后)
    └─ 防火阀关闭(防止蔓延)
  
  恢复阶段(人工确认后):
    ┌─ 通风排气(≥6次/h, 持续30min)
    ├─ 现场检查确认安全
    └─ 人工复位消防系统
```

## 五、硬接线联跳回路

### 5.1 硬接线联跳原理

```
硬接线联跳回路(不依赖通信, 最高可靠性):

  消防信号(无源干接点) ──→ 中间继电器K1
                              │
    ┌─────────────────────────┼─────────────────────────┐
    │                         │                         │
    ▼                         ▼                         ▼
  K1-1(NO)              K1-2(NO)                  K1-3(NO)
  → PCS急停端子         → BMS急停端子              → PCC跳闸线圈
  (短接急停端子)         (断开簇接触器)             (断路器分闸)
  
  信号类型: 无源干接点(NO常开)
  继电器: DC24V中间继电器, 动作时间≤15ms
  触点容量: AC250V/5A 或 DC30V/5A
  接线: 阻燃BV线, 截面≥1.5mm²
  
  冗余设计:
    - 消防信号双路(烟感+温感, 任一触发)
    - 继电器双线圈(防单点失效)
    - 信号线双绞(防电磁干扰)
```

### 5.2 联跳回路接线图

```
硬接线联跳回路接线:

  +24V ───┬── K1线圈 ───┬── 消防信号1(烟感)
          │              │
          │              └── 消防信号2(温感)
          │
          ├── K2线圈 ─── BMS热失控信号
          │
          └── 监视回路 ─→ 消防主机(监视K1/K2状态)

  K1-1(NO) ──→ PCS急停端子(+24V短接)
  K1-2(NO) ──→ BMS急停端子(+24V短接)  
  K1-3(NO) ──→ PCC跳闸线圈(+24V)
  K2-1(NO) ──→ PCS急停端子(+24V短接)
  K2-2(NO) ──→ BMS急停端子(+24V短接)
  
  联跳优先级:
    1. 硬接线(最高优先, 不依赖通信)
    2. 通信联跳(EMS下发, 备用)
    3. 本地急停按钮(现场, 最高优先)
```

### 7.1 温度阈值体系

```
电池舱温度阈值:
  ┌──────────────┬──────────┬──────────────────────────┐
  │  状态          │  温度(℃)  │  动作                      │
  ├──────────────┼──────────┼──────────────────────────┤
  │  正常运行      │  20~30    │  正常充放电                │
  │  偏高          │  30~35    │  加强冷却(提高风量/流量)    │
  │  注意          │  35~40    │  限功率运行(降额至80%)     │
  │  告警          │  40~45    │  限功率运行(降额至50%)     │
  │  降额          │  45~50    │  降额至20%+加强告警        │
  │  停机          │  50~55    │  停止充放电+自然冷却        │
  │  急停          │  >55      │  断开接触器+跳闸           │
  │  热失控预警    │  >80      │  全系统急停+消防联动        │
  └──────────────┴──────────┴──────────────────────────┘

  温度变化率阈值:
    dT/dt > 1℃/s: 热失控预警(BMS检测)
    dT/dt > 2℃/s: 热失控确认(消防联动)
    dT/dt > 5℃/s: 热失控紧急(全系统急停)
```

### 7.2 温度监测点布置

```
温度监测点:
  BMS级:
    - 每个电芯/每2个电芯: 1个NTC(10kΩ, B=3435)
    - 精度: ±1℃
    - 采样周期: ≤1s
  
  模组级:
    - 模组进风口: 1个PT100
    - 模组出风口: 1个PT100

    - 簇平均温度: BMS上报
    - 温差: ≤8℃(正常), >10℃(告警)
  
  舱级:
    - 进风口温度: 1个
    - 回风口温度: 1个
```

### 7.3 温控策略代码

```python
class ThermalControlStrategy:
    """温控策略"""
    
    TEMP_NORMAL = 25
    TEMP_ELEVATED = 35
    TEMP_ATTENTION = 40
    TEMP_ALARM = 45
    TEMP_DERATE = 50
    TEMP_STOP = 55
    TEMP_EMERGENCY = 80
    
    RATE_WARN = 1.0   # ℃/s
    RATE_CONFIRM = 2.0
    RATE_EMERGENCY = 5.0
    
    def __init__(self, target_temp=25):
        self.target_temp = target_temp
    
    def evaluate(self, current_temp: float, temp_rate: float = 0) -> dict:
        """温度评估"""
        if current_temp > self.TEMP_EMERGENCY or temp_rate > self.RATE_EMERGENCY:
            return {"level": "EMERGENCY", "action": "全系统急停+消防联动", "power_limit": 0}
        elif current_temp > self.TEMP_STOP or temp_rate > self.RATE_CONFIRM:
            return {"level": "STOP", "action": "停止充放电+自然冷却", "power_limit": 0}
        elif current_temp > self.TEMP_DERATE:
            return {"level": "DERATE", "action": "降额至20%", "power_limit": 0.2}
        elif current_temp > self.TEMP_ALARM:
            return {"level": "ALARM", "action": "降额至50%", "power_limit": 0.5}
        elif current_temp > self.TEMP_ATTENTION:
            return {"level": "ATTENTION", "action": "降额至80%", "power_limit": 0.8}
        elif current_temp > self.TEMP_ELEVATED:
            return {"level": "ELEVATED", "action": "加强冷却", "power_limit": 1.0}
        elif temp_rate > self.RATE_WARN:
            return {"level": "RATE_WARN", "action": "热失控预警", "power_limit": 0.5}
        else:
            return {"level": "NORMAL", "action": "正常运行", "power_limit": 1.0}
    
    def cooling_control(self, current_temp: float, clusters_temp: list) -> dict:
        """冷却控制策略"""
        temp_diff = current_temp - self.target_temp
        max_cluster_diff = max(clusters_temp) - min(clusters_temp) if clusters_temp else 0
        
        if temp_diff > 15:
            cooling_power = 1.0  # 100%
        elif temp_diff > 10:
            cooling_power = 0.8
        elif temp_diff > 5:
            cooling_power = 0.6
        elif temp_diff > 0:
            cooling_power = 0.3
        else:
            cooling_power = 0.1  # 待机
        
        return {
            "目标温度(℃)": self.target_temp,
            "当前温度(℃)": current_temp,
            "温差(℃)": round(temp_diff, 1),
            "冷却功率(%)": cooling_power * 100,
            "簇间温差(℃)": round(max_cluster_diff, 1),
            "簇温差告警": max_cluster_diff > 8,
            "建议": "检查风道/液冷管路" if max_cluster_diff > 8 else "正常"
        }

# 示例
strategy = ThermalControlStrategy(target_temp=25)
print(strategy.evaluate(47, 0.5))
print(strategy.cooling_control(38, [36, 38, 39, 37]))
```
