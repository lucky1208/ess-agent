## 一、电价体系与政策框架

### 1.1 中国电力市场电价构成体系

```text
用户侧总电费 = 用电电费 + 基本电费（需量/容量）+ 力调电费 + 政府性基金及附加

用电电费 (电量电费):
  按实际用电量 × 执行电价
  执行电价 = 各时段电价（峰/平/谷/尖峰）

基本电费：
  按变压器容量(kVA) 或 按最大需量(kW) 计收
  与用电量无关，每月固定（即使不生产也要缴）

力调电费 (功率因数调整电费):
  根据月度平均功率因数调整
  PF≥0.90：减免
  0.85≤PF<0.90：加收2.5%~5%
  PF<0.85：逐级加收

政府性基金及附加：
  重大水利工程建设基金、可再生能源电价附加、农网还贷等
  ≈0.03~0.05元/kWh
```

## 三、需量电费与容量电费计算

### 3.1 基本电费计算模型

```text
【模式1：按变压器容量计收】
  月基本电费 = 变压器容量(kVA) × 容量电价(元/kVA/月)
  适用场景：用户负荷平稳、变压器容量利用充分

【模式2：按最大需量计收】
  月基本电费 = max(当月实际最大需量, 核定最大需量×40%) × 需量电价(元/kW/月)
  最大需量定义：当月每15分钟平均功率的最大值
  当实际需量 > 核定需量时，超出部分按2倍单价计收

【储能对基本电费的优化逻辑】
  储能放电时（通常在高峰/尖峰时段）降低需量：
    月需量电费节省 = (原最大需量 - 储能削峰后最大需量) × 需量电价
  注意：
    - 储能充电时段不能选在负荷高峰时段（否则反而增加需量）
    - 充电应在低谷/平段进行
```

### 3.2 需量管理的储能量化模型

```text
给定：
  P_load_max = 用户月度最大负荷(kW)
  P_bess     = 储能额定放电功率(kW)
  E_bess     = 储能可用放电能量(kWh)
  price_demand = 需量电价(元/kW/月)

则：
  储能削峰上限 = min(P_bess, P_load_max - P_load_base)
  有效削峰可用时长 = E_bess / 储能削峰上限

  月需量电费节省上限 = min(储能削峰上限, 实际峰值超需量部分) × price_demand
  年需量电费节省 = 月节省 × 12 × 削峰成功率(0~100%)

示例计算：
  某工厂 P_load_max=2000kW, P_load_base=800kW
  P_bess=500kW, price_demand=36元/kW/月
  削峰能力=min(500, 2000-800)=500kW
  月节省=500×36=18,000元
  年节省=18,000×12×0.95=205,200元
```

### 3.3 功率因数调整电费模型

```text
月功率因数 PF_avg = cos(arctan(Q_total/P_total))

PF_avg计算公式：
  P_total = 全月有功电量(kWh)
  Q_total = 全月无功电量(kvarh)
  PF_avg = P_total / sqrt(P_total² + Q_total²)

力调电费费率标准(GB/T 3485)：
  PF≥0.90: 减免力调电费（通常-0.75%至-1.5%）
  PF=0.85~0.90: 加收2.5%
  PF=0.80~0.85: 加收5%
  PF=0.75~0.80: 加收7.5%
  PF<0.75: 加收10%以上

储能PCS的无功能力可辅助调节PF：
  当储能具备SVG功能时，可在不充放电时提供无功补偿
  从而提高功率因数，减少力调电费罚款
```

## 五、IRR内部收益率完整计算模型

### 5.1 IRR定义与公式

```text
IRR (Internal Rate of Return, 内部收益率)
定义：使项目净现值(NPV)为零的折现率

数学公式（解r）：
  NPV = Σ_{t=0}^{n} CF_t / (1+r)^t = 0

其中：
  CF_t = 第t年的净现金流
  t = 年份(从0开始, t=0为初始投资年)
  n = 项目全寿命周期（储能项目通常取15~20年）
  r = IRR（待求的折现率）
```

### 5.2 储能项目IRR标准假设参数

### 5.3 逐年现金流计算模板

```text
【年度收益计算】
  年放电量(kWh) = 装机容量(kWh) × 循环次数/天 × 运行天数 × 效率 × 容量衰减率(t)
  年电费收益(元) = 年放电量 × 峰电放电电价 - 年充电量 × 谷电充电电价
  年需量收益(元) = 月需量节省(kW) × 需量电价(元/kW/月) × 12月
  年总收益 = 年电费收益 + 年需量收益 + 辅助服务收益 + 碳收益

【年度成本计算】
  年折旧 = (设备总投资 - 残值) / 折旧年限
  年利息 = 贷款余额 × 贷款利率
  年运维费 = 初始运维费 × (1+通胀率)^t
  年保险费 = 设备原值 × 保险费率

【年度利润计算】
  年税前利润 = 年总收益 - 年总成本(不含折旧和利息) - 年折旧 - 年利息
  年所得税 = max(0, 年税前利润 × 所得税率)
  年净利润 = 年税前利润 - 年所得税

【年度现金流】
  CF_t (项目投资现金流) = 年净利润 + 年折旧 - 当年资本性支出(PCS更换等)
  CF_0 (初始投资) = -(设备投资 + EPC费 + 开发费 + 其他)
```

### 5.4 IRR典型值参考

```text
工商业储能项目（仅峰谷套利）：
  良好条件(峰谷价差≥0.8元)：IRR = 8%~12%
  一般条件(峰谷价差0.5~0.7元)：IRR = 5%~8%
  较差条件(峰谷价差<0.4元)：IRR < 5% → 不具备投资价值

独立储能/共享储能：
  容量租赁+现货套利：IRR = 5%~8%
  容量租赁+调频服务：IRR = 8%~12%

新能源配储：
  本身为成本项, 不追求独立IRR
  通过减少弃电+调频补贴可能将IRR从"负"提升至微正
```

## 六、NPV净现值与LCOE模型

### 6.1 NPV定义与WACC计算

```text
NPV (Net Present Value, 净现值)
公式：NPV = Σ_{t=0}^{n} CF_t / (1+WACC)^t

WACC (加权平均资本成本):
WACC = E/V × Re + D/V × Rd × (1-Tax)

其中:
  E = 权益资本价值, D = 债务资本价值, V = E+D
  Re = 权益资本成本(通常8%~12%)
  Rd = 债务资本成本(贷款利率, 如4.5%)
  Tax = 企业所得税率(25%)

典型WACC值:
  30%权益 + 70%债务：WACC = 0.3×10% + 0.7×4.5%×0.75 = 5.36%
  50%权益 + 50%债务：WACC = 0.5×10% + 0.5×4.5%×0.75 = 6.69%
  100%权益：WACC = 10%

判定：
  NPV > 0 → 项目有正经济价值, 接受项目
  NPV = 0 → 项目正好达到要求回报率
  NPV < 0 → 项目经济不可行
```

### 6.3 LCOE平准化储能成本

```text
LCOE用于比较储能每kWh的电量成本：
  LCOE = CAPEX / (年充放电量 × 寿命年数 × 效率)

或更准确的公式（考虑折现）：
  LCOE = Σ[(I_t + O_t + F_t)/(1+r)^t] / Σ[E_t/(1+r)^t]

其中：
  I_t = 第t年投资支出
  O_t = 第t年运维支出
  F_t = 第t年燃料/充电支出
  E_t = 第t年放电量(kWh)
  r = 折现率

典型LCOE参考值（2024年水平）：
  工商业储能（2h系统）: ≈0.8~1.2元/kWh
  电网侧储能（4h系统）: ≈0.5~0.8元/kWh
```

## 七、投资回收期计算(静态与动态)

### 7.1 简单回收期（静态）

```text
定义：不考虑资金时间价值, 累计净现金流回正所需的年数

计算：
  寻找最小的 T, 使 Σ_{t=1}^{T} CF_t ≥ |CF_0|

示例计算：
  初始投资：1000万元
  年净收益：150万元（假设各年均匀）
  简单回收期 = 1000 / 150 = 6.67年

静态回收期速算公式（均年收益）：
  回收期 = CAPEX / (年电费收益 + 年需量收益 - 年运维费 - 年充电费)
  
  CAPEX = 装机kWh × 系统单价(元/kWh)
  回收期 ≈ 系统单价(元/kWh) / (循环次数/天 × 330天 × 效率 × 价差(元/kWh))

代入典型值：
  系统单价=800元/kWh, 循环1.2次/天, 效率85%, 价差0.7元/kWh
  回收期 ≈ 800 / (1.2×330×0.85×0.7) = 800 / 235.62 = 3.4年
```

## 十、IRRVaR蒙特卡洛风险仿真模型

### 10.1 VaR/CVaR定义

```text
Value at Risk（VaR）：
  VaR_α = inf{x ∈ R : P(R > x) ≤ 1 - α}

在置信水平 α（如95%）下，最大可能损失。

Conditional VaR（CVaR）：
  CVaR_α = E[R | R ≤ VaR_α]

超过VaR阈值的平均损失（尾部风险期望）。
```

### 10.2 IRRVaR目标函数

```text
max_{P_ess, E_ess}  IRR_expected - λ · CVaR_α(IRR)

其中 λ 为风险厌恶系数（λ > 0），α 为VaR置信水平（通常取0.90或0.95）。
```

### 10.3 蒙特卡洛模拟核心代码(Python)

```python
import numpy as np
from scipy.stats import norm

def monte_carlo_irr(P_ess, E_ess, load_8760, tariff,
                    n_scenarios=1000, random_seed=42):
    """
    蒙特卡洛模拟计算IRR分布
    n_scenarios: 模拟场景数(推荐≥1000)
    """
    np.random.seed(random_seed)
    irr_samples = []

    for s in range(n_scenarios):
        # 随机化关键不确定参数
        price_noise = 1 + np.random.normal(0, 0.15)       # 电价波动±15%
        tariff_scenario = tariff * price_noise

        pv_noise = 1 + np.random.normal(0, 0.20)           # 光伏出力波动±20%

        load_noise = 1 + np.random.normal(0, 0.10)          # 负荷波动±10%
        load_scenario = load_8760 * load_noise

        eff_year = np.random.choice(range(1, 11))
        eta_degradation = (1 - 0.02) ** eff_year           # 设备效率年衰减2%

        irr_s = calc_single_irr_scenario(
            P_ess, E_ess, load_scenario, tariff_scenario, eta_degradation
        )
        irr_samples.append(irr_s)

    return np.array(irr_samples)

def calc_var_cvar(irr_samples, alpha=0.95):
    """计算VaR和CVaR"""
    irr_sorted = np.sort(irr_samples)
    var_idx = int((1 - alpha) * len(irr_samples))
    VaR = irr_sorted[var_idx]
    CVaR = np.mean(irr_sorted[:var_idx])
    return VaR, CVaR

def irrvar_objective(results_df, lambda_risk=0.3, alpha=0.95):
    """
    IRRVaR目标函数：返回每个配置的 IRR - λ·CVaR 值
    """
    score = []
    for _, row in results_df.iterrows():
        irr_samples = row['irr_samples']
        E_IRR = np.mean(irr_samples)
        _, CVaR = calc_var_cvar(irr_samples, alpha)
        score.append(E_IRR - lambda_risk * CVaR)
    return np.array(score)
```

### 11.1 峰谷套利迭代法(方法A)

```text
算法：峰谷套利IRR最优配置迭代法
输入：负荷数据、光伏数据、电价数据、参数边界
输出：最优ESS容量/功率、IRR

2. 双重循环遍历配置：
   for P_ess in [P_min:ΔP:P_max]:

           C_invest = cap_cost * E_ess + power_cost * P_ess
           cash_flows = [-C_invest] + [R_annual - O&M] * L
```

### 11.2 MILP两阶段优化(方法C)

```text
决策变量：
  P_{ch,t}: 充电功率(连续≥0), 96×365维
  P_{dis,t}: 放电功率(连续≥0), 96×365维
  SOC_t: 荷电状态(连续)
  u_{ch,t}, u_{dis,t}: 充放电标志(二进制)

目标函数(日级运营成本最小化):
  min Σ[P_grid,t · Price_t · Δt + u_ch,t · C_sw]

约束条件：
  - 功率平衡: P_load,t + P_ch,t · η_ch = P_pv,t + P_dis,t · η_dis + P_grid,t
  - SOC演化: SOC_{t+1} = SOC_t + P_ch,t · η_ch · Δt - P_dis,t/η_dis · Δt
  - SOC边界: E_ess · SOC_min ≤ SOC_t ≤ E_ess · SOC_max
  - 功率限值: P_ch,t ≤ P_ess · u_ch,t, P_dis,t ≤ P_ess · u_dis,t
  - 互斥约束: u_ch,t + u_dis,t ≤ 1
```

### 11.3 强化学习自适应规划(方法D)

```text
状态空间 S:
  {S_current_hour, S_PV_forecast, S_load_forecast, S_SOC, S_price}

动作空间 A:
  a ∈ [-1, 1],  P_action = a × P_ess

奖励函数 R:
  R_t = (P_dis,t - P_ch,t)·λ_t·η_RT - C_wear·|a_t|

其中 C_wear 为电池单位循环损耗成本:
  C_wear = C_replace / (2 × DOD_max × Cycles_design)
```

## 十四、收益测算决策树

### 14.1 项目经济性快速判断树

```text
初始条件：
  1. 获取峰谷电价：峰=___元/kWh, 谷=___元/kWh
  2. 计算有效价差：△P = 峰电价 - 谷电价 × (1/效率) = __元/kWh
  3. 获取系统单价：CAPEX_total=___元/kWh

快速判断：
  △P ≥ 0.30 元/kWh？ → 否 → ❌ 项目不可行（价差太小）
  │
  └→ 是 → △P ≥ 0.50 元/kWh？
            ├→ 否 → 边际可行, 需要叠加收益
            │          ├→ 有大工业需量费→可行
            │          ├→ 有光伏SP自用→可行
            │          └→ 无叠加收益 → ❌
            │
            └→ 是 → △P ≥ 0.70 元/kWh？
                      ├→ 否 → 可行，需详细测算 (IRR预估5%~8%)
                      └→ 是 → 可行，经济效益良好 (IRR预估8%~12%+)

详细测算（满足全部以下条件IRR≥8%）：
  □ 有效价差 △P ≥ 0.5 元/kWh
  □ 系统单价 ≤ 1000元/kWh
  □ 日循环次数 ≥ 1.2次
  □ 综合效率 ≥ 85%
  □ 年运维费 ≤ 2% of CAPEX
```

## 十六、常见测算误区与规避

### 16.1 电价类误区

| 误区 | 原因 | 纠正 |
|------|------|------|
| 直接用峰电价减谷电价当价差 | 忽略效率损耗 | 有效价差 = 峰电价 - 谷电价/效率 |
| 未考虑充电时也是购电行为 | 充电需向电网付费 | 收益端减去充电费 |
| 假设电价永远不变 | 政策可能调整 | 对电价变化做敏感性±20%分析 |
| 忽略基本电费 | 仅计算电量套利 | 有大工业需量费时需量管理也是收益 |
| 假设放电量100%替代高峰期用电 | 实际不一定匹配 | 需负荷曲线匹配分析 |

### 16.2 技术/财务类误区

| 误区 | 纠正 |
|------|------|
| 效率取99%或PCS标称效率 | 用系统综合效率85%~90% |
| 忽略电池衰减 | 逐年按衰减曲线削减可用放电量 |
| 假设全年满循环 | 取330天/年, 非365天 |
| 只看回收期不看IRR | 必须有IRR/NPV/回收期三维评估 |
| 折现率取太低(3%~4%) | 用WACC(≥5%)或行业基准(≥8%) |
| 漏算PCS更换费用 | PCS寿命10~12年需在中间年加入更换成本 |
| 不考虑税收 | 必须用税后现金流计算IRR |
