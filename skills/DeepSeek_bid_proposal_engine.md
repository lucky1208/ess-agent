## 一、投标文件整体结构与评分模型

### 1.1 标准投标文件目录(7章核心结构)

```text
【储能项目投标文件标准目录】(7章核心技术结构)

第1章  投标函及投标函附录
  1.1 投标函
  1.2 法定代表人身份证明
  1.3 授权委托书
  1.4 投标保证金
  1.5 投标函附录（关键商务条款汇总）

第2章  价格说明书
  2.1 投标报价总表（分供货、安装、服务）
  2.2 设备分项报价表（BOM清单价格）
  2.3 安装施工费用分项
  2.4 调试/培训/运维服务费用

第3章  技术方案（核心章, 评分权重≈40%）
  3.1 项目概述与需求理解
  3.2 系统架构与总体设计
  3.3 主要设备选型与说明
    3.3.1 电池系统(BESS)
    3.3.2 变流器系统(PCS)
    3.3.3 能量管理系统(EMS)
    3.3.4 BMS电池管理系统
    3.3.5 配电与保护系统
    3.3.6 辅助系统(消防/空调/液冷/监控)
  3.4 电网接入方案
  3.5 控制策略与运行模式
  3.6 安全与保护设计
  3.7 通信与网络安全方案
  3.8 技术参数响应表（逐条应答）
  3.9 关键技术亮点/创新点

第4章  设备配置清单（BOM）
  4.1 主设备清单
  4.2 辅助设备清单
  4.3 备品备件清单
  4.4 专用工具清单

第5章  项目实施方案
  5.1 项目管理组织结构
  5.2 关键人员简历
  5.3 项目实施工期计划（甘特图）
  5.4 里程碑节点和交付物
  5.5 风险识别与应对措施

第6章  验收标准
  6.1 FAT验收标准
  6.2 SAT验收标准
  6.3 性能保证指标(Performance Guarantee)
  6.4 质保期与售后承诺

第7章  偏离表与合规矩阵
  7.1 技术偏离表
  7.2 商务偏离表
  7.3 标准合规矩阵
```

### 1.2 评分权重模型(典型央企/国企储能招标)

| 评分维度 | 权重 | 得分关键 |
|---------|------|---------|
| 投标报价 | 30~35% | 不低于最低价超30%扣分 |
| 技术方案 | 35~40% | **最关键** 完整的技术论证 |
| 项目实施方案 | 10~15% | 可行进度+资深团队 |
| 商务方案/企业资质 | 10% | 资质+业绩 |
| 服务承诺/运维方案 | 5~10% | 响应时间+质保期限 |
| 环保/安全/创新 | 0~5% | 加分项 |
| **总计** | **100%** | |

### 3.3 安全与保护设计（6层防御体系）

```text
第1层：电芯级保护
  - 电芯过压/欠压/过温/低温保护
  - 电芯短路保护(熔断器+电芯内部CID)
  - 电芯压差/温差监控和报警

第2层：模组/簇级保护
  - 模块级熔断器
  - 簇级过流保护、簇间均衡保护

第3层：高压箱(BCP)保护
  - 主正/主负接触器+熔断器双重保护
  - 预充回路防止合闸冲击
  - 绝缘监测(正对地/负对地双检测)

第4层：PCS保护
  - DC/AC过压/欠压、过流/过载/短路
  - IGBT过温保护、防孤岛保护
  - HVRT/LVRT穿越(如需)

第5层：并网点保护
  - 进线断路器(短路/过流)
  - 独立防孤岛保护
  - 低/过频率保护、反功率流保护

第6层：消防与应急救援
  - 烟感+温感+可燃气体(H2/CO)三级检测
  - 自动灭火系统
  - 全系统硬接线紧急停机联动
  - 手动急停按钮(主控室+各舱+各PCS柜)
```

### 4.2 成本核算公式

```text
【投资总价核算】

设备总价 = Σ(设备单价 × 数量)
安装施工费 = 设备总价 × (10% ~ 25%)
调试测试费 = 设备总价 × (3% ~ 5%)
运输保险费 = 设备总价 × (1.5% ~ 3%)
项目管理费 = (设备总价 + 安装费) × (2% ~ 5%)
设计/可研费 = 设备总价 × (1.5% ~ 3%)
备件 = 设备总价 × (1% ~ 2.5%)

总投资 = 设备总价 + 安装费 + 调试费 + 运输费 + 管理费 + 设计费 + 其他
每kWh单位投资 = 总投资 / 总装电量(kWh)
每kW单位投资 = 总投资 / 总功率(kW)

报价总价 = 总投资 × (1 + 毛利%)
  毛利率参考：8%~20% (根据市场竞争程度)
```

### 5.3 偏离黄金规则

```text
1. 可偏离项（安全）：
   - 合同金额 < 500万：付款条款可轻微偏离

2. 高风险偏离（注意）：
   - 工期/里程碑：业主关注度极高
   - 质保期：偏离可能商务评分为0
   - 罚款/性能保证条款：难妥协

3. 绝对不可偏离（废标风险）：
   - 投标有效期、投标保证金、关键资质证明
   - "必须/实质性要求"的技术指标
   - 法律/法规要求的条款
```

## 十一、并网接入系统方案设计

### 11.1 公共连接点(PCC)选择原则

| 原则 | 说明 | 典型取值 |
|------|------|---------|
| 就近接入 | 减少线路投资和线路损耗 | 距离 < 500m为佳 |
| 容量匹配 | 接入线路和配变承载能力匹配储能放电功率 | P_储能 ≤ I_线路 × U |
| 产权分界清晰 | 分界点设备归属明确 | 分界开关位于电网资产侧 |
| 电网结构合理 | 避免电磁环网或短路电流超标 | 短路计算需覆盖最大方式 |
| 调度便利性 | 优先接入有调度自动化设施的变电站 | 有远动通道、通信光缆覆盖 |

### 11.2 典型接入方式

```text
方式一：一回T接（最常用）
  变电站10kV母线──10kV线路──[T接点]──储能并网柜
  适用场景：T接点可用，分支线 ≤2km，储能功率 ≤3MW

方式二：专线T接（储能专线）
  变电站10kV母线──10kV专线──[储能并网柜]
  适用场景：储能功率 >3MW 或供电可靠性要求高

方式三：开关站接入
  变电站──35kV/10kV──开关站A──出线──[储能并网柜]
  适用场景：附近有开关站(含备用间隔)

方式四：对侧并网
  储能侧──专用线路──[变电站/开关站35kV/10kV母线]
  适用场景：储能场址距变电站 >3km
```

### 11.3 35kV vs 10kV接入判断条件

| 判断条件 | 35kV接入 | 10kV接入 |
|---------|---------|---------|
| 储能总放电功率 | ≥ 8MW(部分省份 ≥ 6MW) | < 8MW |
| 距离最近35kV变电站 | < 5km且变电站在负荷中心 | 附近无35kV电源点 |
| 调度管理要求 | 必须接入调度自动化系统 | 可选根据功率规模 |

### 11.4 电气计算体系

```text
【潮流计算】
  - 正常方式（最大负荷）：储能满功率放电, 验证电压抬升是否超标(+7%/-10%)
  - 检修方式（N-1）：储能运行时相关线路检修停运
  - 充电工况：储能充电功率最大时验证反向功率对配变影响

电压偏差计算：
  储能放电时: PCC点电压偏差 ≤ +7% (10kV系统即10.7kV以内)
  储能充电时: PCC点电压 ≥ -10% (即9.0kV以上)

【短路电流计算】
  最大方式：验证断路器遮断容量(通常 ≥31.5kA/40kA)
  最小方式：验证保护灵敏度(电流保护 ≥1.5)

储能侧短路电流贡献：
  PCS稳态短路电流：1.1~1.5倍额定电流, 持续2~5s
  PCS在LVRT期间对短路电流支撑能力 (符合GB/T 34120第8.3条)

【无功补偿配置】
  Q_c = P × (tan φ1 - tan φ2)
  方案A（无功优先）：PCS功率因数可调 0.95超前~0.95滞后
  方案B（SVG）：独立配置SVG, 按储能额定功率15%~25%
  方案C（混合）：储能自身无功+配套SVG
```

## 十六、常见失分项与规避策略

### 16.1 高频失分原因与规避

| 失分原因 | 影响程度 | 规避策略 |
|---------|---------|---------|
| 技术方案缺乏对需求的深度理解(像通用模板) | ⭐⭐⭐⭐⭐ (影响技术分40%) | 在方案开头1-2页写"对项目的理解" |
| 未突出关键技术亮点(与其他投标方一样) | ⭐⭐⭐⭐ | 提取2-3个相对竞品的关键亮点 |
| 关键技术参数无法提供证明文件 | ⭐⭐⭐⭐ (诚信风险) | 每项参数标明来源(TUV/第三方报告) |
| 项目管理人员经验不足 | ⭐⭐⭐ | 关键人员必须有同类项目业绩 |
| 过往业绩不匹配或不详尽 | ⭐⭐⭐ | 选择最相关的3~5个业绩重点展示 |
| 价格竞标策略不合理 | ⭐⭐⭐⭐⭐ (30~35%权重) | 考虑综合评分的最优点 |
| O&M运维方案太简陋 | ⭐⭐ | 至少包含巡检周期/响应时间/备件策略/培训 |

### 16.2 差异化竞争策略

```text
技术方案差异化点：
  1. 与本项目地点相关的本地化设计（高温→液冷、高海拔→高原测试、沿海→防腐蚀）
  2. 数据支撑的可靠性论证（引用MTBF数据或实际项目数据）
  3. 详细的控制策略说明（展示EMS优化算法和真实案例）

商务方案差异化点：
  1. 提供质保期外的延长服务套餐
  2. 提供容量保证方案(第8、10、15年可用容量保证)
  3. 提供灵活的付款计划
  4. 包含当地运维人员支持
```
