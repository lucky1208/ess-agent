# EMS策略编译 Skill v1.0

> 设计人：卢继雄 | 基于5年微网项目实战训练

## 概述
将自然语言目标编译为可执行EMS策略配置：削峰填谷、防逆流、需量控制、SOC边界、充电桩限功率。

## 输入
- 自然语言目标 (如"削峰填谷,防逆流,SOC保持10%-90%")
- 电价参数 (峰/谷/平段电价和时长)
- 设备参数 (储能容量/PCS功率/变压器容量)

## 输出
- 策略配置YAML: 优先级/条件/动作/参数
- 策略冲突检测结果
- 策略执行时序图

## 规则

### R1: 削峰填谷策略
- 谷段(低电价): 储能充电, 功率=min(PCS额定, 变压器余量)
- 峰段(高电价): 储能放电, 功率=min(PCS额定, 峰段负荷)
- 优先级: P1

### R2: 防逆流策略
- 条件: 关口表有功<0 (逆流)
- 动作: 降低PCS放电功率或增加充电功率
- 三重保障: 策略层(EMS调整) + PCS功率限制 + 硬件(逆流保护继电器)
- 优先级: P0 (最高)

### R3: 需量控制
- 条件: 实时功率 > 需量阈值×95%
- 动作: 储能放电补缺
- 目标: 最大化需量节省
- 优先级: P1

### R4: SOC边界保护
- SOC<10%: 强制停止放电, 仅允许充电
- SOC>90%: 强制停止充电, 仅允许放电
- SOC<5%: 紧急停机, 保护电池
- 优先级: P0 (不可覆盖)

### R5: 充电桩限功率
- 条件: 变压器负荷+充电桩需求 > 变压器容量
- 动作: 按优先级降低充电桩功率
- 优先级: P2

### R6: 并离网切换
- 并网→离网: PCC检测电网异常→PCS切换V/f控制→储能承担主电源
- 离网→并网: 电网恢复→同期检测→切换PQ控制→合PCC
- 切换时间: <20ms (关键负荷不断电)
- 优先级: P0

## 验证Checklist
- [ ] 策略优先级无冲突
- [ ] SOC边界不可被其他策略覆盖
- [ ] 防逆流三重保障完整
- [ ] 并离网切换逻辑闭环
- [ ] 策略参数在设备能力范围内
