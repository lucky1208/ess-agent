## 一、电价体系与政策框架

### 1.1 中国电力市场电价构成体系

```
用户侧总电费 = 用电电费 + 基本电费（需量/容量）+ 力调电费 + 政府性基金及附加

用电电费 (电量电费)：
  按实际用电量 × 执行电价
  执行电价 = 各时段电价（峰/平/谷/尖峰）

基本电费：
  按变压器容量(kVA) 或 按最大需量(kW) 计收
  与用电量无关，每月固定（即使不生产也要缴）

力调电费 (功率因数调整电费)：
  根据月度平均功率因数调整
  PF≥0.90：减免
  0.85≤PF<0.90：加收2.5%~5%
  PF<0.85：逐级加收

政府性基金及附加：
  重大水利工程建设基金、可再生能源电价附加、农网还贷等
  ≈0.03~0.05元/kWh
```

## 二、区域电价数据库结构

### 2.1 电价数据库表结构设计

```sql
-- 表1: 区域基本信息表
CREATE TABLE region_info (
    id              INT PRIMARY KEY,
    province        VARCHAR(20),     -- 省
    city            VARCHAR(30),     -- 市（如适用）
    grid_company    VARCHAR(50),     -- 电网公司（如：国网XX省电力公司）
    voltage_level   VARCHAR(10),     -- 电压等级：10kV/35kV/110kV/220kV
    tariff_type     VARCHAR(20),     -- 电价类型：一般工商业/大工业
    effective_date  DATE,            -- 生效日期
    expire_date     DATE,            -- 失效日期
    document_ref    VARCHAR(100)     -- 政策文件号
);

-- 表2: 分时电价表（核心表）
CREATE TABLE time_of_use_tariff (
    id              INT PRIMARY KEY,
    region_id       INT,             -- 关联区域
    season          VARCHAR(10),     -- 季节：全年/夏季/冬季/春秋季
    period_type     VARCHAR(10),     -- 时段类型：尖峰/高峰/平段/低谷/深谷
    start_time      TIME,            -- 起始时间
    end_time        TIME,            -- 结束时间
    tariff_yuan_kwh DECIMAL(8,4),    -- 电度电价(元/kWh)
    tariff_ratio    DECIMAL(4,2),    -- 浮动倍数(相对于平段)
    FOREIGN KEY (region_id) REFERENCES region_info(id)
);

-- 表3: 基本电费表
CREATE TABLE demand_capacity_tariff (
    id              INT PRIMARY KEY,
    region_id       INT,
    capacity_price  DECIMAL(8,2),    -- 容量电价(元/kVA/月)
    demand_price    DECIMAL(8,2),    -- 需量电价(元/kW/月)
    max_demand_rate DECIMAL(4,2),    -- 最大需量核定比率
    FOREIGN KEY (region_id) REFERENCES region_info(id)
);

-- 表4: 政府性基金及附加表
CREATE TABLE surcharge (
    id              INT PRIMARY KEY,
    region_id       INT,
    surcharge_name  VARCHAR(100),    -- 基金/附加名称
    amount_yuan_kwh DECIMAL(8,6),    -- 金额(元/kWh)
    FOREIGN KEY (region_id) REFERENCES region_info(id)
);
```

## 三、需量电费与容量电费计算

### 3.1 基本电费计算模型

```
【模式1：按变压器容量计收】
  月基本电费 = 变压器容量(kVA) × 容量电价(元/kVA/月)

  适用场景：
  - 用户负荷平稳、变压器容量利用充分
  - 不适合有储能可削峰的情况

【模式2：按最大需量计收】
  月基本电费 = max(当月实际最大需量, 核定最大需量×40%) × 需量电价(元/kW/月)

  最大需量定义：当月每15分钟平均功率的最大值
  核定最大需量 = 合同约定的最大需量值
  当实际需量 < 核定需量×40%时，按核定需量×40%计收
  当实际需量 > 核定需量时，超出部分按2倍单价计收

【储能对基本电费的优化逻辑】
  储能放电时（通常在高峰/尖峰时段）降低需量，收益为：
    月需量电费节省 = (原最大需量 - 储能削峰后最大需量) × 需量电价

  注意：
  - 储能充电时段不能选在负荷高峰时段（否则反而增加需量）
  - 充电应在低谷/平段进行
```

### 3.2 需量管理的储能量化模型

```
给定：
  P_load_max = 用户月度最大负荷(kW)
  P_bess     = 储能额定放电功率(kW)
  E_bess     = 储能可用放电能量(kWh)
  soc_range  = [SOC_min_operation, SOC_max_operation]
  price_demand = 需量电价(元/kW/月)

则：
  储能削峰上限 = min(P_bess, P_load_max - P_load_base)
  有效削峰可用时长 = E_bess / 储能削峰上限 (需在高峰时段)
  
  月需量电费节省上限 = min(储能削峰上限, 实际峰值超需量部分) × price_demand
  年需量电费节省 = 月节省 × 12 × 削峰成功率(0~100%)

示例计算：
  某工厂 P_load_max=2000kW, P_load_base=800kW
  P_bess=500kW, price_demand=36元/kW/月
  削峰能力=min(500, 2000-800)=500kW
  月节省=500×36=18,000元
  年节省=18,000×12×0.95(成功率)=205,200元
```

### 3.3 功率因数调整电费模型

```
月功率因数 PF_avg = cos(arctan(Q_total/P_total))

PF_avg计算公式：
  P_total = 全月有功电量(kWh)
  Q_total = 全月无功电量(kvarh)
  PF_avg = P_total / sqrt(P_total² + Q_total²)

力调电费费率标准(GB/T 3485)：
  PF≥0.90: 减免力调电费（通常-0.75%至-1.5%）
  PF=0.85~0.90: 加收2.5%
  PF=0.80~0.85: 加收5%
  PF=0.75~0.80: 加收7.5%
  PF<0.75: 加收10%以上

储能PCS的无功能力可辅助调节PF：
  当储能具备SVG功能时，可在不充放电时提供无功补偿
  从而提高功率因数，减少力调电费罚款
```

## 四、储能收益模式全景

### 4.1 收益模式分类

```
【A. 用户侧储能（表后/BTM - Behind The Meter）收益模式】

A1. 峰谷价差套利 (Peak-Valley Arbitrage)
  模式：谷电/平电充电 → 峰电/尖峰电放电
  收益 = Σ(放电电量×高峰电价 - 充电电量×低谷电价)
  
A2. 需量管理 (Demand Charge Management)
  模式：储能放电削减尖峰负荷需量
  收益 = 需量降低值(kW) × 需量电价(元/kW/月) × 月数

A3. 光伏+储能 自发自用优化
  模式：白天光伏发电储存，傍晚用电高峰释放
  收益 = 光伏自发自用增收 + 余量上网增收(如有)

A4. 需量响应/虚拟电厂收益
  模式：参与需求响应、调频、备用等电力市场
  收益 = 响应量(MW) × 补偿价格 + 容量补偿

A5. 动态增容 (Capacity Upgrade Deferral)
  模式：储能延缓配电容量扩容投资
  收益 = 延期增容投资节省(逐年折现)

A6. 基本电费优化
  模式：在计费方式"按需量"下，削峰降低需量
  收益 = 前述需量电费节省模型

A7. 碳收益/绿证交易
  模式：储能参与碳市场、绿证交易
  收益 = 碳减排量(tCO2) × 碳价 + 绿证数量 × 绿证价格


【B. 电网侧储能（表前/FTM - Front Of The Meter）收益模式】

B1. 电力现货市场套利
  模式：低价时段充电、高价时段放电
  收益 = 日前/实时市场价差收益

B2. 调频辅助服务 (一次调频/AGC)
  模式：PCS快速响应频率偏差
  收益 = 调频容量(MW) × 容量补偿(元/MW·h) + 调频里程(MW) × 里程补偿

B3. 备用容量服务
  模式：合同约定的旋转/非旋转备用
  收益 = 备用容量(MW) × 备用容量费(元/MW·h)

B4. 可再生能源消纳
  模式：弃风弃光存储后释放
  收益 = 低价购"弃电"+ 高价售出

B5. 黑启动服务
  模式：电网全停后储能提供黑启动能力
  收益 = 黑启动容量费(一次性或年费方式)

B6. 输配电延期投资 (T&D Deferral)
  模式：储能替代峰荷时段的配电扩容
  收益 = 延期投资节省(类似A5)


【C. 新能源场站配储】

C1. 解决弃风弃光
  模式：电网限电时充电，不限时放电
  收益 = 减少限电损失的电量 × 上网电价

C2. 平滑出力/跟踪计划
  模式：储能平抑新能源出力的分钟级波动
  收益 = 减少考核罚款 + 出清电价提高

C3. 一次调频
  模式：新能源场站提供一次调频服务
  收益 = 调频服务费（各省补偿标准不同）

C4. 政策要求配储 (强制配置)
  模式：各省新能源项目强制配储比例(10%~20%×2h)
  这是成本项，不是收益项，但可叠加B类收益
```

### 4.2 收益模式组合决策树

```
项目类型？
  ├── 工商业储能（用户侧）
  │   ├── 最大收益来源：A1 峰谷价差套利 (占比60~80%)
  │   ├── 次要收益：A2 需量管理 (如有大工业需量费)
  │   ├── 附加收益：A3 光伏配储 / A6 碳收益
  │   └── 纯A1模式下：IRR需满足≥8%(典型投资要求)
  │
  ├── 独立储能/共享储能（电网侧）
  │   ├── 收益组合：B1+B2+B3 或 B1+B2
  │   ├── 有时叠加：B4 消纳新能源
  │   ├── 容量租赁费用（租赁给新能源场站）
  │   └── 投资回报通常由容量租赁+电量收益+辅助服务三部分构成
  │
  └── 新能源配储
      ├── 解决弃风弃光：C1 (正收益)
      ├── 考核罚款减免：C2 (负成本)
      ├── 辅助服务：C3 (附加收益)
      └── 注意：C4 属于成本而非收益
```

## 五、IRR内部收益率计算

### 5.1 IRR定义与公式

```
IRR (Internal Rate of Return, 内部收益率)
定义：使项目净现值(NPV)为零的折现率

数学公式（解r）：
  NPV = Σ_t=0^n  CF_t / (1+r)^t  = 0

其中：
  CF_t = 第t年的净现金流（回收-投资的差额）
  t = 年份(从0开始，t=0为初始投资年)
  n = 项目全寿命周期（储能项目通常取15~20年）
  r = IRR（待求的折现率）

计算方法：
  - Excel/Numpy: IRR(values) 或 IRR(values, guess)
  - 二分法迭代
  - 牛顿法迭代
```

### 5.2 储能项目IRR计算的标准假设

```yaml
【项目共性假设】
project:
  life_cycle_years: 15           # 项目全寿命周期
  construction_period: 1         # 建设期(年)，t=0~1
  tax_rate: 0.25                 # 企业所得税(25%)
  financing_ratio: 0.7           # 融资比例(项目贷款/总投资)
  loan_interest_rate: 0.045      # 长期贷款利率(LPR基准:3.6%+点差≈4.5%)
  loan_term_years: 10            # 贷款年限
  depreciation_method: "直线法"   # 折旧方式
  depreciation_years: 15         # 折旧年限(与项目生命周期一致或独立)
  residual_rate: 0.05            # 残值率(设备)

【投资假设】
investment:
  battery_cost_per_kwh: 600      # 电池系统单价(元/kWh)，含电池+PCS+BMS+集装箱
  epc_cost_ratio: 0.15          # EPC/安装/并网费占设备投资比例(15%~25%)
  development_cost_ratio: 0.05   # 开发/审批/可研/保险/其他(5%~10%)
  # --- 逐年变化假设 ---
  battery_cost_decline_rate: 0.05  # 电池成本年下降率(未来预测)

【运营假设】
operation:
  cycles_per_day: 1.5            # 每日等效满充满放循环次数(峰谷套利模式)
  operating_days_per_year: 330   # 年运行天数（考虑检修/故障停机）
  round_trip_efficiency: 0.88    # 系统综合充放电效率(含辅助功耗)
  capacity_degradation:          # 电池容量衰减
    year_1: 0.98                 # 第1年末可用容量/初始容量(%)

  effective_spread: 0.70

### 5.3 逐年现金流计算模板

```
【年度收益计算】
  年放电量 (kWh) = 装机容量(kWh) × 循环次数/天 × 运行天数 × 效率 × 容量衰减率(t)
  年电费收益 (元) = 年放电量 × 峰电放电电价 - 年充电量 × 谷电充电电价
                  = 年放电量 × 有效价差(将效率折算入价差后)
  年需量收益 (元) = 月需量节省(kW)×需量电价(元/kW/月)×12月 (如适用)

  年总收益 = 年电费收益 + 年需量收益 + 辅助服务收益 + 碳收益

【年度成本计算】
  年折旧 = (设备总投资 - 残值) / 折旧年限
  年利息 = 贷款余额 × 贷款利率
  年运维费 = 初始运维费 × (1+通胀率)^t
  年保险费 = 设备原值 × 保险费率

  年总成本 = 年折旧 + 年利息 + 年运维费 + 年保险费

【年度利润计算】
  年税前利润 = 年总收益 - 年总成本(不含折旧和利息) - 年折旧 - 年利息
  年所得税 = max(0, 年税前利润 × 所得税率)
  年净利润 = 年税前利润 - 年所得税

【年度现金流】
  CF_t (项目投资现金流) = 年净利润 + 年折旧 - 当年资本性支出(PCS更换等)
  CF_0 (初始投资) = -(设备投资 + EPC费 + 开发费 + 其他)
  CF_t (运营期) = 年净利润 + 年折旧 + 年利息×(1-税率)

  简化公式（自由现金流FCFF）：
  FCF = EBIT×(1-T) + 折旧 - 资本性支出 - 营运资本增量

【IRR计算】
  输入序列：[CF_0, CF_1, CF_2, ..., CF_n]
  其中 CF_n = 运营现金流 + 资产残值回收 + 营运资本回收
  
  在Excel中：=IRR(CF_0:CF_n, 0.1)
  如果IRR ≥ 基准收益率（通常8%~10%），则项目可行
```

### 5.4 IRR典型值参考

```
工商业储能项目（仅峰谷套利）：
  良好条件(峰谷价差≥0.8元)：IRR = 8%~12%
  一般条件(峰谷价差0.5~0.7元)：IRR = 5%~8%
  较差条件(峰谷价差<0.4元)：IRR < 5% → 不具备投资价值

独立储能/共享储能：
  容量租赁+现货套利：IRR = 5%~8%
  容量租赁+调频服务：IRR = 8%~12%
  （政策驱动，变动大，需对容量租赁价格做敏感性分析）

新能源配储：
  本身为成本项，不追求独立IRR
  通过减少弃电+调频补贴可能将IRR从"负"提升至微正
```

## 六、NPV净现值模型

### 6.1 NPV定义与公式

```
NPV (Net Present Value, 净现值)
定义：项目全寿命周期现金流的折现总和

公式：
  NPV = Σ_t=0^n  CF_t / (1+WACC)^t

其中：
  WACC (Weighted Average Cost of Capital，加权平均资本成本)
  WACC = E/V × Re + D/V × Rd × (1-Tax)
  
  E = 权益资本价值
  D = 债务资本价值
  V = E + D (总资本)
  Re = 权益资本成本（通常取8%~12%）
  Rd = 债务资本成本（贷款利率，如4.5%）
  Tax = 企业所得税率（25%）

典型WACC值：
  30%权益 + 70%债务：WACC = 0.3×10% + 0.7×4.5%×0.75 = 5.36%
  50%权益 + 50%债务：WACC = 0.5×10% + 0.5×4.5%×0.75 = 6.69%
  100%权益：WACC = 10%

判定：
  NPV > 0 → 项目有正经济价值，接受项目
  NPV = 0 → 项目正好达到要求回报率
  NPV < 0 → 项目经济不可行
```

### 6.3 LCOE (平准化储能成本) 辅助指标

```
LCOE用于比较储能每kWh的电量成本：
  LCOE = CAPEX / (年充放电量 × 寿命年数 × 效率)

或更准确的公式（考虑折现）：
  LCOE = Σ[I_t + O_t + F_t]/(1+r)^t / Σ[E_t]/(1+r)^t

其中：
  I_t = 第t年投资支出
  O_t = 第t年运维支出
  F_t = 第t年燃料/充电支出
  E_t = 第t年放电量(kWh)
  r = 折现率

典型LCOE参考值（2024年水平）：
  工商业储能（2h系统）: ≈0.8~1.2元/kWh
  电网侧储能（4h系统）: ≈0.5~0.8元/kWh
```

## 七、回收期计算

### 7.1 简单回收期（静态）

```
定义：不考虑资金时间价值，累计净现金流回正所需的年数

计算：
  寻找最小的 T，使 Σ_{t=1}^T CF_t ≥ |CF_0|

其中：
  CF_0 = 初始总投资（取正值）
  CF_t = 第t年净现金流(不含初始投资)

示例计算：
  初始投资：1000万元
  年净收益：150万元（假设各年均匀）
  简单回收期 = 1000 / 150 = 6.67年

静态回收期速算公式（均年收益）：
  回收期 = CAPEX / (年电费收益 + 年需量收益 - 年运维费 - 年充电费)
  
  CAPEX = 装机kWh × 系统单价(元/kWh)
  年电费收益 = 装机kWh × 循环次数/天 × 365 × 效率 × 价差
  回收期 ≈ 系统单价(元/kWh) / (循环次数/天 × 330天 × 效率 × 价差(元/kWh))

代入典型值：
  系统单价=800元/kWh, 循环1.2次/天, 效率85%, 价差0.7元/kWh
  回收期 ≈ 800 / (1.2×330×0.85×0.7) = 800 / 235.62 = 3.4年
```

计算：

其中 i = 基准折现率（常用WACC或行业基准收益率，如8%）

### 7.3 回收期判定标准

```
典型项目投资可接受回收期：
  工商业储能（峰谷套利）：
    理想：3~5年（静态回收期）
    一般：5~7年
    临界：>8年需要仔细评估
  
  独立储能/共享储能：
    理想：5~7年
    一般：7~10年
    (因收益来源多元，需看IRR而非单纯回收期)

  注意事项：
    - 静态回收期不反映资金时间价值，仅做初步参考
    - 折现回收期优于简单回收期
    - 回收期短不等于IRR高（需要看全生命周期）
    - 不应仅用回收期做投资决策，需结合IRR+NPV
```

│ 黑启动          │ 一次性     │ 容量费         │ 低频率，高单次价值      │
│ 惯性响应        │ 探索阶段   │ 待定           │ 低(目前储能无惯性)     │
└────────────────┴────────────┴──────────────┴──────────────────────┘

各区域市场典型数据（仅供参考，实际以各省最新市场规则为准）：
  广东调频：容量补偿约8~15元/MW·h + 里程补偿约5~10元/MW（历史）
  华北调频：独立储能参与，补偿规则各省差异大
  华东调频：储能可参与，价格因省而异
  （注意：辅助服务价格随市场供需波动，不做固定假设）
```

### 9.2 调频收益估算模型

```
【AGC调频服务基本模型】

假设一个独立储能电站：
  额定功率 P = 50MW
  可用调频容量 P_reg = 50MW × 80% (留SOC裕量) = 40MW
  日平均可用小时数 = 24h × 可用率(95%) = 22.8h
  月可用容量 = P_reg × 30天 × 22.8h/天 = 40 × 684 = 27,360 MW·h

如月容量补偿为 Y 元/MW·h：
  月容量收益 = 27,360 × Y

如还需考虑调频里程（实际调节的MW·h总量）：
  假设日平均里程 = P_reg × 2倍调节 × 调节小时
  月里程 = ... MW （乘以里程补偿单价）

【调频收益的简化估算】
  每MW储能日调频收益 ≈ 1000~3000元/MW/天（参考华南市场历史值）
  50MW储能日调频收益 ≈ 5~15万元
  年调频收益 ≈ 1800~5500万元（50MW规模）
  （注意：仅为历史参考区间，不做收益承诺）
```

### 9.3 备用容量收益估算

```
【旋转备用 (Spinning Reserve)】

典型需求：系统需要维持一定备用容量（3%~5%峰荷）
储能可通过维持SOC在中间水平提供上行/下行备用

收益模型：
  月备用容量费收益 = P_reserve × 备用容量单价(元/MW/月)
  例如：10MW备用容量 × 3000元/MW/月（示例价）= 3万元/月

【容量市场/容量租赁】

部分省份允许独立储能向新能源场站租赁容量：
  租赁价格：约200~400元/kW/年（示例区间，各区域差异大）
  100MW储能出租50MW容量：
    年容量租赁收入 = 50,000kW × 300元/kW/年 = 1500万元/年

容量租赁是独立储能的主要确定性收入来源之一
```

### 9.4 辅助服务收益叠加原则

```
辅助服务之间不是可以简单加总的，需按"服务互斥性"减去重叠部分：

互斥规则：
  ├── 同一时刻储能不能同时提供调频和电量套利
  │   （因为调频需要在SOC中间水平留上下调节空间）
  │
  ├── 参与旋转备用需留在适中的SOC（如50%），
  │   意味着该时段不能深度充放电
  │
  ├── 一次调频和AGC可叠加考虑（频率发生随机事件）
  │
  └── SVG/无功补偿和电量套利可同时（在功率额定时仍有
      无功容量使用）

收益最大化优化：
  在电价峰段 → 优先放电套利或削峰
  在电价谷段 → 优先充电
  在平段 → 可提供调频/备用（兼充电维持SOC）
  
  日优化决策：
  IF 峰谷价差 > 预期调频收益 → 进行峰谷套利
  ELSE → 参与调频市场
  (需要EMS具备多市场优化调度能力)
```

## 十一、收益测算决策树

### 11.1 项目经济性快速判断树

```
初始条件：
  1. 获取峰谷电价：峰=___元/kWh, 谷=___元/kWh
  2. 计算有效价差：△P = 峰电价 - 谷电价 × (1/效率) = __元/kWh
  3. 获取系统单价：CAPEX_total=___元/kWh
  4. 获取每日循环次数：N=____

快速判断：
  △P ≥ 0.30 元/kWh？ → 否 → ❌ 项目不可行（价差太小）
  │
  └→ 是 → △P ≥ 0.50 元/kWh？
            │
            ├→ 否 → 边际可行，需要叠加重建收益
            │          ├→ 有大工业需量费→可行（基本费优化加持）
            │          ├→ 有光伏SP自用→可行（光伏增收加持）
            │          └→ 无叠加收益 → ❌ 不建议投资
            │
            └→ 是 → △P ≥ 0.70 元/kWh？
                      ├→ 否 → 可行，需详细测算
                      │         └→ 标准IRR预估：5%~8%
                      └→ 是 → 可行，经济效益良好
                                └→ 标准IRR预估：8%~12%+

详细测算（满足全部以下条件IRR≥8%）：
  □ 有效价差 △P ≥ 0.5 元/kWh
  □ 系统单价 ≤ 1000元/kWh
  □ 日循环次数 ≥ 1.2次（或等效年循环≥400次）
  □ 综合效率 ≥ 85%
  □ 年运维费 ≤ 2% of CAPEX
  □ 贷款利息 ≤ 5%

注：以上为经验规则，实际数据以当地电价和市场价格为准
```

### 11.2 项目收益率等级

```
A级项目（高回报 - IRR > 12%）:
  - 峰谷价差 > 0.8元/kWh
  - 系统采购成本低于行业均值20%+
  - 有需量管理+调频等叠加收益
  - 日充放电可达2次
  → 强烈建议投资，争取多量

B级项目（中等回报 - IRR 8%~12%）:
  - 峰谷价差 0.6~0.8元/kWh
  - 系统采购成本在行业均值附近
  - 日充放电1.2~1.5次
  → 建议投资，做详细敏感性分析

C级项目（低回报 - IRR 5%~8%）:
  - 峰谷价差 0.4~0.6元/kWh
  - 需叠加其他收益来源
  - 回收期需更耐心(7~10年)
  → 仅当叠加收益后IRR达标才投资

D级项目（无回报 - IRR < 5%）:
  - 峰谷价差 < 0.4元/kWh
  - 系统成本偏高
  → 不建议投资，待电价政策变化或设备成本下降后重评估
```

## 十二、常见测算误区

### 12.1 电价类误区

| 误区 | 原因 | 纠正 |
|------|------|------|
| 直接用大工业峰电价减谷电价当价差 | 忽略了效率损耗（充100度只放88度） | 有效价差 = 峰电价 - 谷电价/效率 |
| 未考虑充电时也是购电行为 | 充电需向电网付费 | 需在收益端减去充电费，重新计算净套利 |
| 假设电价永远不变 | 政策可能调整峰谷时段/浮动比例 | 对电价变化做敏感性20%±分析 |
| 忽略基本电费 | 仅计算了电量套利 | 如用户有大工业需量费，需量管理也是收益来源 |
| 假设放电量100%替代高峰期用电 | 实际不一定匹配 | 需根据负荷曲线做"储能可替代电量"的匹配分析 |

### 12.2 技术类误区

| 误区 | 原因 | 纠正 |
|------|------|------|
| 效率取99%或PCS标称效率 | 系统综合效率含辅助功耗 | 用系统综合效率85%~90%，不是单PCS效率 |
| 忽略电池衰减 | 假设每年容量不变 | 逐年按衰减曲线削减可用放电量 |
| 假设全年满循环 | 有检修、故障、电网中断 | 取330天/年，不是365天 |
| 忽略充电时间约束 | 谷电时间只有X小时，可能不够充 | 验证谷电时段能否完成全充电 |

### 12.3 财务类误区

| 误区 | 原因 | 纠正 |
|------|------|------|
| 只看回收期不看IRR | 回收期忽略了回收后的长期收益 | 必须有IRR/NPV/回收期三维评估 |
| 折现率取太低(3%~4%) | 低估资本成本 | 用WACC(≥5%)或行业基准(≥8%) |
| 漏算PCS更换费用 | PCS寿命(10~12年)需更换 | 在中间年加入PCS更换成本 |
| 不考虑税收 | 先算出高IRR但没扣税 | 必须用税后现金流计算IRR |
| 忽略通货膨胀对运维费的影响 | 假设运维费10年不变 | 运维费年增2%~3% |
