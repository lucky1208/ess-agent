# 电价收益 Skill v1.0

> 设计人：卢继雄 | 基于5年微网项目实战训练

## 概述
根据各地电价/需量/容量电费, 计算IRR、NPV、回收期、敏感性分析。

## 输入
- 电价: 峰/谷/平段(元/kWh) + 时长
- 储能配置: 容量(kWh)/功率(kW)/投资(万元)
- 负荷曲线: 典型日24h功率
- 运行天数/衰减率/贴现率

## 输出
- 年收益明细: 峰谷套利/需量节省/容量节省
- IRR (内部收益率)
- NPV (净现值)
- 回收期 (年)
- 敏感性分析: 电价±20%/投资±20%/衰减±20%

## 规则

### R1: 峰谷套利计算
- 日收益 = Σ(放电功率×峰电价 - 充电功率×谷电价) × 时长
- 年收益 = 日收益 × 工作天数 × (1-衰减率)^n

### R2: 需量节省
- 原需量 = max(月最大负荷)
- 新需量 = max(月最大负荷 - 储能放电)
- 节省 = (原需量 - 新需量) × 需量电价

### R3: IRR计算
- NPV(IRR) = 0
- 用牛顿法迭代求解

### R4: 敏感性分析
- 电价+20%: 重算IRR/NPV/回收期
- 投资+20%: 重算IRR/NPV/回收期
- 衰减率+20%: 重算IRR/NPV/回收期

## 验证Checklist
- [ ] 峰谷套利>0
- [ ] IRR>0
- [ ] 回收期<设备寿命
- [ ] 敏感性范围覆盖±20%
