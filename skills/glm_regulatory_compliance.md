```
储能电站项目合规审批全流程:

  ① 项目立项 ──→ ② 发改委备案 ──→ ③ 环评 ──→ ④ 安评 ──→ ⑤ 能评
       │               │              │           │          │
    可行性研究       备案证明        环评批复     安评报告    能评批复
       │               │              │           │          │
       ▼               ▼              ▼           ▼          ▼
  ⑥ 并网批复 ──→ ⑦ 消防审查 ──→ ⑧ 施工许可 ──→ ⑨ 施工 ──→ ⑩ 并网验收
       │               │              │           │         │
    接入系统批复     消防审查意见    施工许可证   竣工验收   正式并网
       │               │              │           │         │
       ▼               ▼              ▼           ▼         ▼
  ⑪ 商业运营
  
  关键前置条件:
    ②→③: 备案后方可开展环评
```

## 三、环评(环境影响评价)

### 3.1 环评类别判定

```
储能项目环评类别判定:
  ┌──────────────────┬──────────────┬──────────────────┐
  │  项目规模          │  环评类别     │  编制要求          │
  ├──────────────────┼──────────────┼──────────────────┤
  │  <1MW/2MWh        │  登记表       │  业主自行填报      │
  │  1~10MW/2~20MWh   │  报告表       │  委托环评单位编制  │
  │  >10MW/20MWh      │  报告书       │  委托环评单位编制  │
  └──────────────────┴──────────────┴──────────────────┘
  
  大部分工商业储能: 报告表
  大型独立储能电站: 报告书
```

### 3.2 环评关注要素

```
储能项目环评关注要素:
  1. 电磁辐射
     - PCS变流器产生的工频电磁场
     - 变压器产生的工频电磁场
     - 评估: 站界电磁场强度≤4kV/m(电场)/≤0.1mT(磁场)
     - 储能项目一般远低于限值, 电磁辐射不是主要问题
  
  2. 噪声
     - PCS冷却风机: 65~75dB(A) @1m
     - 变压器: 55~65dB(A) @1m
     - 空调/Chiller: 55~65dB(A) @1m
     - 站界噪声: 昼间≤60dB, 夜间≤50dB(2类区)
     - 措施: 隔声屏障+减振基础+夜间降功率
  
  3. 废水
     - 运行期: 基本无生产废水
     - 事故消防水: 含灭火药剂, 需收集处理
     - 生活污水: 少量, 排入市政管网或化粪池
  
  4. 废气
     - 运行期: 基本无废气排放
     - 事故消防烟气: 短时, 非常规排放
  
  5. 固废
     - 废旧电池: 危废(HW49), 需交资质单位处置
     - 变压器废油: 危废(HW08), 需收集处置
     - 一般固废: 包装材料/废旧金属
  
  6. 环境风险
     - 电池热失控: 最大可信事故
     - 变压器油泄漏: 干式变压器可避免
     - 电解液泄漏: LFP电池电解液量少
```

### 3.3 环评编制单位资质

```
环评编制单位要求(2019年新环评法):
  - 取消环评资质等级要求
  - 编制单位需在环评信用平台注册
  - 编制主持人需持环评工程师证书

    - 了解当地环评单位业绩
    - 费用: 报告表3~10万, 报告书10~30万
    - 周期: 报告表4~8周, 报告书8~16周
```

## 四、并网批复

### 4.1 并网申请与审批

```
并网批复流程:
  1. 准备并网申请材料(见glm_grid_interconnection.md)
  2. 向当地供电公司提交并网申请
  3. 供电公司组织接入系统方案审查
     - 召开审查会(供电公司各专业参加)
     - 形成审查意见(明确接入点/容量/技术条件)
  4. 按审查意见修改设计
  5. 获取并网批复文件
     - 内容: 接入点/容量/保护配置/计量/通信
     - 有效期: 通常1年
  
  并网批复费用:
    - 申请费: 0(供电公司不收费)
    - 接入系统方案编制费: 5~20万(委托设计院)
    - 审查费: 0(供电公司不收费)
    - 系统备用费(部分省份): 按容量收取
  
  注意事项:
    - 部分地区需先取得环评批复才能申请并网
    - 大型项目(>10MW)需省级能源主管部门意见
    - 并网批复有效期通常1年, 逾期需重新申请
```

### 4.2 接入系统方案审查要点

```
接入系统方案审查要点:
  供电公司审查内容:
    1. 接入点选择合理性
       - 变电站容量是否充裕
       - 间隔是否有空余
       - 距离是否合理
    
    2. 短路电流校验
       - 储能接入后短路电流是否超标
       - 开关设备遮断容量是否满足
    
    3. 电能质量评估
       - 谐波/电压偏差/频率偏差
       - 是否需要加装滤波装置
    
    4. 保护配置合理性
       - 保护是否完善(防孤岛/逆流/LVRT)
       - 定值是否与系统配合
    
    5. 通信方案
       - 远动信息是否完整
       - 通道是否可靠
    
    6. 关口计量
       - 计量点设置是否正确
       - 表计精度是否满足
```

### 5.1 消防设计审查

```
消防设计审查流程:
  1. 编制消防设计文件
     - 消防设计专篇
     - 消防设施平面布置图
     - 消防系统图
  
  2. 提交消防审查申请

  5. 获取消防审查合格意见书

    - 防火间距是否符合GB 50016
    - 灭火系统设计是否符合GB 50370/GB 50067
    - 火灾报警系统是否符合GB 50116
    - 疏散通道/安全出口是否符合
    - 消防给水是否符合GB 50974
    - 电池舱消防是否有热失控应对措施
```

## 六、安评(安全评价)

### 6.1 安评要求

```
安全评价要求:
  适用范围:
    - 装机≥10MW的储能电站: 必须做安评
    - 装机<10MW: 部分地区要求, 部分地区不要求
  
  安评类别:
    - 安全预评价(设计阶段): 识别危险源, 提出安全措施
    - 安全验收评价(投运前): 验证安全措施有效性
  
  安评内容:
    1. 危险源辨识
       - 电池热失控(最大危险源)
       - 电气短路/过流
       - 变压器火灾
       - 触电危险
       - 机械伤害
    
    2. 风险评估
       - 故障树分析(FTA)
       - 事件树分析(ETA)
       - LEC法(L=可能性, E=暴露频率, C=后果)
    
    3. 安全措施
       - 工程措施: 消防/温控/保护/绝缘
       - 管理措施: 运维制度/培训/应急
       - 个体防护: PPE配置
```

### 6.2 安评报告内容

```
安全评价报告结构:
  一、编制依据
  二、项目概况
  三、危险有害因素辨识与分析
    3.1 电池系统(热失控/过充/过放/短路/泄漏)
    3.2 电气系统(短路/过流/过压/触电/弧光)
    3.3 变压器(火灾/爆炸/油泄漏)
    3.4 消防系统(失效/误动)
    3.5 自然灾害(雷击/洪水/地震)
  四、评价单元划分
  五、定性定量评价
    5.1 预先危险性分析(PHA)
    5.2 故障树分析(FTA)
    5.3 风险矩阵
  六、安全对策措施
  七、事故应急救援预案
  八、评价结论
  
  编制单位资质:
    - 需持安全评价资质(应急管理局颁发)
    - 费用: 5~15万元
    - 周期: 2~4周
```

### 7.1 合规流程依赖图

```

  项目立项
    │
    ▼
  发改委备案 ←── (项目建议书)

    ├──→ 环评 ←── (备案证明)
    │      │

    │      │

    │      │

    │      │

    │
```

### 7.2 并行可压缩环节

```
可并行环节(缩短总周期):
  环评、安评、能评可并行开展(均以备案为前提)
  
  优化前(串行):
    备案(2周) → 环评(8周) → 安评(4周) → 能评(4周) = 18周
  
  优化后(并行):
    备案(2周) → [环评(8周) ∥ 安评(4周) ∥ 能评(4周)] = 10周
    (取最长: 环评8周)
  
  总周期压缩: 18周 → 10周, 节省8周

  并网批复可与安评/能评并行(部分供电公司允许)
  
  最优路径:
    备案(2周) → [环评(8周) ∥ 安评(4周) ∥ 能评(4周)] 
    → 并网批复(8周) → 消防审查(4周) → 施工许可(2周)
    = 24周(约6个月)
```

### 8.2 费用估算代码

```python
def compliance_cost_estimate(
    project_mw: float,
    project_mwh: float,
    region: str = "default",
    need_eia_book: bool = False,
    need_safety_eval: bool = False
):
    """合规费用估算"""
    
    base_costs = {
        "发改委备案": 0,
        "环评编制": 5 if not need_eia_book else 15,
        "环评评审": 0,
        "安评编制": 8 if need_safety_eval else 0,
        "能评编制": 3,
        "接入方案编制": 10 if project_mw <= 10 else 20,
        "并网审查": 0,
        "消防审查": 1,
        "施工图审查": 2,
        "施工许可": 0,
        "并网验收检测": 2,
    }
    
    total = sum(base_costs.values())
    
    region_factor = {
        "北京": 1.3, "上海": 1.3, "广东": 1.2,
        "浙江": 1.1, "江苏": 1.1, "default": 1.0,
        "内蒙古": 0.9, "新疆": 0.8
    }
    factor = region_factor.get(region, 1.0)
    
    result = {}
    for k, v in base_costs.items():
        result[k] = round(v * factor, 1)
    
    result["合计(万元)"] = round(total * factor, 1)
    result["项目规模"] = f"{project_mw}MW/{project_mwh}MWh"
    result["地区系数"] = factor
    
    return result

# 示例

    project_mw=5, project_mwh=10, 
    region="广东", need_safety_eval=True
)
```

### 8.3 各地差异要点

```
各地合规差异要点:
  国网覆盖区域:
    - 环评要求统一(生态环境部)
    - 并网审批流程统一(国网标准)
    - 消防审查按属地管理
  
  南网覆盖区域:
    - 广东: 环评要求较严, 消防审查严格
    - 广西: 流程相对简化
    - 云南/贵州: 部分项目可简化
  
  内蒙古(蒙西):
    - 新能源配储强制(10%~20%)
    - 环评可简化(荒漠地区)
    - 安评要求较严(化工园区附近)
  
  新疆:
    - 新能源基地配储鼓励
    - 流程可简化(非居民区)
    - 冬季施工受限
  
  西部地区:
    - 流程可能较长(审批能力有限)
    - 费用可能较低
    - 部分地区可享受政策优惠
```
