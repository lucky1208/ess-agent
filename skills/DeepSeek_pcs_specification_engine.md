## 四、LVRT低电压穿越曲线与无功电流公式

### 4.1 LVRT曲线 (GB/T 34120 §6.3)

```
电压(%Un)
   │
100├──────────────────────────────────────────────────────
   │
 90├─┬────────────────────────────────────────────────────
   │  ╲
   │   ╲________                                        电压恢复后:
 20├───────────╲________                                有功恢复≥30%Pn/s
   │                     ╲________                       无功电流继续维持
  0├──────────────────────────────────╲─────            ≥150ms
   │0   50  100  150  200  250  300  350  400  450  500 625
   │                                                      时间(ms)

穿越要求:
  ● U ≥ 90%Un → 持续并网, 无需穿越
  ● 20% ≤ U < 90%Un → 持续并网625ms
  ● U = 0% → 持续并网625ms (零电压穿越)
  ● 故障清除后3s内恢复至并网正常
  ● 期间有功功率可降为零, 无功优先
```

### 4.2 无功电流响应公式

> **核心公式**: $I_{react} \geq 2 \times (1 - U/U_{rated}) \times I_{rated}$, for $0 \leq U/U_{rated} \leq 0.85$

```
无功电流注入要求:
  故障期间: I_react ≥ 2 × (1 - U/Un) × In
  响应时间: ≤30ms (从电压跌落到无功电流输出)
  持续时间: 电压恢复后≥150ms继续维持

计算示例:
  电压跌落至 Un×0.50 (50%):
    I_react ≥ 2 × (1 - 0.50) × In = 1.0 × In    → 输出100%In的无功电流
  
  电压跌落至 Un×0.20 (20%):
    I_react ≥ 2 × (1 - 0.20) × In = 1.6 × In    → 输出160%In的无功电流 (最高2.0×In)

  零电压穿越 (0%):
    I_react ≥ 2 × (1 - 0) × In = 2.0 × In       → 输出200%In (最大限值)

LVRT期间有功/无功优先级:
  1. 优先保证无功电流输出 (满足公式要求)
  2. 剩余电流容量用于有功输出
  3. 总有功电流 + 无功电流 ≤ 1.2×In (器件限值)
```

### 4.3 HVRT (高电压穿越)

```
HVRT要求 (NB/T 31099):
  ● 110%Un → 持续运行 (不限时)
  ● 120%Un → 持续运行1s
  ● 130%Un → 持续运行0.5s
  ● 电压恢复后3s内恢复正常运行
  ● 期间有功功率可降至50%
  ● 具备无功吸收能力 (感性), 维持电压稳定
```

## 五、30项型式试验详解

> 按A~G共7大类30项, 每项含试验方法+合格标准

### A. 并网性能试验 (5项)

| 编号 | 试验项目 | 试验方法 | 合格标准 |
|------|----------|----------|----------|
| **A1** | LVRT | 可编程AC源注入0%~90%Un跌落625ms | 无脱网; 无功满足公式; 3s内恢复 |
| **A2** | HVRT | AC源注入110%/120%/130%Un | 按规定时间穿越, 无损坏 |
| **A3** | 孤岛保护 | 额定功率运行→断开PCC | 检测≤2s, 停止输出, 无设备损坏 |
| **A4** | 并/离网切换 | 并网→离网→并网循环 | 电压中断<10ms, Δf<±0.5Hz |
| **A5** | PF调节范围 | 额定功率下0.9滞后→1.0→0.9超前 | 全范围可调, 精度±0.01, 响应≤1s |

### B. 电能质量试验 (4项)

| **B1** | 电流谐波THDi | 额定PF=1, 测3~49次 | THDi≤5%, 各次≤限值×80% |
| **B2** | 电压谐波THDu | 同上 | THDu≤5% |
| **B3** | 电压波动闪变 | 10min Pst + 2h Plt测量 | Pst≤1.0, Plt≤0.8 |
| **B4** | 直流分量 | AC侧测DC偏移 | DC电流≤0.5%In, DC电压≤0.5%Un |

### C. 效率与功率试验 (4项)

| **C1** | 额定效率 | 25°C,PF=1,100%负载,0.5级功率计 | η≥98.0% (保障值) |
| **C2** | 效率曲线 | 20%~100%负载, 每点15min | 曲线平滑, 50%:≥98.5% |
| **C3** | 待机功耗 | 待机状态测30min功耗 | ≤Pn×0.3% |
| **C4** | PF-负载特性 | PF=0.9/1.0/-0.9, 25%~100% | 全范围精度±0.01 |

### D. 保护功能试验 (4项)

| **D1** | 过/欠压保护 | 逐步升/降压 | OU1:1.10Un/0.2s, OU2:1.15Un/0.1s |
| **D2** | 频率保护 | 逐步升/降频 | 过频51.5Hz/0.2s, 欠频47.5Hz/0.2s |
| **D3** | 过流/短路 | 1.2In/2In注入 | 过流1s降功, 短路0.05s速断 |
| **D4** | 直流过压/过流 | DC侧逐步升压/注入 | 过压0.5s, 过流0.1s, 停机 |

### E. 环境可靠性试验 (5项)

| **E1** | 高温工作 | 55°C下额定4h | 功率正常, 结温≤150°C |

## 十二、谐波滤波与电能质量治理

### 12.1 LCL滤波器设计参数

```
LCL滤波器参数计算:
  f_res = 1/(2π × √(Lr×Lg/L_total × C))
  
  设计目标: f_res在 10×f_grid < f_res < 0.5×f_switching
  例: f_grid=50Hz, f_sw=5kHz → f_res应在500Hz~2500Hz之间

  总电感压降: ≤8% Un (额定电流时)
  阻尼电阻 Rd: 2~5Ω (抑制谐振峰值)
```

## 十五、PCS黑启动与微电网模式

### 15.1 黑启动流程

```
黑启动 (Black Start) 条件:
  直流侧: 电池可用, SOC≥20%, BMS允许放电
  交流侧: 断路器断开(孤岛模式)
  控制电源: UPS/直流屏持续供电

启动流程:
  Step 1: PCS切换至V/F控制模式(电压源)
  Step 2: 软启动建立AC电压, 2s内达到额定电压
  Step 3: 频率稳定至50±0.2Hz
  Step 4: 逐步加载(首批负荷≤30%Pn)
  Step 5: 增加负荷至设计值
  Step 6: 监测电压/频率/谐波稳定

关键技术参数:
  电压建立时间: ≤5s (从指令到额定电压)
  频率精度: ±0.2Hz
  电压THDu: ≤10% (纯电阻负载) / ≤5% (额定负载)
  最大启动冲击电流: ≤2×In
```

## 十八、PCS技术发展趋势与标准前瞻

### 18.1 SiC器件替代IGBT

| 对比项 | IGBT (当前主流) | SiC MOSFET (下一代) |
|--------|----------------|---------------------|
| 开关频率 | 2~5kHz | 20~50kHz |
| 导通损耗 | 中 | 低(无尾电流) |
| 开关损耗 | 高(拖尾电流) | 极低 |
| 结温上限 | 150°C | 200°C+ |
| 效率提升 | 基准 | +1~2% |
| 功率密度 | 基准 | +50~100% |
| 成本 | 基准(2025) | 1.5~3× (2025), 预计2028达到1.2× |
| 适用 | 全系列 | 中高频>1500V应用 |

### 18.2 构网型(GFM) PCS趋势

```
Grid-Forming PCS (构网型变流器):

  构网型PCS: 自主建立电压频率, 不依赖电网

适用场景:

  ● 虚拟同步机(VSG)控制
  ● 惯量响应: 模拟同步发电机转动惯量
  ● 阻尼控制: 抑制功率振荡
  ● 自主频率/电压调节

标准进展:
  ● 国内: NB/T XXXXX-202X (制定中)

```
