## 一、并网接入流程总览

### 1.1 标准并网流程

```
储能电站并网接入全流程:

  ① 方案设计 ──→  ② 并网申请 ──→  ③ 接入系统方案审查 ──→  ④ 并网技术条件
       │                │                  │                      │
    委托设计院      向供电公司提交      供电公司组织审查        出具技术条件书
    出电气方案      并网申请材料        含接入点/容量/保护      约束设备参数
       │                │                  │                      │
       ▼                ▼                  ▼                      ▼
  ⑤ 调度协议 ──→  ⑥ 设备采购 ──→  ⑦ 施工建设 ──→  ⑧ 调试验收
       │                │                │               │
    签调度协议      按技术条件采购    含电气/通信/消防    分系统+整套启动
    明确远动/AGC    关口表指定品牌    监理+质检         供电公司参与
       │                │                │               │
       ▼                ▼                ▼               ▼
  ⑨ 并网验收 ──→  ⑩ 试运行 ──→  ⑪ 正式并网
       │                │              │
    供电公司验收     72h试运行      转商业运营
    30项逐项检查     各项指标合格    纳入调度管理

总周期: 3~6个月(小型工商业) / 6~12个月(大型独立储能)
```

### 1.2 各阶段关键节点与交付物

| 阶段 | 关键节点 | 交付物 | 责任方 | 周期 |
|------|---------|--------|-------|------|
| 方案设计 | 电气方案评审通过 | 接入方案报告 | 设计院 | 2~4周 |
| 并网申请 | 材料齐全受理 | 受理通知书 | 业主 | 1~2周 |
| 接入审查 | 审查会议通过 | 审查意见书 | 供电公司 | 4~8周 |
| 调度协议 | 协议签订 | 调度协议 | 业主+调度 | 2~4周 |
| 施工建设 | 施工完成 | 竣工报告 | 施工单位 | 8~16周 |
| 调试验收 | 验收通过 | 验收报告 | 调试单位 | 2~4周 |
| 试运行 | 72h合格 | 试运行报告 | 业主 | 1周 |
| 正式并网 | 调度下令并网 | 并网确认书 | 调度 | 1天 |

### 3.2 AGC(自动发电控制)

```
储能AGC功能要求:
  控制模式:
    - 恒功率模式: P_ref由调度下发, PCS跟踪执行
    - 恒频模式: 根据频偏自动调节有功(一次调频)
    - 调峰模式: 按调度计划曲线充放电
    - AGC模式: 响应调度AGC指令
  
  AGC响应性能:
    - 响应时间: ≤200ms (收到指令到功率变化起始)
    - 调节速率: ≥10%Pn/s (额定功率的10%/秒)
    - 调节精度: ≤2%Pn (稳态偏差)
    - 调节范围: -Pn ~ +Pn (充电~放电)
  
  AGC通信:
    - 协议: IEC 60870-5-104
    - 周期: 2~4s (调度下发周期)
    - 超时: 10s未收到指令→保持最后指令或转入本地模式
```

### 3.3 AVC(自动电压控制)

```
储能AVC功能要求:
  控制策略:
    - 电压控制: Q = f(U), 维持PCC电压在目标值
    - 无功控制: 按调度指令输出无功
    - 功率因数控制: 维持PCC功率因数在目标值
  
  AVC响应性能:
    - 响应时间: ≤100ms
    - 无功调节范围: -0.5Pn ~ +0.5Pn (cosφ≥0.95时)
    - 调节精度: ≤1%Qn
  
  典型调度无功指令:
    - 电压上限: 10.7kV (107%Un)
    - 电压下限: 9.3kV (93%Un)
    - 电压目标: 10.3kV (103%Un, 考虑线路压降)
```

### 3.4 频率响应

```
储能一次调频:
  死区: ±0.033Hz (50±0.033Hz, GB/T 36547)
  响应时间: ≤200ms
  调差系数: K = Δf/fn / ΔP/Pn = 3%~5% (可整定)
  
  示例(2500kW储能, K=3%):
    Δf = +0.1Hz → ΔP = -(0.1/50)/(3%) × 2500 = -166.7kW (减放电/增充电)
    Δf = -0.1Hz → ΔP = +166.7kW (增放电/减充电)

储能二次调频(AGC):
  由调度通过AGC指令下发
  调节范围: ±Pn
  响应时间: ≤30s (全范围)
```

## 四、并网技术要求

### 4.1 GB/T 36547-2024 储能系统接入要求

```
GB/T 36547主要要求:
  1. 接入电压等级:
     - 装机≤1MW: 接入380V/400V
     - 1MW<装机≤10MW: 接入10kV
     - 10MW<装机≤100MW: 接入35kV/110kV
     - 装机>100MW: 接入220kV及以上
  
  2. 电能质量:
     - 电压偏差: ±5%Un
     - 频率偏差: ±0.2Hz(正常)/±0.5Hz(异常)
     - 谐波电流: 满足GB/T 14549
     - 三相不平衡: ≤2%
  
  3. 功率控制:
     - 有功功率控制: 响应时间≤200ms
     - 无功功率控制: cosφ范围±0.95
     - 充放电转换: ≤1s
  
  4. 保护功能:
     - 防孤岛保护: 孤岛检测时间≤2s
     - 低电压穿越: 满足标准曲线
     - 频率保护: 48.5~51.5Hz
```

### 4.2 GB/T 34120-2024 变流器技术要求

```
GB/T 34120主要要求:
  1. 电气性能:
     - 额定效率: ≥97%(≥100kW)
     - 电流THD: ≤3%(额定)
     - 功率因数: -0.95~+0.95(额定)
  
  2. 保护功能:
     - 交流过流保护
     - 直流过压/欠压保护
     - 防孤岛保护
     - 低电压穿越
     - 过温保护
  
  3. 通信功能:
     - Modbus RTU/TCP
     - IEC 60870-5-104
     - 远程启停/功率调节
```

### 4.3 GB/T 34131-2024 BMS技术要求

```
GB/T 34131主要要求:
  1. 数据采集:
     - 单体电压采集精度: ±0.5%FS
     - 温度采集精度: ±1℃
     - 电流采集精度: ±0.5%FS
     - 采集周期: ≤1s
  
  2. 保护功能:
     - 三级保护(告警/降额/急停)
     - 过充/过放保护
     - 过温保护
     - 绝缘监测
     - 均衡功能
  
  3. SOC/SOH估算:
     - SOC精度: ±5%
     - SOH精度: ±5%
     - 在线估算, 无需离线标定
```

## 六、低电压穿越

### 6.1 LVRT要求(GB/T 36547)

```
低电压穿越曲线要求:
  
  电压(pu)  1.0 ─────────────╲
                        ╲
  0.9 ───────╲──────────╲──── 不脱网连续运行区
               ╲          ╲
  0.2 ──────────╲──────────╲──
                  │          │
  0.0 ───────────┼──────────┼──
                  0.625    2.0  时间(s)

  关键参数:
    - 0.9Un以上: 不脱网连续运行
    - 0.2Un: 至少支撑625ms不脱网
    - 0.2Un~0.9Un: 线性过渡, 2s内不脱网
    - 无功电流支撑: I_q ≥ 1.5 × (0.9 - U_k/Un) × I_n
      U_k: PCC点电压, I_n: 额定电流
```

### 6.2 LVRT动作逻辑

```
LVRT动作逻辑:
  1. 正常运行: U_pcc > 0.9Un, 正常有功/无功输出
  2. 电压跌落: U_pcc < 0.9Un
     → 闭锁防孤岛保护(防止误动)
     → 切除有功电流指令
     → 注入无功电流支撑: I_q = 1.5×(0.9-U_k/Un)×I_n
     → 优先无功支撑, 有功按比例降额
  3. 电压恢复: U_pcc > 0.9Un
     → 恢复有功输出(斜率≤10%Pn/s, 防止功率冲击)
     → 恢复防孤岛保护
  4. 超时跳闸: 跌落持续超过2s → 跳PCS
```

### 6.3 高电压穿越(HVRT)

```
高电压穿越要求:
  1.1Un < U_pcc ≤ 1.2Un: 连续运行≥2s
  1.2Un < U_pcc ≤ 1.3Un: 连续运行≥0.2s
  U_pcc > 1.3Un: 立即跳闸

  HVRT动作逻辑:
    - 过压1.1~1.2Un: 持续2s内不脱网, 吸收无功
    - 过压1.2~1.3Un: 持续0.2s内不脱网
    - 过压>1.3Un: 立即跳闸(保护动作)
```

```
  PT:
    - 精度: 0.2级(计量专用绕组)
    - 变比: 10kV/100V 或 35kV/100V
    - 容量: 30VA(典型)
    - 接线: Y/y接线, 中性点接地
    - 注意: 计量绕组独立, 不与保护共用
  
  综合误差要求:
    ε_total = ε_CT + ε_PT + ε_meter ≤ ±0.7%
    ε_CT: CT比差(0.2S级, ≤±0.2%@100%In)
    ε_PT: PT比差(0.2级, ≤±0.2%@80%Un~120%Un)
    ε_meter: 表计误差(0.2S级, ≤±0.2%@100%In)
```

## 九、各供电公司流程差异

### 9.1 国网(国家电网)

```
国网并网特点:
  1. 接入系统方案:
     - 由供电公司经研所/规划评审中心审查
     - 需提交潮流/短路/电能质量计算报告
     - 审查周期: 4~8周
  
  2. 关口计量:
     - 表计指定: 威胜/兰吉尔(国网集中采购)
     - 采集系统: 国网用电信息采集系统
     - CT精度: 0.2S级强制
  
  3. 调度管理:
     - 10kV接入: 地调管理
     - 35kV接入: 省调管理
     - 通信协议: IEC 104强制
  
  4. 特殊要求:
     - 储能纳入电力系统调峰调频辅助服务
     - 需参加电力市场注册(现货/调频)
     - 储能电站需配置AGC/AVC
```

### 9.2 南网(南方电网)

```
南网并网特点:
  1. 接入审查:
     - 由供电局系统运行部审查
     - 流程相对简化(工商业项目)
     - 审查周期: 3~6周
  
  2. 关口计量:
     - 表计指定: 兰吉尔/威胜
     - 采集系统: 南网计量自动化系统
     - 差异: 南网协议格式略有不同
  
  3. 特殊要求:
     - 储能可参与调频市场
     - 独立储能需与电网签订调度协议
     - 部分省份支持储能参与现货
```

### 9.3 蒙西电网

```
蒙西电网并网特点:
  1. 独立电网:
     - 不属于国网/南网, 独立调度
     - 并网审查流程自主制定
     - 周期可能较长(6~12周)
  
  2. 特殊政策:
     - 新能源配储要求: 按装机10%~20%配储
     - 储能参与调峰辅助服务
     - 火储联合调频模式
  
  3. 技术差异:
     - 频率偏差要求可能不同(电网较小)
     - LVRT曲线可能更严格
     - 通信协议需适配蒙西调度系统
```

### 9.4 地方电网

```
地方电网(增量配网/园区电网)并网特点:
  1. 流程灵活:
     - 审查流程可简化
     - 部分项目可备案后直接并网
     - 周期较短: 2~4周
  
  2. 技术要求:
     - 基本参照国网/南网标准
     - 部分指标可适当放宽
     - 关口计量可能要求较低(0.5S级)
  
  3. 注意事项:
     - 产权分界点需明确
     - 调度关系需厘清(地方调度/不调度)
     - 电费结算方式需约定(峰谷差/容量电价)
```
