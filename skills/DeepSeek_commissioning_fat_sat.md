### 1.2 验收阶段定义决策树

```
项目阶段？
  ├── 设备出厂前 → FAT (Factory Acceptance Test)
  │   ├── 地点：制造商工厂
  │   ├── 范围：单台设备/单舱体
  │   └── 目的：确认设备符合设计规格后发运
  │
  ├── 设备到场后 → Pre-Commissioning (安装前检查)
  │   ├── 外观检查、开箱检查
  │   └── 绝缘检查、安装条件确认
  │
  ├── 安装完成后 → Commissioning (调试)
  │   ├── 单机调试（单体设备上电、通信、功能确认）
  │   └── 分系统调试（BMS+PCS+消防联动）
  │
  ├── 并网前 → SAT (Site Acceptance Test)
  │   ├── 地点：项目现场
  │   ├── 范围：全系统联合验收
  │   └── 目的：确认系统满足合同和并网要求
  │
  └── 并网后 → PAC / FAC (预/最终验收)
      ├── PAC: 试运行168h后签发
      └── FAC: 质保期结束后签发
```

### 1.3 验收等级与判定标准

| 验收项分类 | 等级 | 判定标准 | 不合格处置 |
|-----------|------|---------|-----------|
| **安全关键项** | A级 | 必须100%合格,一票否决 | 整改至合格,不得让步接收 |
| **功能性能项** | B级 | ≥95%合格,每项偏差≤5% | 限期整改,可签条件性PAC |
| **一般检查项** | C级 | ≥90%合格,可附备注 | 记录缺陷,约定期限整改 |
| **文档资料项** | D级 | 全部齐全 | 补充完成后签字 |

## 二、FAT工厂验收测试完整规范

### 2.1 FAT整体流程

```
Step 1: 测试准备 (Day -7)
   ├── 确认出厂检验大纲（ITP）
   ├── 准备测试仪器（已校准、在有效期内）
   ├── 准备测试记录表（纸质+电子双备份）
   └── 召集测试参与方（供应商、业主/监理）

Step 2: 外观与结构检查 (Day 1 上午)
   ├── 设备外观、铭牌、标识检查
   ├── 尺寸、重量、接口位置核对
   └── 防护等级确认（IP等级、防腐等级）

Step 3: 布线/电气连接检查 (Day 1 下午)
   ├── 线缆走向和固定检查
   ├── 端子紧固力矩抽查（10%端子）
   └── 绝缘电阻测试+接地连续性测试

Step 4: 上电检查 (Day 2)
Step 5: 通信功能测试 (Day 2-3)
Step 6: 保护功能测试 (Day 3-4)
Step 7: 性能测试 (Day 4-5)
Step 8: 出厂报告与签字 (Day 5)
```

## 三、SAT现场验收测试完整规范

### 3.1 SAT整体流程

```
Step 1: 安装性检查 (Pre-Commissioning)
Step 2: 单体设备调试
Step 3: 系统联调
Step 4: 保护联动测试
Step 5: 并网测试
Step 6: 性能测试
Step 7: 连续试运行 (168h)
Step 8: 验收签字
```

### 3.2 SAT安装性检查

#### 接地系统验收

| 序号 | 检查项 | 检查内容 | 判定标准 |
|------|--------|---------|---------|
| 1 | 接地电阻 | 接地网综合电阻 | ≤4Ω(综合), ≤1Ω(BESS推荐) |
| 2 | 接地干线 | 截面积/材质 | ≥50mm²铜绞线或40×4扁钢 |
| 3 | 设备接地 | 每台设备外壳接地点 | ≥2个独立接地点 |
| 4 | 等电位连接 | 柜体间等电位线 | ≥16mm²铜线 |
| 5 | 防雷接地 | 避雷针/接闪器安装 | 保护范围覆盖全部设备 |

### 3.3 SAT并网测试规范

#### 电能质量测试(在PCC点测量)

| 测试项 | 测试条件 | 标准限值 | 测试方法 |
|--------|---------|---------|---------|
| 电压偏差 | 不同功率点 | ±7%Un(10kV), ±5%Un(0.4kV) | 电能质量分析仪连续10min |
| 频率偏差 | 并网运行 | ±0.5Hz | 同电压测量 |
| 电压谐波THDu | 100%功率 | ≤5% | 谐波分析取95%概率大值 |
| 电流谐波THDi | 100%功率 | ≤5% | 谐波分析取95%概率大值 |
| 三相不平衡度 | 任何功率点 | ≤2%(短时), ≤1.3%(长时) | 负序/正序电压 |
| 直流分量 | 100%功率放电 | ≤0.5%In | 直流CT或示波器 |
| 功率因数 | 各功率点 | ≥0.95 | 功率分析仪 |

### 3.4 SAT消防联动停机测试(最重要A级安全项)

```
【验收标准】
  消防告警→直流断路器分闸完成: ≤1s
  消防告警→交流断路器分闸完成: ≤1s
  消防告警→PCS停机完成:      ≤500ms
  消防告警→BMS紧急告警置位:   ≤500ms
  所有停机回路:              硬接线动作, 不得依赖软件/通信

【必须验证】:
  □ 消防停机回路是否为硬接线(从消防主机干接点→安全继电器→分闸线圈)
  □ 通信中断情况下, 手动触发消防告警, 系统能否执行紧急停机
  □ 消防停机后, 是否禁止自动重合闸(需人工复位确认)
```

### 3.5 防孤岛保护测试

```
【RLC谐振法测试】
  1. 配置RLC并联负载模拟本地负荷(Q=1谐振条件)
  2. 断开PCC断路器(模拟电网停电)
  3. 记录PCS检测到孤岛的时间和停机时间

【判定标准】:
  - 孤岛检测时间 ≤ 2s
  - PCS必须在检测到孤岛后200ms内停止输出
  - PCS停机后必须锁定(禁止自动重启)
  测试点: 25%, 50%, 75%, 100%额定功率各测1次
```

## 四、二次回路完整性检查(160项清单)

### 4.1 检查流程

```
检查准备 → 逐回路核对接线 → 测量通断/绝缘 → 记录实测值 → 不合格处理 → 签字归档
```

**通用检查方法**:
- **通断测试**: 万用表蜂鸣档, 线间电阻≤1Ω合格
- **绝缘测试**: 500V摇表, 绝缘电阻≥10MΩ(潮湿≥1MΩ)
- **电压测试**: 万用表/示波器, 带电测量电压幅值及波形
- **信号注入**: 保护测试仪或信号发生器注入标准信号

### 4.2 BMS→PCS CAN回路检查(序号1-18)

| 序号 | 检查项目 | 检查方法 | 标准值 |
|------|---------|---------|--------|
| 1 | CAN_H接线正确性 | 测量BMS端CAN_H与PCS端CAN_H导通 | 导通,压接牢靠 |
| 2 | CAN_L接线正确性 | 测量BMS端CAN_L与PCS端CAN_L导通 | 导通,压接牢靠 |
| 3 | CAN_H/CAN_L极性正确 | 示波器观察CAN差分波形 | 眼图无变形,差分1.5~3.5V |
| 4 | 终端电阻匹配 | 万用表测量120Ω终端电阻 | 末端120Ω±5% |
| 5 | CAN总线波特率 | CAN分析仪查看实际波特率 | 与设计值一致 |
| 6 | BMS心跳帧周期 | CAN分析仪测量 | 100ms±10ms |
| 7 | BMS最高SOC告警帧 | 注入SOC=100%模拟 | PCS限功率至0 |
| 8 | BMS最低SOC告警帧 | 注入SOC=0%模拟 | PCS限功率至0 |
| 9 | BMS急停指令 | 人为触发BMS急停 | PCS PWM封锁≤50ms |
| 10 | BMS通信超时保护 | 断开CAN总线 | 超时3s内进入保护 |
| 11 | CAN屏蔽层接地 | 测量屏蔽层与接地排导通 | 单点接地,电阻≤4Ω |

## 五、继电保护定值校验

### 5.1 变压器差动保护定值计算

**已知条件**(10MVA, 35/10.5kV):
- CT变比: 高压侧300/5=60, 低压侧1000/5=200
- 最大穿越电流: 6kA(低压侧三相短路)

**计算步骤**:

高压侧额定电流:
$$I_{NH} = \frac{10\times10^6}{\sqrt{3}\times35\times10^3} = 164.96\text{A}$$

低压侧额定电流:
$$I_{NL} = \frac{10\times10^6}{\sqrt{3}\times10.5\times10^3} = 549.86\text{A}$$

最小动作电流:
$$I_{op.min} = 1.3\times(0.1\times197.95 + 0.05\times197.95) = 38.60\text{A}$$
折算到高压侧CT一次值: 38.60/60 = 0.643A → 设置**0.65A**

**实际配置值**:

| 参数 | 配置值 | 说明 |
|------|--------|------|
| 差动门坎电流I_op.min | 0.5A | 高压侧CT一次值归算 |
| 比率制动斜率K1 | 0.3 | 低电流区段 |
| 比率制动斜率K2 | 0.5 | 高电流区段 |
| 拐点电流I_res.0 | 1.64A | 归算至高压侧 |
| 二次谐波制动比 | 15% | 励磁涌流制动 |
| 速断电流I_sb | 5.0×I_N | 引出线故障 |

### 5.2 出线过流保护三段式定值配合

**已知条件**(10kV线路):
- 最大负载电流: I_L.max=200A
- 末端最大三相短路: I_k.max=3000A
- 最小两相短路: I_k.min=1200A
- 时间级差: Δt=0.3s

**一段(电流速断)**:
$$I_{set.I} = 1.3/0.85\times3000 = 4588\text{A} → 取4500\text{A}$$
动作时间: 0s(瞬时)。灵敏度K_sen=1200/4500=0.27<1.2(灵敏度不足,一段仅保护60%~70%)

**二段(限时速断)**:
$$I_{set.II} = 1.1\times2500 = 2750\text{A} → 取2800\text{A}$$
动作时间: 0+0.3=0.3s

**三段(定时限过流)**:
$$I_{set.III} = 1.2/0.85\times200 = 282.4\text{A} → 取300\text{A}$$
灵敏度: 1200/300=4.0≥1.5 ✓

4. 记录从报警触发到PWM封锁的时间差
5. 重复测试3次, 取最大值

**判定标准**:

4. 记录从保护动作到辅助触点变位的时间差

**判定标准**: ≤2s合格, 2~3s降级, >3s不合格

#### TS-03:EMS功率指令→PCS响应时序测试

3. 下发阶跃功率指令
4. 记录指令发出到功率开始变化的时间

**判定标准**: ≤1s

```
  - CH1(消防急停继电器): 电压下降→低于1V→触发
  - 或 外部触发输入: 使用保护装置"保护动作"空接点输出

录波长度:
  - 预触发: 100ms
  - 总录波: 2000ms
  - 采样率: 10kHz

记录文件:
  - 格式: COMTRADE(*.cfg + *.dat)
  - 命名: TS01_FIRE_PCS_YYYYMMDD_HHMMSS
```

### 9.4 录波分析要点

录波图中应清晰识别以下事件时序:
1. **T0**: 触发事件(消防急停继电器动作/保护动作)
2. **T1**: BMS收到急停信号(如CAN总线停止发送数据)
3. **T2**: PCS PWM封锁边沿时刻
4. **T3**: 断路器跳闸线圈得电时刻
5. **T4**: 断路器完全分断(辅助触点变位)时刻

## 十、性能测试规范

### 10.1 充放电效率测试

```
【测试方法-全功率点法】
  1. 在25%, 50%, 75%, 100%额定功率点各执行1次完整充放电循环
  2. 使用0.2级功率分析仪在交流侧和直流侧独立测量
  3. 加权效率: Σ(效率_i×功率权重_i)

【判定标准】:
  - PCS最大效率: ≥98.5%
  - PCS加权效率: ≥98%
  - 系统综合效率: ≥85%(含辅助功耗)
```

### 10.2 响应时间测试

```
【有功功率响应时间】:
  - 0→50%Pn阶跃, 记录到90%目标时间
  - 判定: ≤500ms(一次调频), ≤2s(AGC调节)
  - 超调量: ≤10%Pn
  - 稳态误差: ≤±2%Pn

【充放电切换响应时间】:
  - 50%Pn充电→50%Pn放电
  - 判定: ≤200ms

【无功功率响应时间】:
  - PF=1→PF=0.9切换
  - 判定: ≤1s
```

### 10.3 容量测试

```
【标准容量测试法】:
  1. C/3倍率充满至SOC=100%
  2. 静置30min
  3. C/2倍率恒功率放电至SOC=0%
  4. 直流侧功率分析仪积分计算放电总能量

【判定标准】:
  电池容量偏差: 实测≥标称值的100%(GB/T 36276)
  或按合同约定: 实测≥95%为可接收

【温度修正】:
  C_25°C = C_meas / [1 + α(T-25)]
  α: 温度系数(LFP约0.003/°C)
```

### 10.4 SOC精度测试

```
【测试方法】:
  1. PCS直流侧安装0.1级电流传感器+能量积分仪
  2. 执行完整充放电循环
  3. 在SOC=20%, 50%, 80%点比较BMS_SOC与SOC_ref

【判定标准】:
  - SOC偏差: ≤±5%(全范围)
  - SOC=20%~80%区间: ≤±3%(GB/T 34131)
```

## 十一、验收决策树与判定标准

### 11.1 验收决策流程

```
测试全部完成?
  ├── A级项(安全关键)100%合格? ──否──→ 【不通过】停止验收, 强制整改
  │
  ├── B级项(功能性能)≥95%合格? ──否──→ 【条件性通过】限期整改
  │
  ├── 连续试运行168h无故障? ──否──→ 【延长试运行】重新计时
  │
  └── 文档资料齐全?
      ├── 否 → 补充后签字
      └── 是 → 【通过】签署PAC
```

### 11.2 关键验收判定项

**一票否决项(A级, 任意一项不合格则验收不通过)**:
- [ ] 消防联动硬接线回路完整且功能正常
- [ ] 防孤岛保护配置正确并验证通过
- [ ] 直流侧熔断器/断路器配置正确
- [ ] ESD紧急停机回路独立且功能正常
- [ ] 绝缘电阻测试全部通过
- [ ] 接地电阻测试达标
- [ ] 并网点电能质量达标

## 十三、典型问题与处置

### 13.1 FAT常见问题

| 问题 | 可能原因 | 处置 |
|------|---------|------|
| 绝缘电阻不达标 | 潮湿/污染/绝缘损坏 | 烘干清洁或更换 |
| 通信超时 | CAN终端电阻缺失/线缆过长 | 加终端电阻/缩短线缆 |
| 保护动作时间超标 | 继电器响应慢/逻辑延时 | 检查继电器/优化逻辑 |
| SOC显示偏差大 | 未校准/BMS算法参数不对 | 重新校准/调整参数 |
| 充放电效率低 | PCS效率低/线缆压降大 | 检查PCS/增加线缆截面积 |

### 13.2 SAT常见问题

| 问题 | 可能原因 | 处置 |
|------|---------|------|
| 并网点电能质量不达标 | PCS谐波大/滤波不足 | 检查PCS/增加滤波器 |
| 防孤岛检测失败 | RLC匹配不当/保护参数不对 | 调整RLC/检查定值 |
| 消防联动延迟超时 | 继电器响应慢/线缆压降 | 更换继电器/增加截面积 |
| 通信中断不能自动恢复 | 冗余配置缺失 | 配置通信冗余 |
| 接地电阻超标 | 接地极不足/土壤电阻率高 | 增加接地极/使用降阻剂 |
