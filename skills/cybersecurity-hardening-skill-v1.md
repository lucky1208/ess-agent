# 网络安全加固 Skill v1.0

> 设计人：卢继雄 | 基于5年微网项目实战训练

## 概述
自动检查并加固站端EMS的SSH、证书、端口、VPN、日志审计、账号权限。

## 输入
- 目标服务器 (IP/SSH)
- 当前配置快照

## 输出
- 安全检查报告: 每项状态(合规/不合规)+风险等级
- 加固操作清单: 具体命令/配置
- 加固后验证结果

## 规则

### R1: SSH加固
- 禁用root远程登录
- 禁用密码认证, 仅允许密钥认证
- 修改默认端口22→随机高位端口
- 设置MaxAuthTries=3
- 设置LoginGraceTime=30

### R2: TLS/SSL证书
- EMQX启用TLS(mqtts://:8883)
- API启用HTTPS(443)
- 证书: 使用Let's Encrypt或自签CA
- 证书有效期检查: >30天

### R3: 端口最小化
- 关闭非必需端口
- 仅开放: SSH/HTTPS/MQTT(TLS)/IEC104
- 防火墙: ufw/iptables规则

### R4: VPN
- 站端到云: WireGuard/IPSec VPN
- 远程运维必须走VPN
- 禁止直接公网SSH

### R5: 日志审计
- 启用auditd: 记录所有sudo/SSH登录/文件修改
- 日志保留: ≥180天
- 日志转发: 到远程syslog服务器

### R6: 账号权限
- 最小权限原则: 每服务独立用户
- 禁用默认账号/弱密码
- 定期轮换密钥(90天)

## 验证Checklist
- [ ] SSH仅密钥认证
- [ ] TLS证书有效
- [ ] 非必需端口关闭
- [ ] VPN通道可用
- [ ] 审计日志开启
- [ ] 无弱密码/默认账号
