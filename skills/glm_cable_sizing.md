## 一、电缆选型三要素

### 1.1 选型原则

```
电缆选型三大要素(缺一不可):
  ┌───────────────────────────────────────────────────┐
  │  ① 载流量校验: I_z ≥ I_working                     │
  │     确保电缆在允许温升下安全载流                     │
  │                                                     │
  │  ② 压降校验: ΔU% ≤ ΔU%_allow                       │
  │     确保末端设备电压在允许范围内                     │
  │                                                     │
  │  ③ 短路热稳定校验: S ≥ S_min = I_k×√t_k/C          │
  │     确保电缆能承受短路电流的热冲击                   │
  │                                                     │
  │  最终截面 = max(截面_载流量, 截面_压降, 截面_热稳定)  │
  └───────────────────────────────────────────────────┘
```

### 1.2 电缆类型选择

```
电缆类型选择原则:
  按电压等级:
```

## 二、载流量计算

### 2.1 基本计算公式

```
载流量计算公式:
  I_z = K_t × K_g × K_n × I_table

  I_table: 电缆标准载流量(A)
    查表依据: GB 50217附录C/D
    基准条件: 空气中30℃ / 埋地25℃ / XLPE 90℃
  
  K_t: 温度修正系数
    K_t = √((θ_max - θ_amb) / (θ_max - θ_ref))
    θ_max: 导体最高允许温度
      XLPE: 90℃
      PVC: 70℃
    θ_amb: 实际环境温度
    θ_ref: 基准环境温度(空气中30℃, 埋地25℃)
  
  K_g: 并列修正系数
    与电缆间距和并列根数有关
    间距≥2倍外径: K_g ≈ 1.0
    间距=1倍外径: K_g = 0.85~0.95
    无间距(紧贴): K_g = 0.70~0.85
  
  K_n: 敷设方式修正系数
    桥架(单层): K_n = 1.0
    桥架(多层): K_n = 0.65~0.80
    穿管: K_n = 0.85~0.90
    电缆沟: K_n = 0.90~0.95
    直埋: K_n = 1.0(基准)
```

## 三、电压降计算

### 3.1 计算公式

```
电压降计算公式:
  
  三相交流线路:
    ΔU% = (√3 × I × L × (R×cosφ + X×sinφ)) / (10 × U_N) × 100%
    
    或用功率表示:
    ΔU% = (P×R + Q×X) / (10 × U_N²) × 100%
  
  直流线路:
    ΔU% = (2 × I × L × R) / (10 × U_N) × 100%
  
  参数说明:
    I: 工作电流(A)
    L: 电缆长度(km)
    R: 交流电阻(Ω/km)
    X: 电抗(Ω/km)
    cosφ: 功率因数
    U_N: 额定电压(V)
    P: 有功功率(kW)
    Q: 无功功率(kVar)
```

## 四、短路热稳定校验

### 4.1 热稳定计算公式

```
短路热稳定校验公式:
  S_min = I_k × √t_k / C

  S_min: 最小允许截面(mm²)
  I_k: 短路电流有效值(A)
  t_k: 短路持续时间(s)
    = 继保动作时间 + 断路器全分闸时间 + 非周期分量等效时间
    典型值: 0.5s(快速保护) / 1.0s(一般保护) / 2.0s(后备保护)
  C: 热稳定系数
```

### 4.2 热稳定系数C值

```
热稳定系数C值:
  ┌──────────────────────────┬──────────┐
  │  电缆类型                  │  C值      │
  ├──────────────────────────┼──────────┤
  │  铜芯XLPE(90℃)           │  142      │
  │  铜芯PVC(70℃)            │  115      │
  │  铜芯橡胶(60℃)           │  100      │
  │  铝芯XLPE(90℃)           │  94       │
  │  铝芯PVC(70℃)            │  76       │
  └──────────────────────────┴──────────┘
```

### 4.3 短路电流计算

```
短路电流计算:
  三相短路电流:
    I_d3 = U_avg / (√3 × Z_Σ)
    U_avg: 平均电压(10kV系统取10.5kV)
    Z_Σ: 短路阻抗(系统阻抗+线路阻抗+变压器阻抗)
  
  单相短路电流(中性点接地系统):
    I_d1 = 3 × U_φ / (Z_1 + Z_2 + Z_0)
  
  示例(10kV系统):
    系统阻抗: Z_s = 0.5Ω (最大方式)
    变压器阻抗: Z_t = U_k% × U²_N / S_N = 6% × 10.5² / 3.15 = 2.1Ω
    
    PCC点短路电流:
    I_d3 = 10500 / (√3 × 0.5) = 12.12kA
    
    变压器低压侧短路电流(折算到高压侧):
    I_d3 = 10500 / (√3 × (0.5 + 2.1)) = 2.33kA
    
    电缆热稳定校验(t_k=0.5s, 铜芯XLPE):
    S_min = 12120 × √0.5 / 142 = 60.3mm² → 选用70mm²以上
```

```
桥架类型选择:
  ┌──────────────┬──────────────────────────────────────┐
  │  类型          │  特点                                  │
  ├──────────────┼──────────────────────────────────────┤
  │  梯式桥架      │  散热好, 重量轻, 成本低, 适用于动力电缆  │

  └──────────────┴──────────────────────────────────────┘
  
  材质选择:
    - 镀锌钢板: 常用, 成本适中, 防腐好
    - 铝合金: 轻量, 耐腐蚀, 成本高
    - 不锈钢: 腐蚀环境, 成本最高
    - 玻璃钢(FRP): 耐腐蚀, 绝缘, 不接地
```

### 6.2 桥架设计要求

```
桥架设计要求:
  填充率:
    - 动力电缆: ≤40% (按电缆截面积总和/桥架有效截面积)
    - 控制电缆: ≤50%
  
  宽度选择:
    - 按电缆排列宽度+20%裕度
    - 标准宽度: 200/300/400/500/600/800/1000mm
  
  层间距离:
    - 动力电缆桥架: ≥300mm
    - 控制电缆桥架: ≥200mm
    - 动力与控制之间: ≥500mm(或隔板分隔)
  
  接地:
    - 桥架全长电气连续
    - 接地线: 镀锌扁钢40×4 或 BV-16mm²
    - 接地点: ≥2处(两端), 长距离每30m一处
    - 接地电阻: ≤4Ω
```

### 8.2 电缆选型综合计算代码

```python
def cable_sizing_comprehensive(
    cable_name: str,
    working_current_a: float,
    cable_length_m: float,
    voltage_kv: float,
    cos_phi: float = 0.95,
    short_circuit_ka: float = 20,
    fault_time_s: float = 0.5,
    ambient_temp_c: float = 40,
    max_conductor_temp_c: float = 90,
    reference_temp_c: float = 30,
    parallel_group_factor: float = 0.85,
    laying_method_factor: float = 1.0,
    allow_vdrop_pct: float = 5.0,
    cable_type: str = "YJV",
    is_armored: bool = False,
    num_cores: int = 3
):
    """电缆选型综合计算(三要素)"""
    import math
    
    # 温度修正系数
    K_t = math.sqrt((max_conductor_temp_c - ambient_temp_c) / 
                     (max_conductor_temp_c - reference_temp_c))
    K_g = parallel_group_factor
    K_n = laying_method_factor
    
    # 需要的载流量
    I_required = working_current_a / (K_t * K_g * K_n)
    
    # 标准载流量表
    if voltage_kv >= 6:
        itable = {25:100, 35:125, 50:155, 70:195, 95:240, 
                  120:280, 150:320, 185:370, 240:450, 300:520, 400:605}
        rtable = {25:0.727, 35:0.524, 50:0.387, 70:0.268, 95:0.193,
                  120:0.153, 150:0.124, 185:0.099, 240:0.075, 300:0.060, 400:0.047}
        xtable = {25:0.098, 35:0.094, 50:0.089, 70:0.085, 95:0.081,
                  120:0.079, 150:0.077, 185:0.075, 240:0.073, 300:0.071, 400:0.070}
    else:
        itable = {10:60, 16:77, 25:100, 35:121, 50:146, 70:181, 
                  95:219, 120:254, 150:287, 185:326, 240:387, 300:438, 400:508}
        rtable = {10:1.830, 16:1.150, 25:0.727, 35:0.524, 50:0.387, 70:0.268,
                  95:0.193, 120:0.153, 150:0.124, 185:0.099, 240:0.075, 300:0.060, 400:0.047}
        xtable = {10:0.082, 16:0.078, 25:0.066, 35:0.064, 50:0.062, 70:0.060,
                  95:0.058, 120:0.056, 150:0.055, 185:0.054, 240:0.053, 300:0.052, 400:0.052}
    
    # 按载流量选截面
    cs_by_current = None
    for cs in sorted(itable.keys()):
        if itable[cs] >= I_required:
            cs_by_current = cs
            break
    
    # 短路热稳定校验
    C = 142 if max_conductor_temp_c >= 85 else 115
    S_min = short_circuit_ka * 1000 * math.sqrt(fault_time_s) / C
    cs_by_thermal = None
    for cs in sorted(itable.keys()):
        if cs >= S_min:
            cs_by_thermal = cs
            break
    
    # 取大值
    cs_selected = max(cs_by_current, cs_by_thermal)
    
    # 电压降校验
    R = rtable.get(cs_selected, 0.05)
    X = xtable.get(cs_selected, 0.07)
    sin_phi = math.sqrt(1 - cos_phi**2)
    L_km = cable_length_m / 1000
    vdrop_pct = (math.sqrt(3) * working_current_a * L_km * 
                 (R * cos_phi + X * sin_phi)) / (10 * voltage_kv * 1000) * 100
    
    if vdrop_pct > allow_vdrop_pct:
        for cs in sorted(itable.keys()):
            if cs > cs_selected:
                R = rtable.get(cs, 0.05)
                X = xtable.get(cs, 0.07)
                vdrop = (math.sqrt(3) * working_current_a * L_km * 
                         (R * cos_phi + X * sin_phi)) / (10 * voltage_kv * 1000) * 100
                if vdrop <= allow_vdrop_pct:
                    cs_selected = cs
                    vdrop_pct = vdrop
                    break
    
    # 电缆型号
    prefix = "NH-" if "NH" in cable_type else ""
    suffix = "22" if is_armored else ""
    model = f"{prefix}YJV{suffix}-{voltage_kv}kV" if voltage_kv >= 6 else f"{prefix}YJV{suffix}-0.6/1kV"
    
    # 价格估算
    price_per_m = cs_selected * 0.55 + 10
    
    return {
        "电缆编号": cable_name,
        "工作电流(A)": working_current_a,
        "电缆长度(m)": cable_length_m,
        "电压等级(kV)": voltage_kv,
        "K_t": round(K_t, 3),
        "K_g": K_g,
        "所需载流量(A)": round(I_required, 1),
        "按载流量选截面(mm²)": cs_by_current,
        "热稳定最小截面(mm²)": round(S_min, 1),
        "按热稳定选截面(mm²)": cs_by_thermal,
        "最终截面(mm²)": cs_selected,
        "电缆型号": model,
        "电缆规格": f"{num_cores}×{cs_selected}mm²",
        "载流量(A)": itable.get(cs_selected, "N/A"),
        "载流量裕度(%)": round((itable.get(cs_selected, 0) * K_t * K_g * K_n / working_current_a - 1) * 100, 1),
        "电压降(%)": round(vdrop_pct, 3),
        "电压降校验": "合格" if vdrop_pct <= allow_vdrop_pct else "不合格",
        "热稳定校验": "合格",
        "估算单价(元/m)": round(price_per_m, 1),
        "估算总价(元)": round(price_per_m * cable_length_m, 0)
    }

# 示例

    cable_length_m=200,
    voltage_kv=10,
    cos_phi=0.95,

)
```
