# Skill: 调试验收二次回路与联动（minimax_net_commissioning_iocontrol）

> **版本**: v1.0
> **适用场景**: 储能电站微网项目现场电气二次回路接线检查、继电保护校验、联动逻辑时序测试、并网前整组传动试验
> **适用人员**: 电气调试工程师、继电保护调试人员、项目经理、监理工程师
> **前置条件**: 设备安装就位、一次设备耐压试验通过、现场安全交底完成、调试仪器校准有效期内
> **输出**: 调试验收报告、继保定值核查表、联动测试记录表

---

## 模块一：二次回路完整性检查（≥160项清单）

### 1.1 检查流程总览

```
检查准备 → 逐回路核对接线 → 测量通断/绝缘 → 记录实测值 → 不合格处理 → 签字归档
```

**通用检查方法说明**：
- **通断测试**：使用万用表蜂鸣档或回路电阻测试仪，线间电阻≤1Ω为合格
- **绝缘测试**：使用500V摇表，绝缘电阻≥10MΩ（潮湿环境≥1MΩ）
- **电压测试**：使用万用表或示波器，带电测量电压幅值及波形
- **信号注入**：使用继电保护测试仪或信号发生器注入标准信号，验证接收端响应

**不合格处理原则**：
- 立即停用该回路，标识缺陷标签
- 填写《缺陷记录单》，拍照留证
- 通知施工方48小时内整改，整改后重新验收
- 重大隐患（如保护回路失效）立即上报项目经理

## 模块二：继电保护定值校验

### 2.1 变压器差动保护定值计算

#### 2.1.1 保护原理与整定原则

变压器差动保护基于**基尔霍夫电流定律**，正常运行或外部故障时，流入变压器的电流代数和为零（忽略励磁电流）。当内部故障时，该和不为零，差动保护动作。

**整定原则**：
1. 躲过变压器励磁涌流（采用二次谐波制动或波形识别制动）
2. 躲过变压器正常运行时的不平衡电流（采用比率制动特性）
3. 躲过外部故障时的最大穿越电流引起的不平衡电流
4. 保证变压器内部故障时（最小短路电流）有足够的灵敏度

#### 2.1.2 计算案例

**已知条件**：
- 主变压器：额定容量 $S_{N}=10\text{MVA}$，额定电压 $U_{N\text{/}H}=35\text{kV}/10.5\text{kV}$
- 最大负载电流：$I_{L\text{max}}=0.8\times I_{N}$（考虑过载系数1.2）
- 变压器接线组别：Yd11
- 差动保护CT变比：高压侧 $n_{TAH}=300/5=60$，低压侧 $n_{TAL}=1000/5=200$
- 最大穿越电流（低压侧三相短路）：$I_{k\text{max}}=6\text{kA}$

**步骤一：计算各侧额定电流**

高压侧额定电流：
$$I_{NH} = \frac{S_N}{\sqrt{3}\times U_{NH}} = \frac{10\times10^6}{\sqrt{3}\times35\times10^3} = 164.96\text{A}$$

低压侧额定电流：
$$I_{NL} = \frac{S_N}{\sqrt{3}\times U_{NL}} = \frac{10\times10^6}{\sqrt{3}\times10.5\times10^3} = 549.86\text{A}$$

**步骤二：计算各侧平衡系数**

基准侧选高压侧。高压侧平衡系数 $K_{PH}=1$。

低压侧平衡系数（考虑CT变比和电压比）：
$$K_{PL} = \frac{I_{NH}}{I_{NL}}\times\frac{n_{TAL}}{n_{TAH}} = \frac{164.96}{549.86}\times\frac{200}{60} = \frac{164.96}{549.86}\times3.333 = 1.000$$

（理论上应等于1，若不为1需在保护装置中设置平衡系数进行补偿）

**步骤三：确定最小动作电流 $I_{op.min}$（差动门坎）**

按躲过变压器额定负载时的不平衡电流整定：
$$I_{op.min} = K_{rel}\times(I_{err0}+K\times I_{N(max)}) $$

其中：
- $K_{rel}=1.3$（可靠系数）
- $I_{err0}=0.1\times I_{N(max)}$（CT误差引起的不平衡电流，10%）
- $K=0.05$（变压器分接头变化引起的不平衡电流，5%）
- $I_{N(max)}=1.2\times I_{NH}=197.95\text{A}$（最大负载电流）

$$I_{op.min} = 1.3\times(0.1\times197.95 + 0.05\times197.95) = 1.3\times29.69 = 38.60\text{A}$$

折算到高压侧（以5A为基准）：
$$I_{op.min(primary)} = \frac{38.60}{60} = 0.643\text{A}$$
设置差动门坎为 **0.65A**（归算至高压侧CT一次值）。

**步骤四：确定比率制动系数和拐点电流**

比率制动特性用于区分内部故障和外部故障。典型特性：

```
动作电流 ↑
        │        内部故障区
        │      ╱
        │    ╱
        │  ╱
        │╱───────  外部故障（穿越电流大）
--------┼────────→ 穿越电流（制动电流）
 拐点  斜率=K1   斜率=K2
```

- 拐点电流 $I_{res.0}=1.0\times I_{N}$（高压侧额定电流）
- 第一段斜率（低斜率段）：$K_1=0.3$（区分轻微内部故障）
- 第二段斜率（高斜率段）：$K_2=0.7$（躲过外部严重故障）

动作电流表达式：
$$I_{op} = I_{op.min} + K_1\times(I_{res}-I_{res.0})\quad (I_{res}\le I_{res.0})$$
$$I_{op} = I_{op.min} + K_1\times(I_{res.0}-I_{res.0}) + K_2\times(I_{res}-I_{res.0})\quad (I_{res}>I_{res.0})$$

**步骤五：灵敏度校验**

最小内部故障电流（高压侧两相金属性短路）：
假设系统阻抗 $Z_s=0.1\text{p.u.}$，变压器阻抗 $Z_T=0.05\text{p.u.}$，则最小短路电流：
$$I_{k.min} = \frac{I_N}{|Z_s+Z_T|} = \frac{164.96}{0.15}\times\frac{1}{1}=1099.7\text{A}$$

折算到CT一次值（高压侧CT变比60）：
$$I_{k.min(CT)} = \frac{1099.7}{60} = 18.33\text{A}$$

制动电流 $I_{res}=I_{k.min(CT)}$，对应的动作电流：
按高斜率段计算：
$$I_{op} = I_{op.min} + K_2\times(I_{res}-I_{res.0}) = 0.65 + 0.7\times(18.33-1.64) = 0.65 + 11.68 = 12.33\text{A}$$

灵敏度系数：
$$K_{sen} = \frac{I_{k.min(CT)}}{I_{op}} = \frac{18.33}{12.33} = 1.49 \ge 2.0\ (\text{不合格})$$

**结论**：灵敏度不足，需要调整。建议：
1. 降低比率制动系数（$K_2$降至0.5）
2. 或降低差动门坎（$I_{op.min}$降至0.5A）
3. 或更换变比更大的CT

调整后验证（$K_2=0.5$，$I_{op.min}=0.5$）：
$$I_{op} = 0.5 + 0.5\times(18.33-1.64) = 0.5 + 8.35 = 8.85\text{A}$$
$$K_{sen} = \frac{18.33}{8.85} = 2.07 \ge 2.0\ (\text{合格})$$

**实际配置值示例**：
| 参数 | 配置值 | 说明 |
|------|--------|------|
| 差动门坎电流 $I_{op.min}$ | 0.5A | 高压侧CT一次值归算 |
| 比率制动斜率 $K_1$ | 0.3 | 低电流区段 |
| 比率制动斜率 $K_2$ | 0.5 | 高电流区段 |
| 拐点电流 $I_{res.0}$ | 1.64A | 归算至高压侧 |
| 二次谐波制动比 | 15% | 励磁涌流制动 |
| 速断电流 $I_{sb}$ | 5.0×I_N | 变压器引出线和部分绕组故障 |

**速断电流整定**：
$$I_{sb} = K_{rel}\times I_{k.max(二次侧三相短路)} = 1.3\times\frac{6000}{164.96} = 47.3\text{A}$$
折算至高压侧：$47.3/60=0.79\text{A}$，取 **5倍额定电流 = 4.92A**（高压侧归算值）。

### 2.2 出线过流保护三段式定值配合曲线

#### 2.2.1 三段式过流保护原理

三段式过流保护包括：
- **一段（电流速断）**：保护线路首端，动作时间0s，电流定值按躲过末端最大短路电流整定
- **二段（限时电流速断）**：保护线路全长，延时段动作，与一段配合（时间配合），按躲过下一线路首端速断整定
- **三段（定时限过流）**：作为本线路和下一线路的后备保护，按躲过最大负载电流整定

#### 2.2.2 计算案例

**已知条件**：
- 线路额定电压：10kV
- 最大负载电流：$I_{L.max}=200\text{A}$
- 本线路末端最大三相短路电流：$I_{k.max}=3000\text{A}$
- 下一线路首端速断定值：$I_{set.下}=2500\text{A}$
- 系统最小方式下本线路末端最小两相短路电流：$I_{k.min}=1200\text{A}$
- 时间级差：$\Delta t = 0.3\text{s}$（保护配合时间级差）

**一段（电流速断）整定**：

按躲过本线路末端最大三相短路电流整定：
$$I_{set.I} = \frac{K_{rel}}{K_{re}} \times I_{k.max} = \frac{1.3}{0.85}\times3000 = 4588\text{A}$$

取 $I_{set.I}=4500\text{A}$（留有一定裕度）。

灵敏度校验：
$$K_{sen.I} = \frac{I_{k.min}}{I_{set.I}} = \frac{1200}{4500} = 0.27 < 1.2\ (\text{灵敏度不足})$$

**结论**：一段保护无法覆盖全线，需要靠二段和三段配合保护全线。

实际配置：一段只保护线路首端的 **60%~70%**，灵敏度不足时依靠二段延伸保护。

**二段（限时电流速断）整定**：

按躲过下一线路首端速断保护最大动作电流整定：
$$I_{set.II} = K_{rel}\times I_{set.下} = 1.1\times2500 = 2750\text{A}$$

取 $I_{set.II}=2800\text{A}$。

动作时间：$t_{set.II} = t_{set.下}+\Delta t = 0+0.3 = 0.3\text{s}$（若下一线路速断为0s）

灵敏度校验（保护线路全长）：
$$K_{sen.II} = \frac{I_{k.min}}{I_{set.II}} = \frac{1200}{2800} = 0.43 < 1.25\ (\text{灵敏度不足})$$

**说明**：本案例系统阻抗较大，最小短路电流较小，二段灵敏度不足。实际工程中需要：
1. 降低二段电流定值（但需与一段配合）
2. 或采用反时限过流保护
3. 或加装距离保护

**三段（定时限过流）整定**：

按躲过最大负载电流整定：
$$I_{set.III} = \frac{K_{rel}}{K_{re}}\times I_{L.max} = \frac{1.2}{0.85}\times200 = 282.4\text{A}$$

取 $I_{set.III}=300\text{A}$。

动作时间：按阶梯原则配合，整定时间需与上一级保护配合。

灵敏度校验：
$$K_{sen.III} = \frac{I_{k.min}}{I_{set.III}} = \frac{1200}{300} = 4.0 \ge 1.5\ (\text{合格})$$

**反时限过流配合曲线**（IEC 60255-151）

| 段别 | 曲线类型 | 启动电流(A) | 10倍启动电流时动作时间(s) | 时间常数 |
|------|---------|-------------|-----------------------|---------|
| 一段 | 定时限（0s速断） | 4500 | — | — |
| 二段 | 定时限 | 2800 | 0.3 | — |
| 三段 | 标准反时限（SIT） | 300 | 1.0 | 0.05 |

### 2.3 零序电流保护定值（接地故障）

#### 2.3.1 计算案例

**已知条件**：
- 系统额定电压：35kV
- 系统接地方式：小电阻接地（接地电阻$R_g=10\Omega$）
- 线路零序电流互感器变比：$n_0 = 100/5 = 20$
- 上一级零序保护定值：$I_{0.set.上}=30\text{A}$（一次值）
- 本线路末端接地故障最小零序电流：$I_{0.min}=50\text{A}$（一次值）
- 时间级差：$\Delta t=0.3\text{s}$

**零序过流一段（速断）**：

按躲过本线路末端发生单相接地时可能产生的最大零序电流整定（考虑线路分布电容电流的影响）：
$$I_{0.set.I} = K_{rel}\times 3I_{0.max} = 1.3\times(50+10) = 78\text{A}$$

取 $I_{0.set.I}=80\text{A}$（一次值）。

折算至零序CT二次值：$I_{0.set.I(二次)} = 80/20 = 4\text{A}$。

灵敏度校验：
$$K_{sen.I} = \frac{I_{0.min}}{I_{0.set.I}} = \frac{50}{80} = 0.625 < 1.5\ (\text{不合格})$$

**零序过流二段（限时）**：

按与上一级零序保护配合整定：
$$I_{0.set.II} = K_{rel}\times I_{0.set.上} = 1.1\times30 = 33\text{A}$$

取 $I_{0.set.II}=35\text{A}$（一次值），折算至二次：$35/20=1.75\text{A}$。

灵敏度校验：
$$K_{sen.II} = \frac{50}{35} = 1.43 \ge 1.25\ (\text{合格})$$

动作时间：
$$t_{set.II} = t_{上}+\Delta t = 0.5+0.3 = 0.8\text{s}$$

**零序过流三段（定时限后备）**：

按躲过本线路最大电容电流整定：
电容电流 $I_c = \omega C U_0$，假设本线路电容电流约15A：
$$I_{0.set.III} = K_{rel}\times I_c = 1.2\times15 = 18\text{A}$$

取 $I_{0.set.III}=20\text{A}$（一次值），折算至二次：$20/20=1\text{A}$。

灵敏度校验：
$$K_{sen.III} = \frac{50}{20} = 2.5 \ge 1.5\ (\text{合格})$$

### 2.4 低频减载/低压减载整定

#### 2.4.1 低频减载（自动负荷切除，UFR）

**整定原则**：
- 按频率分轮次动作，每轮切除一定比例负荷
- 动作频率按系统稳定要求和负荷特性设置
- 每轮之间设置时间延迟（防止误动）
- 最后一轮为紧急减载（保系统稳定）

**实际配置值示例**：

| 轮次 | 动作频率(Hz) | 动作延时(s) | 切除负荷比例 | 备注 |
|------|-------------|------------|-------------|------|
| 第一轮 | 49.0 | 0.2 | 5% | 基本轮 |
| 第二轮 | 48.8 | 0.2 | 5% | 基本轮 |
| 第三轮 | 48.6 | 0.5 | 5% | 基本轮 |
| 第四轮 | 48.4 | 0.5 | 5% | 基本轮 |
| 第五轮 | 48.2 | 0.5 | 5% | 基本轮 |
| 紧急轮 | 48.0 | 0.2 | 10% | 紧急轮 |
| 特殊轮 | 49.0 | 5.0 | — | 频率恢复自动重合 |

**频率采集死区**：$\pm 0.02\text{Hz}$（防止频率波动误动）

**闭锁条件**：
- 系统频率快速下降（$df/dt > -0.5\text{Hz/s}$）时短时闭锁（防止系统振荡误动）
- 电压低于70%额定电压时闭锁低频减载（防止电压暂降误动）
- 系统频率低于48Hz持续超过60s，自动追加切除负荷

#### 2.4.2 低压减载（自动负荷切除，UVR）

**整定原则**：
- 按电压分轮次动作，电压越低，切除越多
- 与低频减载配合，共同维护系统稳定性
- 设置低压加速判据（电压快速下降时加速动作）

**实际配置值示例**：

| 第二轮 | 0.80pu (8.0kV) | 0.5 | 5% | 基本轮 |
| 第三轮 | 0.75pu (7.5kV) | 1.0 | 5% | 基本轮 |
| 第四轮 | 0.70pu (7.0kV) | 1.0 | 5% | 基本轮 |
| 紧急轮 | 0.65pu (6.5kV) | 0.2 | 10% | 紧急轮 |

### 2.5 防孤岛保护定值

#### 2.5.1 防孤岛保护参数详解

防孤岛保护是储能系统并网运行时最重要的保护功能之一，用于检测电网失压并迅速将储能系统与电网分离，防止形成孤岛运行。

**ROCOF（Rate of Change of Frequency，频率变化率）保护**

整定公式：
$$ROCOF_{set} = \frac{df}{dt}\bigg|_{max\_allowed} = \frac{0.5\text{Hz}}{0.5\text{s}} = 1.0\text{Hz/s}$$

实际配置：
- 动作值：$1.0\text{Hz/s}$
- 延迟时间：$0.5\text{s}$（防误动）
- 闭锁条件：启动瞬间（$t<0.2\text{s}$）屏蔽ROCOF保护

**欠压保护（UV）**

| 相别 | 动作电压(pu) | 动作电压(V) | 持续时间(s) | 动作 |
|------|------------|------------|------------|------|
| 三相欠压 | 0.8pu | 184V（380V系统） | 2.5 | 跳闸 |
| 单相欠压 | 0.8pu | 184V | 2.0 | 告警+跳闸 |

**过压保护（OV）**

| 相别 | 动作电压(pu) | 动作电压(V) | 持续时间(s) | 动作 |
|------|------------|------------|------------|------|
| 三相过压 | 1.10pu | 253V（380V系统） | 1.0 | 跳闸 |
| 单相过压 | 1.10pu | 253V | 0.5 | 告警+跳闸 |

**频率保护**

| 保护类型 | 动作频率(Hz) | 延迟(s) | 动作 |
|---------|------------|--------|------|
| 低频保护 | 47.5 | 0.2 | 跳闸 |
| 高频保护 | 51.5 | 0.2 | 跳闸 |
| 低频保护（严苛） | 48.0 | 0.1 | 紧急跳闸 |
| 高频保护（严苛） | 51.0 | 0.1 | 紧急跳闸 |

**主动频率偏移（AFD，Active Frequency Drift）**

适用条件：PCS具备主动孤岛检测功能时。

配置参数：
- 频率偏移量：$+0.5\text{Hz}$（向高频方向偏移）
- 偏移持续时间：$0.3\text{s}$
- 响应电网断开时，PCS注入频率偏移信号，迫使逆变器停止

| TS-15 | EMS离网指令→PCS切换至离网模式 | EMS下发离网切换指令 | PCS完成模式切换并稳定输出 | ≤5s | 指令发出到PCS离网模式运行稳定 | ≤5s | 离网切换 |

### 3.3 时序测试操作方法

#### 3.3.1 消防告警→PCS停机时序测试（TS-01）

**测试工具**：示波器（4通道以上）、万用表、继电保护测试仪、消防回路测试仪

**测试步骤**：
1. 在消防主机急停输出继电器两端并联示波器通道1（触发源）
2. 在PCS PWM输出端（门极驱动信号）连接示波器通道2
3. 触发消防主机的手动报警按钮（或使用消防回路测试仪注入报警信号）
4. 记录从报警触发到PWM封锁的时间差
5. 重复测试3次，取最大值作为最终结果
6. 测试过程中同时记录CAN总线上的BMS急停帧

**判定标准**：
- 时间差≤500ms：合格
- 500ms < 时间差 ≤ 800ms：降级处理，需整改
- 时间差>800ms：不合格，立即整改

**录波要求**：测试时同步启动故障录波仪，记录整个时序过程的电气量（电压、电流、继电器状态）

#### 3.3.2 防孤岛检测→PCC分断时序测试（TS-02）

**测试工具**：继电保护测试仪（输出继电器开出）、故障录波仪、数字万用表

**测试步骤**：
1. 将继电保护测试仪的开出继电器接入防孤岛保护装置的跳闸输入端（或使用保护装置自身的测试功能）
2. 将故障录波仪的电流/电压通道接入PCC断路器两侧的CT/PT
3. 将断路器辅助触点接入故障录波仪的开关量输入通道
4. 设置继电保护测试仪输出ROCOF保护动作信号（或直接触发防孤岛保护装置的测试模式）
5. 同时触发继电保护测试仪开出和故障录波仪启动
6. 记录从保护动作开出到断路器辅助触点变位（分位）的时间差

**判定标准**：
- 时间差≤2s：合格
- 2s < 时间差 ≤ 3s：降级处理
- 时间差>3s：不合格

**录波分析**：检查故障录波图，确认保护动作、跳闸指令发出、断路器分断三个事件的时序关系正确。

#### 3.3.3 EMS功率指令→PCS响应时序测试（TS-03）

**测试工具**：通信分析仪（CAN/Modbus协议分析仪）、功率计、同步时钟

**测试步骤**：
1. 将通信分析仪接入EMS与PCS之间的通信线缆（CAN或以太网）
2. 配置通信分析仪的触发条件为EMS发出的功率指令帧
3. 在PCS交流输出端接入功率计，测量实际有功功率
4. 同步记录通信分析仪的时间戳和功率计的功率变化曲线
5. 从EMS下发一个已知的功率设定值（如50%额定功率），记录从指令发出到功率达到稳定的时间
6. 重复测试5次，计算平均响应时间

**判定标准**：
- 平均响应时间≤1s：合格
- 1s < 平均响应时间 ≤ 2s：降级处理
- 平均响应时间>2s：不合格

#### 3.3.4 调度AGC指令→实际功率变化时序测试（TS-05）

**测试工具**：SCADA历史数据查询、功率曲线记录设备、时间同步服务器

**测试步骤**：
1. 在调度主站发起一次AGC指令下发（要求储能站有功功率从P1调整到P2）
2. 记录调度主站发出指令的精确时间戳（SCADA系统记录）
3. 查询储能站本地SCADA系统中PCS实际有功功率的变化时刻
4. 计算从调度指令发出到PCS实际功率达到目标值±5%的时间差
5. 查询该时间段内的AGC指令和实际功率的响应曲线

**判定标准**：
- 时间差≤4s：合格
- 4s < 时间差 ≤ 6s：降级处理
- 时间差>6s：不合格

### 4.1 整组传动试验概述

### 4.2 整组传动试验步骤详解

#### 4.2.1 继电器→断路器联动传动（静态试验）

**试验目的**：不投入实际运行设备，通过继电保护测试仪注入模拟量，验证保护动作、跳闸回路、断路器分断的完整链路。

**试验步骤**：

**第一步：准备阶段**
1. 确认断路器处于分位，断路器操作把手已锁定在"试验"位置
2. 确认跳闸回路已连接（未隔离），合闸回路已连接
3. 将故障录波仪接入CT/PT回路，设置为触发记录模式
4. 在保护装置面板上确认当前定值和压板状态

**第二步：变压器差动保护传动**
1. 使用继电保护测试仪，在高压侧CT一次侧注入电流模拟量
2. 计算差动保护动作电流值，设置为定值的0.95倍（不动作）和1.05倍（动作）
3. 注入电流超过定值，观察保护装置面板显示动作
4. 测量保护装置跳闸开出继电器触点闭合（用万用表通断档）
5. 观察断路器机构合闸保持继电器释放（储能释放）
6. 测量断路器主触点分离（辅助触点变位）
7. 记录从保护动作到断路器分断的总时间
8. 重复测试3次，记录每次的动作时间
9. 恢复断路器合闸，准备下一项测试

**第三步：出线过流保护传动**
1. 在出线CT回路注入过流信号，逐步增加电流至过流定值
2. 观察保护装置面板显示过流告警→动作
3. 测量跳闸开出继电器动作
4. 观察断路器分断
5. 如配置反时限，验证动作时间与电流值的配合关系（做3~5个点的反时限曲线）

**第四步：零序过流保护传动**
1. 在零序电流互感器回路注入零序电流
2. 观察零序保护动作，记录动作电流和动作时间
3. 验证与上级零序保护的时限配合（通过整组传动试验）

**第五步：防孤岛保护传动**
1. 使用继电保护测试仪输出继电器开出（模拟保护动作）
2. 将开出继电器接入防孤岛保护装置的跳闸输入
3. 测量PCC断路器跳闸线圈得电，断路器分断
4. 记录整个时序时间（≤2s）

**第六步：消防联动传动（禁喷模式）**
1. 触发消防主机的手动报警按钮（或使用消防回路测试仪）
2. 测量消防主机急停继电器输出触点闭合
3. 测量BMS接收急停信号后封锁PWM（示波器观察PWM波形）
4. 测量PCS响应停机（PWM封锁）
5. 记录时间差（≤500ms）
6. **注意**：不触发消防灭火系统喷淋，只测试联动回路

#### 4.2.2 并网整组传动（动态试验）

**前提条件**：
- 静态传动试验全部合格
- 调度批准有限度的并网试验
- 所有安全措施到位

**试验内容**：

1. **模拟并网状态下继电保护动作试验**：在储能系统并网运行时，通过故障发生器在10kV出线上注入模拟短路电流，验证保护装置→断路器的整组动作。

2. **模拟防孤岛试验**：在并网状态下，人为断开PCC断路器（模拟电网故障），验证防孤岛保护动作→PCC断路器跳闸的时序。验证孤岛检测时间≤2s，且PCS在检测到孤岛后正确停机。

3. **功率调节传动试验**：从调度主站下发AGC指令，验证EMS→PCS→实际功率变化的整组响应。测量从指令发出到功率变化的时间（≤4s）和精度。

4. **离网切换传动试验**：在并网状态下，从EMS下发离网切换指令，验证PCS从并网模式切换到离网模式的过程和时序（≤5s）。

### 4.3 保护动作录波分析要求

**录波仪配置要求**：
- 采样率：≥4800点/秒（满足谐波分析要求）
- 模拟量通道：电压（3相+零序）、电流（3相+零序）、直流电压
- 开关量通道：保护动作信号、跳闸指令、断路器位置信号、报警信号
- 触发方式：保护动作触发（保护开出信号）或电压/电流越限触发
- 预触发时间：≥100ms（记录故障前数据）
- 后触发时间：≥500ms（记录故障后数据）

**录波分析项目**：

| 分析项目 | 分析内容 | 合格标准 |
|---------|---------|---------|
| 保护动作时间 | 从故障发生到保护动作的时间 | 符合定值要求 |
| 断路器分断时间 | 从保护动作到主触点分离的时间 | ≤100ms（跳闸回路） |
| 故障电流波形 | 三相电流有效值、相位、谐波含量 | 波形无异常 |
| 故障电压波形 | 三相电压跌落/恢复过程 | 符合故障特性 |
| 重合闸动作（如配置） | 重合闸动作时间、合闸成功性 | 符合整组配合 |
| 保护选择性 | 上下级保护动作时序配合 | 动作选择性正确 |
| 保护灵敏度 | 故障电流与保护定值的配合 | 灵敏度系数≥1.2 |

## 附录B：不合格项处理流程

```
发现不合格项
    ↓

    ↓

    ↓

    ↓

    ↓

    ↓

    ↓
消除缺陷标签，签字归档
```

## 附录C：常见不合格项处理对照表

| 不合格现象 | 可能原因 | 整改方法 |
|-----------|---------|---------|
| CAN通信超时 | 线缆屏蔽不良/终端电阻不匹配/波特率不一致 | 检查屏蔽接地，调整终端电阻，统一波特率设置 |
| 跳闸回路端电压不足 | 导线截面过小/接点接触不良/电源容量不足 | 更换大截面导线，清理接点，增加电源容量 |
| 保护动作时间超标 | 定值配置错误/继电器老化/跳闸机构卡涩 | 重新整定定值，更换继电器，清理维护断路器机构 |
| 防孤岛检测失败 | 检测参数设置不当/电网扰动频繁 | 调整防孤岛检测阈值，增加延时防误动 |
| 继保定值偏差超过5% | 定值输入错误/装置参数变更 | 重新下载定值，核对设计值 |
| 通信响应时间超标 | 通信链路拥塞/设备处理延迟/链路距离过长 | 优化通信周期，升级设备，检查链路质量 |
| 模拟量精度超标 | 变送器故障/接线松动/量程设置错误 | 校准变送器，紧固接线，调整量程 |
| 绝缘电阻不合格 | 线缆破损/潮湿/端子污染 | 更换受损线缆，干燥处理，清洁端子 |

**本Skill使用说明**：
- 在进行每项检查前，应先确认安全措施已落实
- 检查时发现不合格项，应立即停用相关回路，拍照留证，并启动整改流程
- 所有测试记录应及时填写，不得事后补填
- 测试过程中应保持通信记录仪（如CAN分析仪）的全程记录，以便追溯
- 本Skill应与储能电站现场实际情况结合使用，特定项目的特殊要求应补充入本Skill的专项检查表
