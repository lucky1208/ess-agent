# 故障复盘 Skill v1.0

> 设计人：卢继雄 | 基于5年微网项目实战训练

## 概述
从BMS/PCS/EMS告警日志中提取根因, 给出处置建议和备件建议。

## 输入
- 告警日志 (BMS/PCS/EMS, 时间/级别/内容)
- 设备型号和固件版本

## 输出
- 根因分析: 最可能原因+概率
- 处置建议: 立即/短期/长期
- 备件建议: 型号/数量/采购周期

## 规则

### R1: BMS告警分类
- 过温(>55°C): 检查温控/液冷流量/环境温度
- 过压/欠压: 检查PCS输出/簇间均衡
- 绝缘降低: 检查潮湿/接线/绝缘监测仪
- 热失控预警: 立即停机,检查PACK温度梯度

### R2: PCS告警分类
- 过流: 检查负荷突变/短路/参数设置
- 过压/欠压: 检查电网电压/变压器/接地
- 通信中断: 检查网线/交换机/EMQX

### R3: EMS告警分类
- 策略执行失败: 检查点表地址/数据类型
- SOC异常跳变: 检查BMS通信/SOC算法/采样间隔

### R4: 根因推演
- 多告警关联: BMS过温+PCS故障 → 温控系统失效
- 时序分析: 第一个告警为根因候选

## 验证Checklist
- [ ] 根因分析有依据
- [ ] 处置建议分级(立即/短期/长期)
- [ ] 备件型号与设备匹配
