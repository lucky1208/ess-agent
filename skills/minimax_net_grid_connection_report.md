# Skill: 储能电站并网接入方案报告

## 定位与适用范围

本Skill为储能项目并网交付专用，覆盖从接入点选择论证、电气计算、并网柜方案、调度接口配置到并网申请材料编制的全流程。适用于：

- **10kV用户侧工商业储能**（如浙江、江苏、广东等省份的分布式储能项目）
- **35kV电网侧大储**（如江苏、湖南等省份的共享储能/独立储能电站）
- **0.4kV低压储能**（如小区光储微网、工商业余储互补场景）

储能并网接入系统方案设计报告（简称"接入方案"）是储能项目并网前向电网公司提交的核心技术文件，用于论证接入点的合理性、电气计算的合规性、以及调度通信接口的可行性。报告需由具备相应资质的电力设计院（或电网公司认可的设计单位）编制，并通过电网公司组织的审查。

## 一、接入系统方案设计（35kV/10kV）

### 1.1 接入点选择论证方法

#### 1.1.1 公共连接点（PCC）选择原则

PCC（Point of Common Coupling，公共连接点）是储能系统与电网的产权分界点，也是电气计算和保护配置的基准点。选择原则如下：

| 原则 | 说明 | 典型取值 |
|------|------|---------|
| 就近接入 | 减少线路投资和线路损耗，优先选择现有配电设施附近 | 距离目标配电设施 < 500m 为佳 |
| 容量匹配 | 接入线路和配变的承载能力需与储能放电功率匹配 | 储能放电功率 ≤ 线路热稳定电流 × 电压 |
| 产权分界清晰 | 分界点设备归属明确，通常为电网侧杆塔/开关站 | 分界开关通常位于电网资产侧 |
| 电网结构合理 | 避免形成电磁环网或增加短路电流超标风险 | 短路电流计算需覆盖新接入后最大方式 |
| 调度便利性 | 优先接入有调度自动化设施的变电站/开关站 | 有远动通道、通信光缆覆盖的区域 |

**PCC位置确定流程**：
1. 收集目标区域配电网接线图（含导线型号、长度、配变容量）
2. 初步筛选3~5个候选接入点
3. 对每个候选点进行潮流计算和短路电流计算
4. 综合投资、调度便利性，确定最优PCC
5. 绘制地理接线示意图，标注PCC坐标和线路路径

#### 1.1.2 35kV vs 10kV接入判断条件

| 判断条件 | 35kV接入 | 10kV接入 |
|---------|---------|---------|
| 储能总放电功率 | ≥ 8MW（部分省份 ≥ 6MW） | < 8MW（常规工商业储能） |
| 距离最近35kV变电站 | < 5km，且变电站在负荷中心 | 附近无35kV电源点 |
| 区域电网结构 | 35kV网架坚强、通道资源充足 | 10kV网架可满足要求 |
| 投资经济性 | 35kV接入投资高，但可减少对配电网的冲击 | 投资较低，接入灵活 |
| 调度管理要求 | 必须接入调度自动化系统，签订调度协议 | 可根据功率规模选择是否接入调度 |

> **经验规则**：储能放电功率 ≥ 6MW 或单回10kV线路无法消纳时，优先论证35kV接入方案。当地供电公司若有明确的功率门槛规定，以当地文件为准（如浙江要求 4MW 以上走35kV 或10kV专线）。

#### 1.1.3 典型接入方式

**方式一：一回T接（最常用）**
```
变电站10kV母线 ──10kV线路──[T接点]──储能并网柜
                              ↑
                         储能经一回T接分支线接入
```
适用场景：10kV线路上有可用T接点，分支线长度 ≤ 2km，储能功率 ≤ 3MW。
注意：需核实T接点原有设备的载流能力，是否需要加固或更换。

**方式二：专线T接（储能专线）**
```
变电站10kV母线 ──10kV专线──[储能并网柜]
                      ↑
               专线接入，产权归储能投资方
```
适用场景：储能功率较大（>3MW）或对供电可靠性要求高。专线建设成本由储能投资方承担，但可自主管理线路。

**方式三：开关站接入**
```
变电站 ──35kV/10kV── 开关站A ──出线──[储能并网柜]
                                ↑
                    开关站预留间隔供储能接入
```
适用场景：附近有开关站（含备用间隔），储能可就近接入开关站出线，节省线路投资。

**方式四：对侧并网（专线对侧接入）**
```
储能侧  ──专用线路── [变电站/开关站35kV/10kV母线]
```
适用场景：储能场址距变电站距离较远（>3km），采用专线对侧接入，确保送出通道独立。

### 1.2 接入系统电气计算

电气计算是接入方案报告的核心章节，用于论证储能接入后电网各节点的电压、电流、短路电流是否满足规程要求。计算需覆盖以下工况：

#### 1.2.1 潮流计算

| 工况 | 说明 | 计算目的 |
|------|------|---------|
| 正常方式（最大负荷） | 储能满功率放电，电网负荷处于高峰 | 验证电压抬升是否超标（+7%/−10%）、线路是否过载 |
| 检修方式（N-1） | 储能运行时，相关供电线路检修停运 | 验证迂回通道的输送能力 |
| 事故方式（故障隔离后） | 相邻线路或主变故障，储能孤岛运行 | 验证储能对重要负荷的持续供电能力 |
| 充电工况 | 储能充电功率最大时（尤其是光伏+储能） | 验证反向功率对配变的影响 |

**电压偏差计算要求**：
- 储能放电时，PCC点电压偏差应控制在标称电压的 +7% 以内（10kV系统即 10.7kV 以内）
- 储能充电时，PCC点电压不应低于标称电压的 -10%（即 9.0kV 以上）
- 若电压偏差超标，需配置有载调压变压器或无功补偿装置

**线路载流量校验**：
```
校验公式：I_储能放电 ≤ I_线路额定 × 0.85（考虑环境温度修正）
校验公式：I_储能充电反向 ≤ I_线路额定 × 0.85
```

#### 1.2.2 短路电流计算

| 计算类型 | 边界条件 | 目的 |
|---------|---------|------|
| 最大方式（系统最大运行方式） | 所有电源并列，系统阻抗取最小 | 验证断路器遮断容量是否满足（通常要求 ≥ 31.5kA / 40kA） |
| 最小方式（系统最小运行方式） | 一回线路检修，系统阻抗取最大 | 验证保护的灵敏度是否满足（电流保护最小灵敏系数 ≥ 1.5） |

**储能侧短路电流贡献**：
储能 PCS 在故障时具备瞬时过电流输出能力（1.5~2倍额定电流，0.1~0.5s），但持续时间有限。计算时需说明：
- PCS 提供的稳态短路电流（通常为 1.1~1.5 倍额定电流，持续 2~5s）
- PCS 在低电压穿越期间对短路电流的支撑能力（符合 GB/T 34120 第 8.3 条）
- 储能侧短路电流对电网侧保护的配合关系（避免储能侧故障时保护越级动作）

**计算表格模板**：

| 节点 | 基准容量(MVA) | 最大方式 Ik''(kA) | 最小方式 Ik''(kA) | 断路器额定遮断(kA) | 是否满足 |
|------|-------------|-----------------|-----------------|-----------------|---------|
| PCC点 | 100 | 12.5 | 8.3 | 31.5 | ✓ |
| 储能进线柜 | 100 | 11.8 | 7.9 | 25.0 | ✓ |
| 变压器高压侧 | 100 | 15.2 | 9.8 | 40.0 | ✓ |

#### 1.2.3 调压计算与无功补偿配置

储能系统需具备无功调节能力，用于协助电网调压。无功补偿配置计算如下：

**无功补偿容量计算**：
```
Q_c = P × (tan φ1 - tan φ2)

其中：
P = 储能额定放电功率 (MW)
tan φ1 = 补偿前功率因数的正切值（假设为 0.5，即 cos φ1 ≈ 0.9）
tan φ2 = 补偿后目标功率因数（通常 ≥ 0.95）
```

**无功补偿配置方案示例**：

| 配置方案 | 说明 | 适用场景 |
|---------|------|---------|
| 方案A（无功优先模式） | 储能PCS功率因数可调范围 0.95（超前）~0.95（滞后），优先利用储能自身无功能力 | 储能放电功率较小，对电网电压影响有限 |
| 方案B（SVG/无功补偿装置） | 单独配置静止无功发生器（SVG），无功容量按储能额定功率的 15%~25% 配置 | 储能功率较大，电压偏差超标，需专用无功补偿 |
| 方案C（混合模式） | 储能自身无功 + 配套SVG，综合补偿 | 大型电网侧储能（35kV接入） |

#### 1.2.4 稳定校核（如电网公司要求）

部分省份（尤其是西北/华北高比例新能源省份）对大储项目要求开展暂态稳定校核：
- **低电压穿越（LVRT）**：储能PCS在电网电压跌落至 20% 额定电压时应持续并网 625ms，并在2s内恢复
- **高频穿越**：当电网频率升至 51.0Hz 时，储能应至少维持 10min 并网运行（具体按电网要求）
- **故障穿越能力证明**：需提供 PCS 型式试验报告，证明具备上述穿越能力

### 1.3 并网柜/开关站方案

#### 1.3.1 并网柜一次接线图模板

**10kV并网柜典型配置（单母线接线）**：

```
                    ┌──────────────────────────────────────────┐
                    │              10kV母线                    │
                    └──────┬─────────┬──────────┬─────────────┘
                           │         │          │
                    ┌──────┴───┐  ┌─┴─────┐  ┌─┴────────┐
                    │  进线柜   │  │计量柜 │  │出线柜(去 │
                    │ (储能侧) │  │(高压   │  │10kV母线) │
                    │          │  │计量)  │  │          │
                    └──────┬───┘  └───────┘  └──────────┘
                           │
                    ┌──────┴─────────────────────────────┐
                    │        储能变压器高压侧           │

                           │
                    ┌──────┴─────────────────────────────┐

```

**并网柜柜体配置要求**：

| 柜体编号 | 柜体名称 | 主要设备 | 技术要求 |
|---------|---------|---------|---------|
| #1 | 储能进线柜 | 断路器 + 隔离开关 + 电流互感器(0.5s级) | 额定电流 ≥ 储能额定电流 × 1.2，遮断容量 ≥ 31.5kA |
| #2 | 计量柜 | 高压计量柜（主副表）+ 电压互感器 | 计量精度 0.5S/0.2S，接入电网关口计量点 |
| #3 | 出线柜 | 断路器 + 隔离开关 + 零序电流互感器 | T接点侧保护，含方向过流保护 |
| #4 | 母线设备柜 | 母线避雷器 + 电压互感器柜 | 用于母线过电压保护和母线电压监测 |
| #5 | 接地变柜（如需要） | 接地变压器 + 消弧线圈 | 用于10kV网络经消弧线圈接地方式 |

#### 1.3.2 计量点配置

| 计量方式 | 适用场景 | 计量装置要求 | 安装位置 |
|---------|---------|------------|---------|
| 高压计量（高供高计） | 35kV接入；10kV接入且储能功率 ≥ 500kW | 主副表配置，精度 ≥ 0.5S；主表安装在并网柜计量柜中 | 储能并网柜（产权分界点） |
| 低压计量（高供低计） | 0.4kV低压接入；10kV接入且储能功率 < 500kW | 单表配置，精度 1.0S | 储能变压器低压侧 |
| 关口计量 | 电网侧大储（结算用） | 主副表双表配置，精度 0.2S，接入省级关口计量系统 | 电网侧变电站侧（由电网提供） |

> **重要**：关口计量点必须由电网公司安装和校验，储能侧不得自行安装结算用电能表。计量电流互感器变比需与储能额定电流匹配，通常为 200/5A、400/5A、600/5A。

#### 1.3.3 保护配置

**10kV并网柜保护配置**：

| 保护类型 | 动作配置 | 整定原则 | 动作后果 |
|---------|---------|---------|---------|
| 方向过流保护（速断） | 电流 > 整定值（通常 5~8倍额定），时间 < 0.3s | 躲过储能最大反馈电流，考虑方向性（指向电网侧为正方向） | 跳开储能侧断路器 |
| 方向过流保护（过流一段） | 电流 > 整定值（1.5~2倍额定），时间 0.5~1.0s | 与上一级保护配合，灵敏系数 ≥ 1.5 | 跳开储能侧断路器 |
| 过负荷保护 | 电流 > 1.1倍额定，时间 10~30s | 躲过正常充电电流峰值 | 发告警 |
| 零序电流保护 | 零序电流 > 整定值，时间 0.5~2s | 用于单相接地故障检测 | 跳闸或告警（按接地方式） |
| 低频/高频保护 | 频率 < 49.0Hz 或 > 51.0Hz | 按电网要求整定（部分省份要求 48.5~51.5Hz） | 延迟跳闸（按频率偏差量级） |
| 低压/过压保护 | 电压 < 0.85Un 或 > 1.1Un | 按并网点电压允许范围整定 | 延迟跳闸（0.1~2.0s） |

**35kV并网柜保护配置**：

| 保护类型 | 配置要求 |
|---------|---------|
| 纵联差动保护 | 必须配置，储能侧和电网侧各一套，通过光纤通信实现差动保护 |
| 距离保护 | 作为后备保护，整定阻抗按线路参数计算 |
| 过流保护 | 方向过流，作为距离保护的后备 |
| 母线保护 | 开关站35kV母线需配置母线差动保护 |
| 自动重合闸 | 线路故障跳闸后按电网要求配置 |
| 断路器失灵保护 | 断路器拒动时跳开上级电源侧断路器 |

## 二、电网调度接口方案

### 2.1 AGC功率指令接口

#### 2.1.1 IEC104接口配置

储能电站需通过 IEC 60870-5-104（简称 IEC104）规约接收电网 AGC（自动发电控制）系统下发的功率指令。以下为典型配置示例：

**104规约基本参数配置**：

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 传输层端口 | 2404 | IEC104 约定端口，通常为 TCP 2404 |
| 应用层端口 | 2404 | 与传输层一致 |
| 公共地址（ASDUCOMMONADDR） | 根据调度分配 | 储能电站调度地址，如 0x0001 |
| 遥控源地址（SRCADDR） | 00~FF | 通常由调度侧配置 |
| 信息对象地址（IOA） | 见遥测/遥控配置表 | 每个信息点唯一 |
| 传输原因（COT） | 3（突发）/4（循环）/5（初始化）/20（激活确认） | 遥测为3/4，遥控确认为20 |
|  APDU超时 | 30s | 无应答重发 |
| APDU 重试次数 | 3 | 最大重发次数 |

**ASDU类型配置**：

| ASDU类型 | 方向 | 用途 | 常见配置 |
|---------|------|------|---------|
| M_SP_NA_1（类型1） | 上行 | 单点遥信（告警、状态） | 告警信号、保护动作信号 |
| M_DP_NA_1（类型3） | 上行 | 双点遥信（断路器状态） | 断路器分合位 |
| M_ME_NA_1（类型9） | 上行 | 归一化测量值（遥测） | 有功、无功、电压、频率 |
| M_ME_NC_1（类型13） | 上行 | 浮点测量值（高精度遥测） | 功率、能量（优先使用此类型） |
| C_SC_NA_1（类型45/未确认-1） | 下行 | 单点遥控（分/合） | 断路器遥控 |
| C_DC_NA_1（类型46/未确认-1） | 下行 | 双点遥控（分/合） | 断路器遥控 |
| C_SE_NC_1（类型50） | 下行 | 设定值命令（设点） | AGC功率指令（优先使用此类型） |
| C_SE_NA_1（类型48） | 下行 | 归一化设点命令 | 备用设点命令 |

**IOA地址规划示例**：

| 信息点描述 | IOA地址（十六进制） | 数据类型 | 说明 |
|-----------|-----------------|---------|------|
| 有功功率遥测 | 0x000001 | M_ME_NC_1 | 单位：0.1MW，分辨率0.1MW |
| 无功功率遥测 | 0x000002 | M_ME_NC_1 | 单位：0.1Mvar |
| 储能充放电状态 | 0x000003 | M_SP_NA_1 | 0=充电 1=放电 2=待机 |
| AGC指令（设点） | 0x000101 | C_SE_NC_1 | 下行，调度下发功率目标值 |
| AGC使能遥控 | 0x000102 | C_SC_NA_1 | 下行，0=停用 1=启用 |
| PCS 断路器合 | 0x000201 | C_SC_NA_1 | 下行，遥控合闸 |
| PCS 断路器分 | 0x000202 | C_SC_NA_1 | 下行，遥控分闸 |

#### 2.1.2 AGC响应时间要求

| 指标 | 要求值 | 说明 |
|------|--------|------|
| 指令接收延迟 | ≤ 1s | 从调度主站下发到储能就地控制系统收到的时间 |
| 功率响应时间 | ≤ 4s | 收到AGC指令后，储能实际功率开始变化的时间 |
| 功率变化率 | ≤ 额定功率的 20%/min（默认） | 可通过调度协议约定更大变化率 |
| 稳态误差 | ±2%（或协议约定值） | 实际功率与目标值的偏差 |
| 通信中断处理 | 通信中断 ≥ 30s，储能进入本地调度模式 | 保持最后一帧指令的功率或进入待机 |

**功率变化率计算示例**：
```
储能额定功率：10MW
默认变化率：20%/min → 2MW/min → 33.3kW/s
若调度下发从 5MW→8MW 的指令：
  需耗时：(8-5)/2 = 1.5min（90s）
  响应时间：≤ 4s（满足）
```

#### 2.1.3 AGC指令格式详解

**设点命令（C_SE_NC_1）帧结构示例**：

```
68 1F          # APDU长度（31字节）
03             # 控制域：未编号的监视功能（TESTFR ACT）
00 00 00 00    # 发送序号
00 00 00 00    # 接收序号
67             # 类型标识（TYP=103，设点命令）
01             # 可变结构限定符（SQ=0，N=1）
0E             # 传输原因（COT=14，激活）
00 01          # ASDU公共地址（储能站地址）
01 01 00 00    # IOA=0x000101（AGC设点）
09 00 00 00    # QDS（品质描述，0x00=有效）
40 1E 00 00    # 归一化值：8000（对应 8.0MW）
```

**响应帧（肯定确认）**：
```
68 14          # APDU长度（20字节）
00             # 控制域：未编号的确认
XX XX XX XX    # 发送序号
YY YY YY YY    # 接收序号
67             # 类型标识（回叫）
01             # 可变结构限定符
0D             # 传输原因（COT=13，激活确认）
00 01          # ASDU公共地址
01 01 00 00    # IOA（对应触发指令）
00             # 激活状态（0=成功）
```

### 2.2 AVC无功控制接口

#### 2.2.1 无功调度指令格式

AVC（自动电压控制）系统向储能下发无功/功率因数指令，通常使用以下格式：

| 指令类型 | ASDU类型 | IOA | 格式说明 |
|---------|---------|-----|---------|
| 无功功率设点 | C_SE_NC_1（类型50） | 0x000103 | 单位：0.1Mvar，范围：-Qmax~+Qmax |
| 功率因数设点 | C_SE_NA_1（类型48） | 0x000104 | 归一化值，如：8000=0.8 |
| AVC使能 | C_SC_NA_1（类型45） | 0x000105 | 0=停用 1=启用 |

**无功指令响应要求**：
- 响应时间：≤ 60s（较 AGC 更宽松）
- 无功调节范围：功率因数 0.95（滞后）~ 0.95（超前）
- 部分电网要求储能具备进相运行能力（吸收无功），需在 PCS 型式试验中验证

#### 2.2.2 功率因数控制模式 vs 无功功率控制模式

| 控制模式 | 调度下发内容 | 就地控制策略 | 适用场景 |
|---------|------------|------------|---------|
| 功率因数模式 | cos φ 目标值（0.90~0.95） | 根据有功功率自动计算所需无功 | 调度侧只关心功率因数 |
| 无功功率模式 | Q 目标值（Mvar） | 直接跟踪目标无功值 | 调度侧需精确控制节点电压 |
| 综合模式 | 同时下发 P 和 Q | 按优先级执行（P优先） | 大型储能需同时满足有功和无功调度 |

### 2.3 调度数据网配置

#### 2.3.1 纵向加密装置配置（必须项）

根据国家能源局《电力监控系统安全防护规定》（国能安全〔2015〕36号），调度数据网纵向必须部署纵向加密认证装置（电力专用纵向加密装置）。

**纵向加密装置配置要求**：

| 配置项 | 要求 |
|-------|------|
| 加密算法 | 国密算法（SM1/SM2/SM3/SM4），不得使用国外加密算法 |
| 设备型号 | 通过国家电网电力专用横向隔离装置和纵向加密装置检测的产品（如卫士通、华为、南瑞等） |
| 部署位置 | 储能电站侧纵向加密装置安装在调度数据网入口处（与通信屏同列） |
| 密钥管理 | 由电网调度机构统一下发装置证书和密钥 |
| 隧道建立 | 与调度主站侧纵向加密装置建立IPSec VPN隧道，端口：500/UDP |
| 通信协议 | 纵向加密后不得改变原 IEC104 规约的通信行为 |

**拓扑示例**：
```
调度主站 ──调度数据网（纵向加密隧道）──[纵向加密装置A]── 储能电站
                                                  │
                                                  ↓
                                          [本地SCS/远动装置]
                                                  │
                                                  ↓
                                          [并网柜保护测控单元]
```

#### 2.3.2 调度数据网通道要求

| 项目 | 要求 | 说明 |
|------|------|------|
| 通道数量 | 主通道 × 1 + 备通道 × 1（必须双通道） | 主备通道须物理路由隔离（不得同缆） |
| 传输速率 | ≥ 2Mbps | 通常采用 E1/G.703 或以太网口 |
| 迂回路由 | 主通道故障时备通道自动切换，切换时间 ≤ 50ms | 需与通信运营商协调 |
| 通信规约 | IEC104（调度侧）+ DNP3.0（备通道）或 IEC104 备用 | 备通道通常与主通道规约相同 |
| 网络地址 | 调度数据网使用 10.x.x.x 或 192.168.x.x 私网地址段 | 由调度机构统一下发 |

### 2.4 远动通信配置

#### 2.4.1 远动通信规约选择

| 规约 | 适用场景 | 传输速率 | 优势 |

#### 2.4.3 对时方案

| 对时方式 | 实现方式 | 精度 | 说明 |
|---------|---------|------|------|
| NTP（网络时间协议） | 主备双机配置，通过调度数据网从调度主站NTP服务器对时 | ±50ms | 推荐用于远动装置和通信设备 |
| IRIG-B（直流IRIG-B） | 通过专用时钟装置或保护测控装置的IRIG-B接口对时 | ±1ms | 推荐用于保护装置（精度要求高） |
| GPS/北斗对时 | 场站安装 GPS/北斗时钟天线，双模接收 | ±1ms | 必备设备，为 IRIG-B 提供时间源 |
| IEEE 1588（PTP） | 精密时间协议，对网络交换机有要求 | ±100ns | 适用于要求 ns 级精度的场合（通常不用） |

> **推荐配置**：GPS/北斗时钟（主）+ IRIG-B（保护测控对时）+ NTP（自动化设备对时）。调度侧通常要求远动装置时间偏差 ≤ 1s，事件顺序记录（SOE）分辨率 ≤ 10ms。

| 并网柜（高压柜） | 额定电压(kV)、额定电流(A)、额定遮断容量(kA)、柜体结构、接地方式 | 12kV、630A、31.5kA、铠装中置式、经消弧线圈接地 |
| 直流汇流柜 | 额定电流(A)、最大输入路数、熔断器规格、防雷等级 | 1000A、8路、1000A/1000V、II级防雷 |
| 监控/EMS | 响应时间(ms)、支持规约、通信接口 | ≤100ms、IEC104+Modbus、网口+串口 |
| 纵向加密装置 | 加密算法、端口配置、吞吐量(Mbps) | SM1、2×LAN口、100Mbps |
| 远动装置 | 规约支持、端口数量、对时方式 | IEC104+DL/T 634.5101、8×串口+4×网口、NTP+IRIG-B |

### 3.6 型式试验报告清单

| 认证标准 | 适用产品 | 检测项目 | 检测机构 |
|---------|---------|---------|---------|
| GB/T 34120-2023 | PCS（储能变流器） | 低电压穿越、高电压穿越、过载能力、效率、谐波、并网适应性 | 国家能源局指定实验室（如中国电力科学研究院） |
| GB/T 36558-2018 | 锂电池储能系统 | 电池单体/模组/簇的安全测试（针刺、过充、短路、热失控） | 具备CNAS/CMA资质的第三方检测机构 |
| GB/T 34070-2017 | BMS | SOC估算精度、均衡能力、过压/欠压保护 | 第三方检测机构 |
| GB/T 19964 | 光伏/储能并网逆变器 | 电能质量、功率因数、电网适应性 | 国家认可实验室 |
| NB/T 42088 | 电化学储能系统 | 整体系统效率、辅助功耗、噪声 | 电力行业权威检测机构 |
| IEC 62619 | 储能电池舱（出口项目） | 安全测试、热失控传播 | 国际认证机构（如TÜV、UL） |

> **第三方测试要求**：大储项目（≥10MW）通常要求提供完整的PCS和BMS型式试验报告，且报告出具时间距并网申请日不超过2年。用户侧小储可适度简化，但高计计量装置的校准证书（计量检定证书）必须提供。

## 四、典型接入方案案例

### 案例一：10kV用户侧工商业储能接入方案（浙江）

**项目背景**：
- 项目名称：浙江宁波某工业园区 2MW/4MWh 用户侧储能项目
- 投资方：浙江某能源服务公司
- 建设地址：宁波鄞州某工业园区内，距10kV专线约200m
- 并网电压：10kV，一回T接至园区专用10kV线路
- 储能配置：1MW/2MWh × 2 = 2MW/4MWh，共2套储能单元，每套含2台500kW PCS、1台1250kVA变压器

**接入方式**：
```
110kV鄞州变电站 10kV侧
  │
  ├─10kV专线（园区供电专线，型号：YJV-3×300，载流量500A）
  │    │
  │    ├─[T接点A]─分支线─园区10kV配电室（负荷）
  │    │
  │    └─[T接点B]─储能专线（型号：YJV-3×120，150m）─储能并网柜
                                                   │
                                                   ├─计量柜（高供高计）
                                                   ├─1号储能单元（1MW/2MWh）
                                                   └─2号储能单元（1MW/2MWh）
```

**并网柜配置**：
| 柜体 | 设备 | 参数 |
|------|------|------|
| #1储能进线柜 | 真空断路器 + 隔离开关 + CT（400/5A） | 额定电流630A，遮断31.5kA |
| #2计量柜 | 高压计量柜（主表+副表，三相四线） | 0.5S级，0.2S级各1块 |
| #3出线柜 | 断路器 + 隔离开关 + 零序CT | 同#1配置 |
| #4母线设备柜 | 避雷器 + PT柜 | 氧化锌避雷器 |

**调度接口配置**：
- 调度管辖：宁波地调
- 通信规约：IEC104，端口2404
- AGC接口：接收设点命令（类型50），响应时间≤4s，功率变化率≤0.4MW/min（20%×2MW/min）
- AVC接口：功率因数控制（目标 cos φ = 0.95）
- 纵向加密：部署纵向加密装置（SM1算法），与宁波地调主站建立IPSec隧道
- 远动数据：遥测22点、遥信32点、遥控8点、遥脉4点

**申请流程与时间节点**：

| 阶段 | 时间节点 | 工作内容 | 责任方 |
|------|---------|---------|-------|
| 1. 接入申请 | D-90 | 向宁波供电公司提交接入申请报告（含电气计算） | 储能投资方 |
| 2. 方案评审 | D-60 | 供电公司组织接入方案评审，出具评审意见 | 供电公司 |
| 3. 设计对接 | D-60~D-30 | 与供电公司确认并网柜、计量、保护的详细配置 | 设计院+供电公司 |
| 4. 设备采购 | D-45~D-15 | 采购PCS/BMS/并网柜/远动装置，签订调度协议 | 储能投资方 |
| 5. 施工安装 | D-30~D+15 | 完成电气施工、通信通道开通、设备安装 | 施工方 |
| 6. 验收并网 | D+30 | 供电公司验收，签署并网协议，正式并网 | 双方 |

**关键注意事项**：
- 浙江要求用户侧储能AGC指令响应时间≤4s，储能EMS需具备快速跟踪能力
- 储能专线T接点需经供电公司确认，并提供T接点加固方案（如原线路载流量不足）
- 型式试验报告须提前准备，重点关注 GB/T 34120-2023 和 GB/T 36558

### 案例二：35kV电网侧大储接入方案（江苏）

**项目背景**：
- 项目名称：江苏淮安某共享储能电站100MW/200MWh接入系统工程
- 投资方：江苏某大型能源集团
- 建设地址：淮安郊区，距220kV淮阴变电站约3.5km
- 并网电压：35kV，两回专线接入220kV淮阴变电站35kV母线
- 储能配置：10MW/20MWh × 10套 = 100MW/200MWh，每套独立并网柜
- 调度管辖：江苏省级调度（省调直调）

**接入点选择**：
经技术经济比较，推荐接入220kV淮阴变电站35kV侧，具体理由：
1. 淮阴变电站35kV母线短路电流水平为25.5kA，储能接入后仍满足断路器遮断要求
2. 变电站有备用35kV间隔2个，可直接使用，无需新增开关柜
3. 储能至变电站距离3.5km，采用双回35kV钢芯铝绞线（LGJ-240）投资约280万元

**电气主接线**：
```
220kV淮阴变电站 35kV母线
     │
     ├──35kV间隔1──35kV储能专线1（3.5km，LGJ-240）──储能开关站35kV母线
     │                                                    │
     │                                       ┌──────────┼───────────┐
     │                                       │          │           │
     │                                   1号储能单元  2号储能单元  ...10号储能单元
     │
     └──35kV间隔2──35kV储能专线2（备用通道，迂回路由）──[与间隔1并联]
```

**潮流计算结论（正常方式最大放电）**：

| 节点 | 电压(kV) | 有功功率(MW) | 无功功率(Mvar) | 备注 |
|------|---------|-------------|--------------|------|
| 220kV淮阴变35kV侧 | 36.1 | -90（受进） | +12.5 | 储能满放，功率因数0.99 |
| 储能开关站35kV母线 | 35.8 | +100（注入） | +14.0 | 含无功补偿投入 |
| 单回35kV线路 | 35.8→36.1 | 50.2 | 7.1 | 线路损耗0.8% |
| 储能单元侧（10kV侧） | 10.3 | 10×9.8=98 | 13.5 | 逆变侧损耗约1.5% |

**并网柜和保护配置**：
- 每套储能单元配置一面35kV进线柜（含纵联差动保护）+ 一面35kV计量柜
- 开关站35kV母线配置母线差动保护（数字式，保护配置在开关站保护屏）
- 储能侧保护：纵联差动保护（光纤通信，差动保护动作时间≤30ms）+ 方向过流保护（后备）+ 低频/高频保护
- 电网侧：淮阴变35kV间隔配置距离保护，与储能侧差动保护配合

**调度接口配置**：
- 调度管辖：江苏省调直调（省调SCADA系统）
- AGC指令：接收省调AGC系统设点命令（类型50），响应时间≤4s，功率变化率按省调要求（通常50MW/min）
- AVC指令：接收省调AVC系统无功设点或功率因数指令
- 通信通道：主通道采用调度数据网（纵向加密）经省调数据中心，备用通道经地调迂回
- 远动装置配置：主备双机，远动数据量：遥测200+点、遥信400+点、遥控50+点、遥脉20+点
- 对时方案：主用NTP（省调NTP服务器），备用IRIG-B（北斗时钟）

**并网申请特殊要求**：
- 须向国网江苏电力公司提交接入系统方案设计报告，通过省级评审
- 需签订省调调度协议（省调直调项目协议条款更严格）
- 须提供完整的PCS和BMS型式试验报告（GB/T 34120、GB/T 36558）
- 需通过电力工程质量监督站的质量监督检查（中间验收+整套启动验收）
- 调度数据网纵向加密装置须通过国家电网安全检测

### 案例三：0.4kV低压并网（小区光储微网）

**项目背景**：
- 项目名称：苏州某住宅小区光储微网项目（分布式光伏+储能）
- 光伏装机：100kWp（屋顶光伏）
- 储能配置：50kW/200kWh（磷酸铁锂）
- 并网电压：0.4kV（低压侧并网）
- 特殊要求：小区物业配电容量有限，储能需与光伏协调运行

**接入方式**：
```
10kV配电变压器低压侧（0.4kV）
  │
  ├── 总开关柜
  │    │
  │    ├── 居民用电负荷（已接）
  │    ├── 光伏逆变器（100kW）──光伏板阵
  │    └── 储能并网柜（50kW/200kWh）──储能电池系统
  │         │
  │         ├── 计量表（低压计量，低供低计）
  │         ├── 断路器（630A，三相四极）
  │         └── 防雷保护（II级SPD）
```

**低压接入的特殊要求**：

| 项目 | 要求 | 说明 |
|------|------|------|
| 容量限制 | 储能放电功率 ≤ 配电变压器容量的 25% | 防止储能放电导致变压器过载 |
| 反向功率 | 储能充电功率 ≤ 光伏出力（或配变剩余容量） | 防止储能充电导致配变反向功率超标 |
| 电压偏差 | 储能并网点电压偏差 ≤ ±7%（230V系统即 214~246V） | 低压配电网阻抗较大，电压偏差更敏感 |
| 谐波要求 | 总谐波畸变率（THD）≤ 5% | PCS须满足 GB/T 14549 并通过检测 |
| 保护简化配置 | 可使用塑壳断路器 + 剩余电流动作保护器（RCD） | 保护配置较高压侧简化，但须满足 GB 50054 |

**保护配置简化方案**：

| 保护功能 | 配置 | 整定值 |
|---------|------|--------|
| 过流保护（短路） | 塑壳断路器（630A，Ics=25kA） | 瞬时动作（电磁脱口器），整定 5~10倍额定 |
| 过负荷保护 | 断路器热磁脱口器 | 长延时动作，整定 1.13~1.45倍额定 |
| 漏电保护 | 剩余电流动作保护器（300mA） | 动作时间 ≤ 0.1s |
| 欠压/过压保护 | EMS 内置软件保护 | 欠压 < 195V，过压 > 253V，0.1s 跳闸 |
| 防孤岛保护 | PCS 内置防孤岛功能 | 检测到孤岛后 0.2s 内断开 |

**调度接口配置（低压储能）**：

| 项目 | 配置 | 说明 |
|------|------|------|
| 调度接入 | 通常免调度直调（功率 < 100kW） | 不接入调度数据网，但须向供电公司报备 |
| 监控要求 | 接入当地电网的分布式能源管理平台（通常为省级平台） | 通过4G/光纤上传运行数据 |
| 通信规约 | Modbus TCP 或 DL/T 645（电能表规约） | 通过 EMS 统一采集后转发 |
| 保护通信 | 无需纵向加密 | 功率小，安全防护要求较高压侧降低 |
| 对时要求 | NTP 或 GPS 时钟（可选） | 通常使用 NTP 对时即可 |

**并网申请流程（简化）**：
1. 向当地供电公司提交《分布式储能并网申请表》
2. 提交设备参数表（含PCS/BMS认证证书）
3. 提交电气接线图和保护配置图
4. 供电公司现场验收，检查并网柜和计量装置
5. 签署《分布式储能并网协议》
6. 装表并网

## 五、附录与工具

### 附录A：常用计算公式

**线路载流量计算**：
```
I_max = S / (√3 × U_n)

其中：
I_max = 允许载流量（A）
S = 导线截面积（mm²）
U_n = 额定电压（kV），10kV系统取10.5kV
√3 = 1.732

示例：LGJ-240 导线在10kV系统中的允许载流量
I_max = 240 × 修正系数 / (√3 × 10.5) ≈ 500A（标准条件）
```

**短路电流计算（无限大电源系统假设）**：
```
I_k'' = U_n / (√3 × Z_k)

其中：
Z_k = 线路阻抗（Ω）= R + jX
R = r × L（电阻率 × 长度）
X = x × L（电抗率 × 长度）

示例：10kV线路长度2km，r=0.27Ω/km，x=0.35Ω/km
Z_k = √(0.27×2)² + (0.35×2)² = √0.2916 + 0.49 = 0.883Ω
I_k'' = 10.5 / (1.732 × 0.883) = 6.85kA
```

**功率因数与无功补偿计算**：
```
Q_c = P × (tan φ1 - tan φ2)
tan φ = √(1/cos²φ - 1)

示例：储能额定功率 2MW，目标功率因数 0.95（滞后），现有 cos φ = 0.90
tan φ1 = √(1/0.9² - 1) = √(1.234 - 1) = 0.484
tan φ2 = √(1/0.95² - 1) = √(1.107 - 1) = 0.329
Q_c = 2000 × (0.484 - 0.329) = 310 kvar（无功补偿容量）
```

## 使用说明

本Skill作为储能项目并网交付的标准化工具，可按以下方式使用：

1. **项目启动阶段**：查阅第二章（接入点选择）和第三章（电气计算），确定接入方案方向
2. **报告编制阶段**：直接引用第三章的报告模板，填充项目数据
3. **调度配置阶段**：按第四章（调度接口）配置远动和AGC/AVC参数，填写调度配置表
4. **并网申请阶段**：按第三章（并网申请材料清单）准备材料，逐一核对清单
5. **案例参考**：当遇到类似项目（10kV用户侧/35kV电网侧/0.4kV低压）时，参考第四章典型案例的配置和流程

如遇各省电网的特殊要求（如浙江要求、江苏要求），需另行查阅当地电网并网技术规范，并与当地供电公司调度部门确认。
