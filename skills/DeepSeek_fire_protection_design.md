## 二、气体探测与时间窗口

### 2.1 热失控演化时间线 (H₂→CO→VOC→HF)

电池热失控是一个分阶段的物理化学过程，不同阶段释放的特征气体可用于分级预警：

```
时间轴： T0(正常) ─► T+30min(升温) ─► T+60min(SEI分解) ─► T+90min(电解液分解) ─► T+120min(热失控爆发) ─► T+150min(火灾)
    │              │                 │                       │                       │                     │
    ▼              ▼                 ▼                       ▼                       ▼                     ▼
  无气体释放    H₂ 缓慢增加      CO/H₂ 显著增加         VOC 大量产生           HF 产生+烟雾          明火+爆炸
  状态: 正常    状态: 早期预警    状态: 中期警告          状态: 晚期警告         状态: 危急            状态: 灾难
```

**四级检测窗口精确定义：**

| 阶段 | 时间窗口 | 特征气体 | 温度范围 | 告警级别 | 响应措施 |
|------|----------|----------|----------|----------|----------|
| **W1 早期预警** | 正常→SEI分解(T+30~60min) | H₂≥100ppm, CO≥20ppm | <55°C | 一级(监视) | BMS加强监控,启动冷却 |
| **W2 中期警告** | SEI分解→电解液分解(T+60~90min) | CO≥50ppm, H₂≥500ppm | 55~70°C | 二级(警告) | 声光报警,通知值守人员,限制功率 |
| **W3 晚期危急** | 电解液分解→热失控(T+90~120min) | VOC≥100ppm, HF≥3ppm | 70~90°C | 三级(火灾确认) | 消防联动,隔离电池簇,启动灭火 |
| **W4 终极确认** | 热失控爆发 (T>120min) | 烟雾浓度≥15%, 火焰探测 | >90°C | 火灾报警 | 全联动,紧急停机,排烟启动 |

### 2.2 各阶段气体浓度阈值与响应策略

| 检测气体 | 探测器类型 | 安装位置 | W1阈值 | W2阈值 | W3阈值 | W4阈值 | 响应时间 |
|----------|-----------|----------|--------|--------|--------|--------|----------|
| **H₂** (氢气) | 催化燃烧/电化学 | 舱顶部,距顶≤0.3m | 100ppm | 500ppm | 1000ppm | — | ≤10s |
| **CO** (一氧化碳) | 电化学(0~500ppm) | 舱中部1.5~2.0m | 20ppm | 50ppm | 100ppm | — | ≤15s |
| **VOC** (挥发性有机物) | PID光离子化 | 舱顶部,每舱≥2 | — | — | 100ppm | 200ppm | ≤20s |
| **HF** (氟化氢) | 电化学/红外 | 排气口附近 | — | — | 3ppm | 10ppm | ≤15s |
| **烟雾** | 光电感烟 | 均匀布置,间距≤5m | — | — | — | 15%衰减率 | ≤30s |
| **温度** | 线型感温/PT100 | 电池簇顶部 | ΔT≥8°C | 55°C | 60°C | 70°C | ≤5s |

### 3.2 全氟己酮(Novec 1230)用量计算

> **公式**：$W = \frac{V}{S} \times C \times K$

**参数定义：**

| 参数 | 含义 | 单位 | 典型值/取值方法 |
|------|------|------|----------------|
| $W$ | 灭火剂设计用量 | kg | 计算结果 |
| $V$ | 防护区净容积 | m³ | 总容积—不可燃物体体积 |
| $S$ | 灭火剂20°C比容积 | m³/kg | Novec 1230: 0.067 (20°C, 1atm) |
| $C$ | 设计浓度 | % | 锂电火灾: 6.0% |
| $K$ | 海拔修正系数 | — | 海拔≤1000m: K=1.0; 1000~2000m: K=1.1; >2000m: K=1.2 |

**补充修正：**

```
温度修正系数 Kt:
  当环境温度≠20°C时：
  Kt = (273.15 + T_ambient) / (273.15 + 20)
  
喷放时间要求 ts:
  全氟己酮喷放时间 ≤10s (GB 50370要求)
  实际设计喷放时间取 8~10s
  
喷嘴设计流量 q_nozzle:
  q_nozzle = W / ts / n_nozzle
```

### 3.3 细水雾系统水力计算

> **设计流量公式**：$Q = V \times q_{design}$

| 参数 | 公式 | 说明 |
|------|------|------|
| 设计流量 Q | $Q = V \times q$ | V: 保护体积(m³), q: 喷放强度(L/min·m³) |
| 喷放强度 q | 3~6 L/min·m³ (中压) / 2~4 L/min·m³ (高压>10MPa) | 锂电火灾建议取6 L/min·m³ |
| 储水量 S | $S = Q \times t$ | t: 持续喷雾时间, 锂电火灾≥60min |
| 泵功率 P | $P = \rho g Q H / \eta$ | H: 扬程(m), η: 泵效率(0.70~0.85) |
| 管径 D | $D = \sqrt{4Q/(\pi v)}$ | v: 经济流速(主管2~3m/s, 支管1.2~1.5m/s) |

### 3.4 CO₂ 全淹没系统计算

> **公式**：$W = K_b \times (0.2A + 0.75V)$ (表面火灾) 或 $W = K_b \times (0.2A + 0.75V) \times M$ (深位火灾)

| 防护区类型 | A | V | Kb | 最小设计浓度 | 喷放时间 |
|------------|---|-----|-----|-------------|---------|
| 电气设备(表面火灾) | 保护区面积m² | 净容积m³ | 1.0(低压) / 1.4(高压) | 34% (表面) | ≤1min |
| 电缆夹层(深位火灾) | 同上 | 同上 | 同上 | 50% | ≤7min |
| 锂电池火灾 | 同上×1.3 | 同上 | Kb×1.2 | 75% | ≤2min |

### 3.5 三种设计计算示例

#### 示例一：全氟己酮灭火系统 (40ft集装箱舱)

**已知条件：** 舱体12m×2.4m×2.8m, 净容积80m³(扣除设备10m³), 海拔500m

```
步骤1: 设计浓度 C = 6.0% (锂电, Novec 1230)
步骤2: 比容积 S = 0.067 m³/kg @20°C
步骤3: 海拔修正 K = 1.0 (≤1000m)
步骤4: 灭火剂用量 W = V/S × C × K = 80/0.067 × 0.06 × 1.0 = 71.64kg
步骤5: 取整 W = 72kg, 选用 2×40kg瓶组
步骤6: 喷嘴数量: 每喷嘴保护≤12m², 80÷12≈7个
步骤7: 每喷嘴流量: 72÷7÷10s = 1.03kg/s
步骤8: 喷放时间验证: 72÷(7×1.03) = 10.0s ✓ (满足≤10s)
步骤9: 管道设计: DN40主管, DN25支管, 不锈钢无缝管承压≥6MPa

结论: 系统配置满足要求, 设计余量约12%
```

#### 示例二：高压细水雾系统 (5MW集装箱储能区)

**已知条件：** 保护体积V=200m³, 持续喷雾t=60min, 高压泵P≥10MPa

```
步骤1: 喷放强度 q = 4 L/min·m³ (高压细水雾)
步骤2: 设计流量 Q = 200 × 4 = 800 L/min = 13.33 L/s
步骤3: 储水量 S = 800 × 60 = 48000 L = 48m³
步骤4: 选用不锈钢水箱 50m³
步骤5: 泵扬程计算 (高差+沿程+末端10MPa):
       H_total = 10m(高差) + 5m(沿程) + 1000m(10MPa) = 1015m
步骤6: 泵功率 P = 1000 × 9.81 × 0.01333 × 1015 / 0.75 = 177.2kW
       选用 200kW 高压柱塞泵 (1用1备)
步骤7: 主管径 D = √(4×0.01333/(3.14×2.5)) = 0.082m → DN100
步骤8: 喷嘴布置: 每喷嘴流量3.5L/min@10MPa, 数量=800/3.5≈229个
       实际布置: 15列×16行=240个, 间距1.5m×1.5m

结论: 系统配置满足60min持续冷却要求, 冗余20%
```

#### 示例三：联动时序验证 (2MW项目)

**已知条件：** 要求告警到灭火启动总时间≤2s

```
时序分解:
┌────────────────────────┬──────────┬──────────┬──────────┐
```

## 四、消防联动时序设计

### 4.1 告警→联动的完整时序链

```
█ 消防告警联动完整时序链 (目标总时间 ≤ 2000ms)

T=0ms        探测器物理触发 (气体浓度超阈值/温度超限/烟雾衰减超限)
    │
T=0~50ms     信号电缆传输至火灾报警控制器 (NH-KVV 2×1.5mm²)
    │
T=50~100ms   报警控制器ADC采样+滤波+阈值比较+报警判定
    │
T=100~400ms  报警确认逻辑 (可配置延时0~300ms, 默认50ms)
    │         ├─ 1oo1: 单探测器直接输出 (0ms)
    │         ├─ 1oo2: 任意一个触发 (0ms)
    │         ├─ 2oo3: 三取二表决 (50ms确认)
    │         └─ 延时确认: 持续报警Nms后触发 (默认50ms)
    │
T=150~450ms  联动控制器输出硬接线信号: ①→BMS停机 ②→PCS急停
    │
T=250~550ms  BMS接收到停机指令, 执行电池簇断电
    │
T=300~600ms  BMS→PCS硬接线急停信号
    │
T=500~800ms  PCS执行IGBT封锁, 停止功率转换
    │
T=600~900ms  高压断路器接收分闸信号, 启动分断
    │
T=800~1100ms 灭火系统瓶组电磁阀开启 (两步确认: ①报警确认 ②手动/延时触发)
    │
T=1000~1300ms 灭火剂管道充压, 喷头开始喷射
    │
T=1300~2000ms 灭火剂持续喷射, 火灾被抑制
    │
T=2000ms+    持续监控: 温度/气体浓度监测 → 确认灭火成功/复燃检测
```

### 4.2 硬接线 vs 软件联动双通道设计

| 对比维度 | 硬接线通道 (Primary) | 软件通信通道 (Secondary) | 双通道保障 |

| 成本 | 高(独立线缆+继电器) | 低(复用网络) | 总成本增加<15% |

|----------|---------------|---------|--------|---------|---------|
| **磷酸铁锂(LFP)** | 600~800°C | H₂+CO为主, 中等产气量 | 少 | 中 | 水喷淋优先, 需持续冷却≥30min |
| **三元锂(NCM)** | 900~1200°C | CO+CO₂+HF+O₂, 产气量大 | **有**(热分解释放O₂) | **极高** | 强制水喷淋+持续冷却≥2h, 气体灭火剂无效 |
| **钛酸锂(LTO)** | 400~600°C | 产气量较小 | 无 | 低 | 标准水喷淋/细水雾 |
| **钠离子电池** | 450~650°C | H₂+CO为主, 产物含Na | 无 | 中 | 细水雾优先, 防止电解液流淌 |
| **全钒液流电池** | 不适用(水基) | — | 无 | 极低 | 抗腐蚀喷淋+围堰收集, 关注V₂O₅毒性 |
| **固态电池** | 600~800°C(理论) | 少量H₂+CO | 极少 | 低 | 标准水喷淋即可 |

### 8.2 各电池类型专用探测策略

| 电池类型 | 关键检测气体 | 附加传感器 | 特殊注意事项 |
|----------|-------------|-----------|-------------|
| LFP | CO+H₂ | 线型感温(每簇) | HF为终极确认指标,≤3ppm立即联动 |
| NCM | CO+H₂+O₂ | O₂传感器(检测氧释放)+线型感温(密切) | 氧浓度>22%视为热失控前兆,立即执行停机 |
| 钠离子 | CO+H₂+VOC | 电解液蒸气(PID) | 钠盐沉积可能污染探测器,需每月清洁 |
| 液流电池 | H₂(充电副产物) | pH传感器(电解液泄漏) | 电解液泄漏触发围堰泵启动,中和服务 |
| 固态电池 | CO+H₂ | 声学传感器(电池壳体形变) | 固态电池失效模式以壳体变形为主 |

### 8.3 NCM电池特殊消防设计

由于三元锂(NCM)电池热失控会自身释放O₂(氧化剂),因此:
- **气体灭火剂(七氟丙烷/Novec 1230/CO₂)对NCM火灾基本无效**
- 必须采用水基灭火系统(水喷淋/细水雾)
- 持续冷却时间需≥2小时(等待电池能量完全耗尽)
- 消防用水量: LFP的2~3倍
- 须配置泄压面积: ≥0.20m²/m³(LFP要求≥0.16m²/m³)

## 十二、电池热失控抑制系统(水冷/浸没)

### 12.1 电池簇级液冷消防一体化设计

```
液冷+消防一体化方案:
  正常运行时: 液冷板维持电池温度25~35°C (电池簇间温差≤3°C)
  热失控初期: 液冷板增大流量, 泵转速提升至120%, 目标降温速率≥5°C/min
  热失控确认: 切换至消防模式, 液冷管路阀组切换, 细水雾通过液冷管路直喷电池簇内部

系统配置:
  液冷机组: 制冷量≥电池发热量×1.5
  消防泵: 高压柱塞泵(≥10MPa), 1用1备
  阀组: 电动三通阀(切换冷却/消防模式), 响应≤1s
  管路: SS316L不锈钢, 耐腐蚀(应对HF酸性气体)
  喷嘴: 集成于液冷板内, 每模组1~2个, K系数0.5~1.0
```

### 12.2 浸没式液冷消防方案

```
浸没式液冷(Submersion Cooling):
  介质: 氟化液(Novec 7000系列) / 矿物油
  优点: 电芯浸没在绝缘冷却液中, 热量直接传导, 隔绝氧气
  消防优点: 浸没介质本身即是灭火介质, 热失控时无明火风险

系统参数:
  浸没槽容积: 电池体积×1.5
  介质循环: 泵流量≥500L/min(MW级系统)
  介质过滤: 5μm过滤器, 防止导电颗粒积聚
  温度控制: 介质温度≤35°C, 热失控时紧急排热至外部散热器
```

### 14.1 三级应急响应流程

```

  [动作] 声光报警启动 / BMS限制功率50% / 启动冷却系统
    ↓
  [通知] 值班人员电话+短信 / 运维主管短信
    ↓
  [到场] 运维人员30min内到场确认
    ↓
  [升级] 如5min内人工未确认→自动升级三级

三级响应 (火灾确认 - 三重确认或手动触发):
  [触发] VOC≥100ppm OR HF≥3ppm OR 手动按钮 OR 温度>60°C

  [动作] PCS急停 / 断路器分闸 / 排烟启动 / 灭火喷射
    ↓
  [通知] 119火警 / 公司消防负责人 / 应急指挥中心
    ↓
  [疏散] 全站人员撤离至安全集合点
    ↓
  [持续] 灭火后持续监测≥2h, 确认无复燃
    ↓
  [恢复] 人工评估安全后, 逐步恢复供电
```
