## 一、电气主接线图（SLD）审图要点

### 1.1 常见主接线形式及适用场景

#### 1.1.1 单母线接线（Single Bus）

```
         ┌─────────┐
  ───────┤  母线   ├───┬─── 馈线1
         │         │   ├───┬─── 馈线2
  进线 ──┤         │   │   └─── 馈线3
         └─────────┘   │
```

| 特征 | 说明 |
|------|------|
| 优点 | 接线简单清晰，设备少，投资省 |
| 缺点 | 母线或母线隔离开关故障时全站停电，检修时需停运全部馈线 |
| 适用场景 | 小型储能电站（装机≤1MW）、不重要负荷、临时用电 |
| 储能场站适用性 | 几乎不推荐用于并网储能场站 |

#### 1.1.2 单母线分段接线（Single Bus with Sectionalized）

```
         ┌───────┬───────┐
  ───────┤段1母线├段2母线├───┬─── 馈线1
  进线1 ─┤       │       │   ├───┬─── 馈线2
         │       │       │   │   └─── 馈线3
         └───┬───┘───┬───┘   │
             │分段断路器│
```

| 特征 | 说明 |
|------|------|
| 优点 | 母线故障时只影响一半负荷，可用分段断路器转移负荷 |
| 缺点 | 检修进线或主变时需全站停电，馈线侧操作不便 |
| 适用场景 | 中小型储能电站（装机1~10MW）、有备用电源的场景 |
| 储能场站适用性 | 10kV并网的小型储能项目常用此方式 |

#### 1.1.3 双母线接线（Double Bus）

| 特征 | 说明 |
|------|------|
| 优点 | 检修任意母线时不影响供电，调度灵活 |
| 缺点 | 设备多，操作复杂，继电保护配置复杂 |
| 适用场景 | 大型储能电站、需频繁倒闸操作的重要负荷 |
| 储能场站适用性 | 35kV及以上电压等级的大型储能项目可能采用 |

#### 1.1.4 简易环网接线（简易环网 / 拉手式）

```
    ┌──────────────────────────────┐
    │           10kV母线            │
    └──┬────────┬────────┬─────────┘
       │PCS1    │PCS2    │PCS3
     ┌─┴─┐   ┌─┴─┐   ┌─┴─┐
     │   │   │   │   │   │
     └───┘   └───┘   └───┘
       │       │       │
    ───┴───────┴───────┴──── 10kV电网
```

| 特征 | 说明 |
|------|------|
| 优点 | 任意一路电源失电，另一路可承担全部负荷 |
| 缺点 | 需两路独立电源，工程条件要求高 |
| 适用场景 | 工商业储能微电网、与柴发配合的可靠供电系统 |
| 储能场站适用性 | 微电网型储能项目的首选接线方式 |

#### 1.1.5 储能场站典型接线选择速查

| 并网电压等级 | 推荐接线形式 | 典型装机规模 | 说明 |
|-------------|------------|-------------|------|
| 0.4kV 低压并网 | 低压配电柜单母线 | ≤400kW | 就近接入公变，低压侧直连 |
| 10kV 中压并网 | 单母线分段或简易环网 | 400kW~10MW | 最常见配置 |
| 35kV 高压并网 | 单母线分段（大型站）或双母线 | 10MW~100MW | 电网侧升压站配置 |
| 110kV 超高压并网 | 双母线或3/2断路器 | ≥100MW | 共享储能电站 |

## 二、电缆选型计算

### 2.1 电缆截面选择三原则

#### 2.1.1 温升校验（按持续允许电流选截面积）

**核心公式**：
```
I_z ≥ I_max
```
- `I_z`：电缆在给定敷设条件下的持续允许载流量（A）
- `I_max`：电缆所在回路最大持续工作电流（A）

**最大持续工作电流计算**：
```
交流三相回路：
  I_max = P_max / (√3 × U_n × cosφ × η)

直流回路：
  I_max = P_max / (U_n × η)

其中：
  P_max = 设备额定功率（kW）
  U_n = 额定电压（kV）
  cosφ = 功率因数（通常取0.95）
  η = 效率（通常取0.98）
```

**【计算示例】PCS交流侧电缆选型**

已知条件：
- PCS额定功率：P_N = 500kW
- 额定电压：U_n = 10kV
- 功率因数：cosφ = 0.95（进相）
- 效率：η = 0.98
- 敷设方式：桥架敷设（空气中），环境温度40°C

计算步骤：
```
Step 1: 计算最大持续工作电流
  I_max = 500 / (1.732 × 10 × 0.95 × 0.98)
        = 500 / 16.13
        ≈ 31.0 A

Step 2: 确定敷设修正系数
  环境温度修正系数 K_1 = 0.91（40°C，XLPE绝缘，GB/T 12706）
  多根电缆并排修正系数 K_2 = 0.85（6根电缆同层敷设）
  综合修正系数 K = K_1 × K_2 = 0.91 × 0.85 = 0.774

Step 3: 计算修正后所需载流量
  I_z_needed = I_max / K = 31.0 / 0.774 ≈ 40.1 A

Step 4: 查载流量表选截面积
  查 10kV XLPE绝缘三芯电缆空气中敷设（K_1=1时）：
    3×25mm² → I_z = 105A  → 修正后 I_z' = 105 × 0.774 ≈ 81.3A ≥ 40.1A ✓
    3×16mm² → I_z = 87A   → 修正后 I_z' = 87 × 0.774 ≈ 67.4A ≥ 40.1A ✓

  推荐选型：3×25mm² YJV22-8.7/15kV（考虑一定裕量）
```

#### 2.1.2 电压降校验（△U ≤ 5%）

**核心公式（交流三相）**：
```
ΔU = √3 × I × L × (r × cosφ + x × sinφ) / U_n

电压降百分比：
ΔU% = ΔU / U_n × 100%

简化计算（忽略电抗，铜芯电缆近似）：
ΔU% = (P_max × L) / (γ × S × U_n²) × 100

其中：
  I = 负荷电流（A）
  L = 电缆长度（km）
  r = 交流电阻（Ω/km），铜芯≈0.125Ω/km（10kV级）
  x = 感抗（Ω/km），三芯电缆≈0.1Ω/km
  γ = 铜导电率（57，S/m）
  S = 电缆截面积（mm²）
  U_n = 额定线电压（V）
```

**【计算示例】电压降校验**

已知条件（延续上例）：
- 电缆长度：L = 0.3 km
- 选用截面：S = 25 mm²
- 铜芯电阻（20°C）：r_20 = 0.727 Ω/km
- 修正到工作温度90°C：r_90 = r_20 × 1.12 = 0.814 Ω/km
- 功率因数：cosφ = 0.95

计算步骤：
```
Step 1: 计算电压降
  ΔU = √3 × 31.0 × 0.3 × (0.814 × 0.95 + 0.1 × 0.312) / 10000
      = 1.732 × 31.0 × 0.3 × 0.807 / 10000
      = 1.732 × 31.0 × 0.3 × 0.807 / 10000
      ≈ 1.30 V

Step 2: 计算电压降百分比
  ΔU% = 1.30 / 10000 × 100% = 0.013%

Step 3: 判定
  0.013% ≤ 5%  ✓ 满足要求

注：0.3km的10kV线路压降非常小，本案例即使选用6mm²也满足压降要求。
    但实际选型需综合温升、短路耐受和机械强度要求。
```

**低压侧电压降特殊要求**：
- 储能电池簇至PCS DC侧：△U ≤ 1%（因电压低，电流大，压降影响明显）
- 低压动力回路（400V）：△U ≤ 5%，但对于长距离（>100m）建议逐段校验

#### 2.1.3 经济电流密度选截面积（年最大负荷利用小时数决定）

**核心公式**：
```
S_e = I_max / J

其中：
  S_e = 经济截面积（mm²）
  I_max = 最大持续工作电流（A）
  J = 经济电流密度（A/mm²），查表取值
```

**经济电流密度表（GB/T 15860 / 铜导体）**：

| 年最大负荷利用小时数 T（h） | 经济电流密度 J（A/mm²） |
|--------------------------|------------------------|
| <3000（峰谷调节） | 2.50 |
| 3000~5000（工商业储能） | 2.00 |
| 5000~7000（削峰填谷） | 1.50 |
| >7000（长期调频/备用） | 1.00 |

**【计算示例】经济电流密度选型**

已知条件：
- PCS额定功率：P_N = 500kW
- 最大工作电流：I_max = 31.0 A
- 年利用小时数：T = 4000h（工商业储能，每日2充2放）
- cosφ = 0.95

计算步骤：
```
Step 1: 查经济电流密度
  T = 4000h → J = 2.00 A/mm²

Step 2: 计算经济截面积
  S_e = 31.0 / 2.00 = 15.5 mm²

Step 3: 选标称截面积
  取标准截面 16mm²

Step 4: 综合判定
  温升要求 → 25mm²（安全裕量）
```

## 三、电缆敷设设计

### 3.1 电缆通道设计

#### 3.1.1 电缆桥架选型

**桥架类型对比**：

| 桥架类型 | 材质 | 适用场景 | 载荷能力 | 耐火性 |
|---------|------|---------|---------|--------|
| 槽式（ закрытый） | 铝合金/钢制 | 控制电缆、弱电电缆 | 中 | 好 |
| 梯式（孔格式） | 钢制热镀锌 | 动力电缆、大截面电缆 | 高 | 一般 |
| 托盘式 | 铝合金/钢制 | 动力+控制混合 | 高 | 较好 |
| 耐火型槽盒 | 防火涂层钢制 | 消防配电、重要回路 | 中 | 优（耐火≥90min） |

**桥架尺寸计算方法**：
```
步骤1: 计算电缆总截面积
  S_total = Σ(S_i × n_i)
  其中 S_i 为单根电缆截面积，n_i 为电缆根数

步骤2: 确定填充率
  动力电缆：填充率 ≤ 40%（留维护空间）
  控制电缆：填充率 ≤ 50%
  混合敷设：填充率 ≤ 40%

步骤3: 计算桥架截面积
  S_bridge = S_total / η_fill
  其中 η_fill 为填充率（取0.40）

步骤4: 确定桥架宽度
  标准宽度系列：100 / 150 / 200 / 300 / 400 / 500 / 600 / 800mm
  选最接近且略大的标准宽度

步骤5: 计算桥架高度
  H = S_bridge / W_bridge
  标准高度系列：50 / 75 / 100 / 150 / 200mm
  选标准高度，且高度应保证电缆弯曲半径要求
```

**【计算示例】桥架选型**

已知条件：
- 电缆清单：
  - 10kV动力电缆 YJV22-8.7/15kV 3×95mm²：2根（外径d₁≈42mm）
  - 0.6/1kV动力电缆 WDZA-YJY 4×70mm²：6根（外径d₂≈38mm）
  - 控制电缆 KVV22：8根（外径d₃≈18mm）

计算步骤：
```
Step 1: 计算各电缆截面积
  10kV电缆截面积：A₁ = π×(42/2)²×2 = 3.14×441×2 ≈ 2771 mm²
  低压动力电缆：A₂ = π×(38/2)²×6 = 3.14×361×6 ≈ 6802 mm²
  控制电缆：A₃ = π×(18/2)²×8 = 3.14×81×8 ≈ 2036 mm²
  S_total = 2771 + 6802 + 2036 = 11609 mm²

Step 2: 取填充率40%
  S_bridge = 11609 / 0.40 ≈ 29023 mm²

Step 3: 选桥架尺寸
  设桥架宽度W = 400mm
  所需高度 H = 29023 / 400 ≈ 72.6mm
  选标准规格：400×100mm 槽式桥架

Step 4: 校验填充率
  实际截面积 = 400×100 = 40000mm²
  实际填充率 = 11609 / 40000 = 29.0% ≤ 40% ✓
  
  推荐选型：400×100mm 槽式钢制桥架（热镀锌）
```

#### 3.1.2 电缆沟设计

**电缆沟截面设计原则**：

| 参数 | 设计要求 |
|------|---------|
| 最小宽度 | 单侧人行通道 ≥ 800mm；双侧人行 ≥ 1000mm |
| 最小深度 | 电缆距沟底 ≥ 100mm，距沟顶 ≥ 150mm |
| 排水设计 | 沟底设0.5%坡度，底部设排水点或集水井 |
| 防火分隔 | 每50~100m设一道防火隔板（耐火极限≥1h） |
| 防水处理 | 户外电缆沟需做防水层，沟壁加防水涂层 |
| 接地 | 金属桥架每隔20m接地一次，接地电阻≤10Ω |

**电缆沟截面速算（经验公式）**：
```
电缆沟宽度（mm）= 电缆总宽度 + 2 × 100mm（每侧维护余量）
电缆沟深度（mm）= 最大电缆外径 + 100mm（底部余量）+ 100mm（顶部余量）
               + n层 × (最大电缆外径 + 20mm层间距)
```

#### 3.1.3 穿管设计

**保护管选型原则**：

| 保护管类型 | 材质 | 适用场景 | 规格 |
|-----------|------|---------|------|
| SC（焊接钢管） | 钢 | 承受机械外力处、埋地 | DN25~DN200 |
| KBJ（扣压式薄壁钢） | 钢 | 室内明装、承受一般外力 | DN20~DN100 |
| PVC管 | 聚氯乙烯 | 无机械外力、干燥环境 | DN20~DN100 |
| PE管 | 聚乙烯 | 埋地、电缆穿管 | DN32~DN200 |
| 金属软管 | 钢制镀锌 | 设备进出口、弯曲处 | DN20~DN100 |

**管径选择公式**：
```
d ≥ 1.05 × d_max_cable

其中：
  d = 保护管内径（mm）
  d_max_cable = 电缆外径（mm）
  1.05 = 间隙系数（便于穿线）

单根电缆：管内径 ≥ 1.05 × 电缆外径
多根电缆：Σ(电缆截面积) ≤ 0.4 × 保护管截面积
```

| 防静电接地 | ≤ 100Ω（泄漏电阻） | 防静电工作区 |
| 联合接地 | ≤ 1Ω | 特殊要求（通信机房） |

**高土壤电阻率地区处理措施**：

| 处理方法 | 适用条件 | 效果 |
|---------|---------|------|
| 换土法 | 局部高阻区域 | 降低接地极周围土壤电阻率 |
| 深井接地 | 地下水位较高处 | 利用低阻土层 |
| 外引接地 | 场区外有低阻土壤 | 将接地极引至远处低阻区 |
| 降阻剂 | 一般高阻地区 | 化学降阻，经济有效 |
| 多支线接地极组合 | 高阻地区 | 增加接地极数量，降低整体电阻 |
| 接地模块 | 腐蚀环境/高阻地区 | 寿命长，降阻效果好 |

| 接地极类型 | 规格 | 适用场景 | 寿命 | 备注 |
|-----------|------|---------|------|------|
| 角钢接地极 | L50×50×5×2500mm热镀锌 | 一般地区 | 15~20年 | 最常用，经济 |
| 铜包钢接地极 | Φ20×2500mm | 腐蚀环境/高阻地区 | 30年+ | 耐腐蚀性能好 |

**接地极布置原则**：
- 水平接地极埋深：≥ 0.8m（冻土层以下）
- 垂直接地极长度：≥ 2.5m（打入式）
- 接地极间距：≥ 2倍长度（减少屏蔽效应）
- 接地网布置：采用环形或放射状网格，主干线截面积 ≥ 50mm²扁钢

#### 4.2.4 接地电阻计算公式

**接地电阻估算（简化公式）**：
```
单支垂直接地极（棒式）：
  R ≈ ρ / (2πL) × ln(4L/d)

其中：
  ρ = 土壤电阻率（Ω·m）
  L = 接地极长度（m）
  d = 接地极直径（m）

水平接地网：
  R ≈ ρ / (2πL) × ln(L² / (h·d))

其中：
  L = 接地线总长度（m）
  h = 埋设深度（m）
  d = 接地线等效直径（m）

并联多支接地极：
  R_total = R / (n × η)

其中：
  n = 接地极数量
  η = 利用系数（0.7~0.9，随数量增加而降低）
```

**【计算示例】接地电阻设计**

已知条件：
- 土壤电阻率：ρ = 100 Ω·m（一般黏土）
- 需接地电阻：R ≤ 10 Ω
- 选用角钢接地极：L50×50×5×2500mm（d=0.05m，L=2.5m）

计算步骤：
```
Step 1: 计算单支垂直接地极接地电阻
  R_single = 100 / (2π×2.5) × ln(4×2.5/0.05)
           = 6.37 × ln(200)
           = 6.37 × 5.30
           ≈ 33.7 Ω

Step 2: 确定所需接地极数量
  取利用系数 η = 0.8（6根布置）
  n = R_single / (R × η) = 33.7 / (10 × 0.8) ≈ 4.2
  取 n = 5 根

Step 3: 校验5根接地极并联后的接地电阻
  取 η = 0.75（5根布置，间距2倍长度=5m）
  R_total = 33.7 / (5 × 0.75) = 33.7 / 3.75 ≈ 8.99 Ω ≤ 10 Ω ✓

Step 4: 布置方案
  5根L50角钢，间距5m，环形布置，水平连接扁钢50×5mm
  埋深0.8m，接地网总长度约25m

  推荐接地方案满足要求
```

#### 4.3.1 变压器中性点接地方式（TN-S系统）

**TN-S系统特点**：
- 整个系统有单独的中性线N和保护线PE
- PE线与N线在变压器中性点分开后，不再合并
- 适用于对安全要求高的场所

**接线示意**：
```
           变压器
    10kV△ ──┬── Y侧
             │
          ┌──┴──┐
          │N    │
          └──┬──┘
             │N线
    ┌───────┼───────┐
    │       │       │
   L1      L2      L3
    │       │       │
    ├───────┼───────┤ PE保护线
    │       │       │ （黄绿双色）
    │       │       │
   设备    设备    设备
    │       │       │
    └───────┼───────┘
            │
          ┌─┴─┐
          │PE │
          └───┘
```

| 接地参数 | 要求 |
|---------|------|
| 变压器中性点接地方式 | TN-S系统 |
| 中性点接地电阻 | ≤ 4Ω（10kV侧） |
| 保护线PE截面积 | 不小于相线截面积的50% |
| PE线与N线分合要求 | 分开后不得再次合并 |
| 重复接地 | 配电室总配电箱处需做重复接地 |

#### 4.3.2 电池系统接地

| 接地类型 | 做法 | 说明 |
|---------|------|------|
| **悬浮接地（IT系统）** | 电池系统与大地绝缘，仅通过高阻接地（≥1MΩ） | 优点：单点接地故障不影响电池运行；缺点：绝缘监测复杂 |
| **单点接地** | 电池系统直流侧中性点一点接地 | 储能行业最常用方案，安全且监测简单 |
| **推荐方案** | 直流侧单点接地（电池簇汇流柜处设接地排） | PE接地电阻≤10Ω，直流系统绝缘监测 |

- 电池舱金属外壳 → PE线 → MEB
- 电池BMS工作接地 → 独立工作接地端子（不得与PE混接）
- 电池簇直流电缆金属屏蔽层 → 两端接地（接地电阻≤4Ω）
- PCS直流侧接大地 → DC侧接地继电器动作后接地

| 通信类型 | 接地方式 | 屏蔽层处理 |
|---------|---------|-----------|

| RS485总线 | 设备工作地接至通信接地排 | 屏蔽层一端接地（信号源侧） |
| 光缆 | 不需要屏蔽层接地 | 加强芯在终端机房接地 |
| CAN总线 | 设备工作地接至通信接地排 | 双绞线无屏蔽，仅信号地接地 |
| 无线通信（4G/5G） | 设备工作地接至通信接地排 | 天线馈线屏蔽层在机房侧接地 |

#### 6.5.2 三极法测试接线

```
        辅助电流极 P2
             │
    ─────┬───┴───┐
         │       │
        C1 P1 P2 C2
         ││││││││
    ─────┤├─┼─┼─┤├───── 接地网
         ││ │ │ ││
         ││ │ │ ││
         ││ │ │ ││
         ││ │ │ ││
    ════════════════════ 电位极 P1（被测接地网）
    （电位极位于电流极与接地网之间，约为0.62倍距离处）
```

**布线要求**：
- 电流极布置距离：d ≥ 4D（D为接地网最大对角线长度）
- 电位极布置位置：0.618d 处（理论最佳点）
- 电流极和电位极与被测接地网尽量成直线
- 电压极和电流极地桩应打入湿润土壤
