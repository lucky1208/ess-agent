# 投标文件 Skill v1.0

> 设计人：卢继雄 | 基于5年微网项目实战训练

## 概述
根据客户需求书自动生成技术方案、配置清单、偏离表、实施计划、验收标准。

## 输入
- 客户需求书 (PDF/Word)
- 微网方案JSON (拓扑/容量/设备/保护/策略)

## 输出
- 技术方案 (Word): 系统架构/设备选型/保护配置/策略说明
- 配置清单 (Excel): 序号/名称/型号/数量/单价/合计
- 偏离表: 需求项/响应/偏离说明
- 实施计划 (甘特图): 设计/采购/施工/调试/验收
- 验收标准: FAT/SAT项目+通过标准

## 规则

### R1: 技术方案结构
1. 项目概述
2. 系统架构(含拓扑图)
3. 设备选型(含参数对比)
4. 电气一次接线
5. 保护配置
6. EMS策略
7. 通信方案
8. 消防配置

### R2: 偏离表规则
- 每需求项必须响应
- 偏离项需说明原因和替代方案
- 优于需求项标注"正偏离"

### R3: 实施计划
- 设计: 2周
- 采购: 4-8周(视设备交期)
- 施工: 2-4周
- 调试: 1-2周
- 验收: 1周

## 验证Checklist
- [ ] 每需求项已响应
- [ ] 配置清单与方案一致
- [ ] 偏离项有替代方案
- [ ] 实施计划时间合理
