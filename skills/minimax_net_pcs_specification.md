# Skill: PCS 储能变流器技术规格书自动生成

## 1. 概述

### 1.1 定位

本 Skill 用于自动生成 PCS（Power Conversion System，储能变流器）采购招标技术规范书。输出结果可直接用于储能系统设备采购招标文件，具有完整的章节结构、可量化的技术指标、可执行的测试要求。

### 1.2 适用范围

适用于以下场景的 PCS 设备采购：
- 电网侧储能电站（调频/调峰/备用）
- 用户侧工商业储能系统
- 电源侧新能源配储系统
- 微电网/孤岛系统

### 1.3 核心输出

```
PCS技术规格书（Word/PDF格式）
├── 封面 + 目录
├── 第1章 总则
├── 第2章 使用条件
├── 第3章 技术规格
├── 第4章 并网性能
├── 第5章 通信接口
├── 第6章 保护功能
├── 第7章 型式试验要求
├── 第8章 验收与交付
├── 第9章 质量保证
└── 附录（检查表/数据点表）
```

### 第3章 技术规格

**3.1 基本参数**

```
PCS基本参数表:
  额定功率: ___kW（___MWh储能系统配套）
  额定交流电流: ___A（@___Vac）
  额定直流电流: ___A
  额定效率: ≥98%（保障值，参照GB/T 34120第6.1节，额定负载，cosφ=1）
  最大效率: ≥99%（典型值，60%负载率，cosφ=1）
  待机功耗: ≤额定功率的0.3%（即自耗电≤___kW）
  功率因数: 额定功率时PF≥0.99（滞后/超前）
  功率因数调节范围: 0.9超前~0.9滞后（可调）
  电流THD: ≤5%（额定负载，PF=1）
  电压THD: ≤5%（额定负载）

拓扑结构: [两电平/三电平/ANPC/模块化多电平]
  - 推荐：三电平拓扑（效率更高，谐波更低）
  - 模块化：支持N+1冗余，单模块故障不影响整机运行

工作模式:
  - 并网离网双模式（手动/自动切换）
  - 并网模式：恒功率/恒直流电压/恒交流电压
  - 离网模式：V/F控制，作为电压源运行
  - 黑启动：支持柴油机/电池双源黑启动（可选）
```

**3.2 效率指标详解**

PCS 效率是采购招标的核心指标，须明确区分保障值与典型值：

```
效率指标要求:

  1. 额定效率（保障值）
     定义：在额定功率、cosφ=1、额定直流电压、25±5℃环境条件下
     要求：≥98.0%（必须达到，不可低于）
     测试标准：GB/T 34133第5.2节
     测试方法：功率计法，直接测量输入/输出功率
     允差：保障值下浮1%即视为不合格

  2. 最大效率（典型值）
     定义：在60%额定功率、cosφ=1、额定直流电压、25±5℃条件下
     要求：≥99.0%
     说明：典型值不作为验收门槛，但需提供测试报告

  3. 效率-负载曲线
     卖方需提供20%~100%负载率的效率曲线图
     50%负载率效率：≥98.5%（典型值）
     25%负载率效率：≥97.5%（典型值）

  4. 德国BDEW标准效率（可选）
     如项目有德国市场或参照BDEW要求：
     - 25%负载率效率：≥96%
     - 50%负载率效率：≥97%
     - 75%负载率效率：≥97%
     - 100%负载率效率：≥96%

  5. 辅助功耗
     - 待机功耗：≤额定功率×0.3%（不含通信设备功耗）
     - 热备份功耗：≤额定功率×1.5%（模块上电但无功率输出）
     - 夜间自耗电：≤50W（储能系统夜间维护功耗）

  6. 效率测试条件
     - 环境温度：25±5℃
     - 直流电压：额定直流电压±5%
     - 功率因数：cosφ=1（滞后）
     - 测试时间：每工况稳定运行30min后测量
     - 测量仪表精度：0.5级及以上功率计
```

### 第4章 并网性能要求

**4.1 低电压穿越（LVRT）**

PCS 必须具备低电压穿越能力，当电网电压跌落时保持并网运行，并提供无功支撑。

```
LVRT要求（参照GB/T 34120-2017第6.3节）：

  穿越区域定义：
    电压跌落至 0% ~ 90% 额定电压，持续时间 0ms ~ 625ms
    具体穿越曲线如下：

    电压（% Un）
        |
    100%|- - - - - - - - - - - - - - - - - - - - - - - - - - - -
        |                      ___________
    90% |                     /           \
        |                    /             \
    20% |___________________/               \_________________
        |0    50   100  150  200  250  300  350  400  450  500  625
        |
        时间（ms）

  穿越要求：
    1. 电压跌落至 90% 时，无需穿越，持续并网
    2. 电压跌落至 20%~90% 时，持续并网 625ms，电压恢复后继续运行
    3. 电压跌落至 0% 时，持续并网 625ms，电压恢复后继续运行
    4. 电压恢复后，有功功率应以 ≥30% 额定功率/秒的速率恢复

  无功电流响应（故障期间）：
    - 电压跌落期间，PCS 应向电网输出无功电流
    - 无功电流 I react ≥ 2 × (1 - U/U rated) × I rated（0 ≤ U ≤ 0.85 Un）
    - 即：电压跌落越深，输出的无功电流越大（最高可达 2 倍额定电流）
    - 无功电流响应时间：≤30ms
    - 无功电流持续时间：电压恢复后继续维持 ≥150ms

  低电压穿越合格判定：
    - 期间无脱网
    - 故障清除后 3s 内恢复至正常并网状态
    - 有功功率恢复斜率满足要求
```

**4.2 高电压穿越（HVRT）**

```
HVRT要求（参照NB/T 31099-2016）：

  穿越参数：
    - 电压升高至 110% Un：持续运行
    - 电压升高至 120% Un：持续运行 1s
    - 电压升高至 130% Un：持续运行 0.5s

  高电压穿越期间要求：
    - 有功功率自动降至额定功率的 50%（可调）
    - 具备无功调节能力，维持电网电压稳定
    - 电压恢复正常后 3s 内恢复正常运行
```

**4.3 一次调频（可选功能）**

```
一次调频功能要求：

  有功-频率下垂特性：
    - 默认下垂系数：4%（即 0.04，即频率变化 1Hz，有功变化 4% 额定功率）
    - 可调范围：1%~10%（通过EMS设置）
    - 调频死区：±0.05Hz（可调）
    - 响应时间：≤2s（从频率变化到有功调节完成）
    - 精度：±0.5%（功率调节精度）

  调频模式：
    1. 定频率模式（droop 控制）
    2. 定功率模式（一次调频作为叠加功能）
    3. 虚拟同步机模式（VSG，可选）

  频率范围响应：
    - 50Hz ± 0.5Hz：正常运行，持续有功调节
    - 49.5Hz ~ 47.5Hz：降功率运行，最大可降额至 50%
    - 50.5Hz ~ 51.5Hz：降功率运行，最大可降额至 50%

  说明：一次调频功能需在合同中明确是否包含，不含此功能者可不参与本节检测
```

**4.4 AGC / AVC 调度响应**

```
功率调度功能要求：

  AGC（自动发电控制）：
    接收调度指令：≤4s 响应（从收到指令到功率变化到目标值±2%）
    功率变化率（ramp rate）：
      - 默认：10%/min（可调，范围 1%~100%/min）
      - 紧急模式：50%/min（持续时间≤60s）
    通讯协议：IEC 60870-5-104（优先）或 IEC 61850-3 或 Modbus TCP
    指令格式：
      - 有功设定值：P_set（0~100% 额定功率）
      - 有功变化率：Ramp（%/min）

  AVC（自动电压控制）：
    接收调度指令：≤1s 响应
    控制方式：功率因数设定或无功功率设定
    通讯协议：同 AGC

  指令优先级：
    1. 本地急停（最高优先级）
    2. 远程急停
    3. 调度有功指令
    4. EMS 功率指令
    5. 本地功率设定

  功率跟踪精度：
    - 稳态误差：≤±2% 额定功率
    - 动态误差：≤±5%（过渡过程）
```

**4.5 防孤岛与并网保护**

```
防孤岛保护要求：

  检测方法：
    1. 被动检测（电压/频率/相位异常检测）
    2. 主动检测（频率偏移、电流扰动）

  孤岛判定时间：
    - 正向功率运行：≤2s
    - 逆向功率运行（放电）：≤2s

  并网保护定值：
    | 保护功能        | 定值      | 动作时间  |
    |----------------|----------|----------|
    | 过电压（OU1）  | 1.10 Un  | 0.2s     |
    | 过电压（OU2）  | 1.15 Un  | 0.1s     |
    | 欠电压（U1）   | 0.90 Un  | 0.2s     |
    | 欠电压（U2）   | 0.20 Un  | 0.625s   |
    | 过频率         | 51.5Hz   | 0.2s     |
    | 欠频率         | 47.5Hz   | 0.2s     |
    | 频率变化率     | 1Hz/s    | 1.0s     |

  恢复合闸条件：
    - 电压恢复至 0.85Un~1.10Un 范围内
    - 频率恢复至 49.5Hz~50.5Hz 范围内
    - 持续时间满足 5s（可调）
    - 检测到主电网同步后，自动或手动并网
```

### 第5章 通信接口规范

**5.1 通信架构**

```
PCS通信架构：

  就地层：
    PCS 控制单元 ←→ HMI 触摸屏（RS485/CAN）
    PCS 控制单元 ←→ 电池BMS（CAN/RS485）
    PCS 控制单元 ←→ 本地EMS（以太网）

  监控层：
    PCS ←→ 就地EMS ←→ 远程控制中心（调度）
                    ←→ 远程运维平台（云端）

  通讯接口配置：
    - 通讯口数量：≥2 个以太网口（100/1000Mbps）
    - 接口类型：RJ45 或 LC 光口
    - 通讯协议：IEC 61850 / IEC 104 / Modbus TCP
    - 冗余配置：双网口热备（故障自动切换，切换时间 ≤1s）
```

**5.3 IEC 61850 配置规范**

```
IEC 61850 数据模型配置：

  逻辑设备（LD）划分：
    LLN0（Logical Node Zero）：公共逻辑设备
    PDC1（Power Conversion Device）：PCS核心控制
    PROT（Protection）：保护功能
    MMXU（Measurement）：电气量测
    SPCS（Static Power Conversion）：PCS状态和控制

  逻辑节点（LN）配置：
    PCC/PDC/SDPC/INS：并网点控制/功率控制/状态/启停
    MMXU1：交流三相电压电流测点
    MMXU2：直流电气测点
    SPCS1：PCS运行模式控制
    GAPC：功率因数调节
    SWGR：开关状态（并网断路器）
    RBRF：防孤岛保护

  GOOSE 配置：
    通讯参数：
      - APPID：0x0001~0x3FFF
      - VLAN ID：可配置（默认 0）
      - 数据集：StatusDS（状态），ControlDS（控制）
      - GOOSE 重传间隔：1s（正常），50ms（故障）

  SMV（采样值）：
    如需要 IEC 61850-9-2 采样值传输：
    - 采样率：4800 点/秒（80 点/周期×60Hz）
    - 通道：Ia，Ib，Ic，Ua，Ub，Uc
    - 品质位：valid/invalid/overflow 等

  MMS 配置：
    浏览器服务：支持 IEC 61850-8-1 Client
    文件服务：支持 ICD/CID/SCD 文件上载
    控制服务：Report/SGOOSE 直接控制

  典型 ICD 文件结构：
    Server
      ├─ LDevice: LD_PCC（并网控制）
      │    ├─ LN0 (LLN0)
      │    ├─ SPCS（PCS控制）
      │    ├─ MMXU（测点）
      │    └─ GAPC（功率因数）
      ├─ LDevice: LD_PROT（保护）
      │    ├─ RBRF（孤岛保护）
      │    └─ PTOV/PTUV（过欠压）
      └─ LDevice: LD_MEAS（测量）
           └─ MMXU（电气量）
```

**5.4 冗余通信配置**

```
通信冗余要求：

  双网口配置：
    - 网口1：主通讯口，正常运行时使用
    - 网口2：备用通讯口，故障时自动切换
    - 切换时间：≤1s（检测到网口1故障后自动切换）
    - 切换方式：热备（无需人工干预）

  双机热备（可选）：
    - 两台PCS控制主控器热备
    - 主控故障后，备控 0.5s 内接管
    - 数据同步：控制参数、故障记录实时同步
    - 切换过程：无功率中断（如果切换在 50ms 内完成）

  通讯故障告警：
    - 通讯中断 >5s：本地告警
    - 通讯中断 >60s：上报远程监控
    - 通讯恢复后：自动重连，数据同步
```

### 第6章 保护功能要求

**6.1 保护配置**

| 保护功能 | 定值 | 延时 | 动作 |
|----------|------|------|------|
| 直流过压保护 | 1.1×额定直流电压 | 0.5s | 停机 |
| 直流欠压保护 | 0.8×额定直流电压 | 1s | 停机（根据BMS指令）|
| 直流过流保护 | 1.5×额定直流电流 | 0.1s | 停机 |
| 交流过压保护（OU1）| 1.10Un | 0.2s | 停机 |
| 交流过压保护（OU2）| 1.15Un | 0.1s | 停机 |
| 交流欠压保护（U1）| 0.90Un | 0.2s | 降功率 |
| 交流欠压保护（U2）| 0.20Un | 0.625s | LVRT穿越 |
| 过频保护 | 51.5Hz | 0.2s | 停机 |
| 欠频保护 | 47.5Hz | 0.2s | 降功率 |
| 频率变化率保护 | dF/dt > 1Hz/s | 1s | 停机 |
| 交流过流保护 | 1.2×额定交流电流 | 1s | 降功率 |
| 交流短路保护 | 2×额定电流 | 0.05s | 速断 |
| 过温保护 | 结温>140℃ | 0s（立即）| 降功率或停机 |
| 防孤岛保护 | 主动+被动双重检测 | ≤2s | 停机 |
| 缺相保护 | 任意一相电压<50V | 0.5s | 停机 |
| 反放电保护 | 直流侧电流反向 | 0.1s | 停止逆变 |

**6.2 保护逻辑**

```
保护动作逻辑优先级：
  Level 1（最高优先级）：
    急停按钮触发 → 直接切断功率模块触发电路 → 立即关断 IGBT
    通讯急停指令 → 同上
  Level 2：
    直流母线过压 → 立即关断，闭合放电电阻（如果配置）
    短路故障 → 速断，跳闸并网断路器
  Level 3：
    过温降功率 → 降低有功至 50%（15min 内持续则停机）
    过载 → 按限时过载曲线运行（1.1×额定持续 60s/10min）
  Level 4：
    通讯故障 → 维持当前功率 5min，后降功率至待机
```

**7.2 各试验项目详细要求**

```
A1. 低电压穿越（LVRT）试验

  试验方法：
    1. 将PCS连接至电网模拟器（可编程三相交流源）
    2. 设置模拟器输出，注入不同深度的电压跌落：
       - 90% 跌落，持续 0ms（不发散，但持续并网）
       - 80% 跌落，持续 625ms
       - 50% 跌落，持续 625ms
       - 20% 跌落，持续 1200ms
       - 0% 跌落，持续 625ms（零电压穿越）
    3. 记录 PCS 响应波形（有功、无功、电压、电流）
    4. 验证：期间无脱网，无过流，无振荡

  合格标准：
    - 每种跌落深度均满足穿越要求
    - 电压恢复后 3s 内有功恢复至跌落前水平 ±10%
    - 无功电流响应满足公式：I react ≥ 2×(1-U/Un)×I rated

  参考标准：GB/T 34120-2017 第6.3节

A2. 高电压穿越（HVRT）试验

  试验方法：
    - 1.10Un：持续 10s
    - 1.20Un：持续 1s
    - 1.30Un：持续 0.5s
    记录 PCS 行为

  合格标准：电压恢复后正常运行，无损坏

A3. 孤岛保护试验

  试验方法：
    1. 并网运行，以额定功率向电网馈电
    2. 突然断开公共耦合点（PCC）断路器
    3. 记录从断开到 PCS 停止并网输出的时间
    4. 测试不同功率等级（100%、50%、25%额定功率）

  合格标准：
    - 孤岛检测时间 ≤2s
    - 期间无设备损坏
    - 电压频率在保护动作前不超过限值

A4. 并网/离网切换试验

  试验方法：
    1. 离网模式启动，建立电压频率
    2. 同步后合闸并网，观察切换过程
    3. 并网运行后主动断开，切换至离网模式
    4. 记录切换过程波形，验证无扰动

  合格标准：
    - 切换过程电压中断时间 <10ms
    - 切换过程频率波动 <±0.5Hz
    - 切换后 PCS 运行状态正常

A5. 功率因数调节范围试验

  试验方法：
    - 额定功率运行时，逐步调节 PF：0.9滞后→1.0→0.9超前
    - 记录无功功率输出和响应时间

  合格标准：
    - PF 调节范围满足 0.9滞后~0.9超前
    - PF 精度 ±0.01
    - 响应时间 ≤1s

B1. 电流谐波试验

  试验方法：
    - 额定功率运行，PF=1
    - 测量 3~49 次电流谐波
    - 对比 GB/T 14549-1993 表1限值的 80%

  合格标准：
    - THDi ≤ 5%
    - 各次谐波分量满足限值要求

B2. 电压谐波试验

  试验方法：同电流谐波，测量电压侧谐波

  合格标准：THDu ≤ 5%（取决于电网背景）

B3. 电压波动与闪变试验

  试验方法：
    - 额定功率运行
    - 在PCC点测量电压波动率和闪变值（Pst/Plt）

  合格标准：
    - 电压波动 d ≤ 3%（短时）/ 2.5%（长时）
    - Pst ≤ 1.0，PIt ≤ 0.8

C1. 额定效率测试（关键验收项）

  试验方法：
    1. 环境温度：25±5℃
    2. 直流电压：额定值 ±5%
    3. 功率因数：cosφ=1（滞后）
    4. 测试负载点：100% 额定功率
    5. 使用 0.5 级精度功率计直接测量输入/输出功率
    6. 每工况稳定运行 30min 后开始测量
    7. 记录 10 组数据（每组间隔 1min），取平均值

  合格标准：
    效率 ≥ 98.0%（保障值，不低于）

  计算公式：
    η = P_out(AC) / P_in(DC) × 100%

  注：如 PCS 含变压器，则效率测试包含变压器损耗，结果可能略低
```

**（继续 B3~G3 试验项目详细内容，与 A1~C1 格式保持一致，每项200~400字）**

```
B3. 电压波动与闪变试验

  试验方法：
    1. PCS 额定功率运行，PF=1
    2. 在公共连接点（PCC）安装闪变仪和三相电压记录仪
    3. 记录 10min 电压数据，计算短时闪变值 Pst
    4. 记录 2h 电压数据，计算长时闪变值 Plt
    5. 测试不同功率因数工况（0.9滞后/1.0/0.9超前）

  合格标准：
    - 电压波动 d ≤ 3%（短时条件），≤ 2.5%（长时条件）
    - 短时闪变值 Pst ≤ 1.0
    - 长时闪变值 Plt ≤ 0.8
    - 参考 GB/T 12326-2008

  试验设备：闪变仪（Class A 或更高），精度满足 IEC 61000-4-15

B4. 直流分量试验

  试验方法：
    1. 额定功率运行，cosφ=1
    2. 在交流输出侧测量直流分量（电流/电压）
    3. 使用真有效值万用表或示波器记录 DC 偏移

  合格标准：
    - 直流电流分量 ≤ 0.5% 额定交流电流
    - 直流电压分量 ≤ 0.5% 额定相电压

C2. 效率-负载曲线测试

  试验方法：
    1. 在额定直流电压、cosφ=1、25±5℃ 条件下
    2. 测试以下负载点的效率：20%，25%，30%，40%，50%，60%，75%，100%
    3. 每点稳定运行 15min 后测量
    4. 绘制效率-负载曲线图

  合格标准：
    - 各负载点效率均须提供（作为典型值）
    - 50% 负载效率 ≥ 98.5%
    - 25% 负载效率 ≥ 97.5%
    - 曲线应平滑，无异常跌落

C3. 待机功耗测试

  试验方法：
    1. PCS 进入待机状态（功率模块不工作，控制单元上电）
    2. 测量 30min 内 PCS 的有功功耗（从直流侧）
    3. 测量热备份模式功耗（功率模块上电但无功率输出）

  合格标准：
    - 待机功耗 ≤ 额定功率 × 0.3%（含控制单元功耗）
    - 热备份功耗 ≤ 额定功率 × 1.5%
    - 不含外接通讯设备功耗（如有）

C4. 功率因数-负载特性测试

  试验方法：
    1. 在 cosφ=0.9滞后、cosφ=1、cosφ=0.9超前 三种条件下
    2. 分别测试 25%、50%、75%、100% 负载的功率因数
    3. 验证 PF 精度和调节范围

  合格标准：
    - 功率因数精度 ±0.01
    - 任意负载点 PF 可调节至 0.9滞后~0.9超前

D1. 交流过/欠压保护试验

  试验方法：
    1. 逐步升高交流电压至 1.10Un、1.15Un，观察保护动作
    2. 逐步降低交流电压至 0.90Un、0.20Un，观察保护动作和LVRT行为
    3. 记录每个保护定值的实际动作值和动作时间

  合格标准：
    - 过压 OU1（1.10Un）动作时间 0.2s±0.05s
    - 过压 OU2（1.15Un）动作时间 0.1s±0.02s
    - 欠压 U1（0.90Un）动作时间 0.2s±0.05s
    - 欠压 U2（0.20Un）按 LVRT 要求处理（0.625s 内不脱网）

D2. 频率保护试验

  试验方法：
    1. 逐步升高电网频率至 51.5Hz，观察过频保护
    2. 逐步降低电网频率至 47.5Hz，观察欠频保护
    3. 测试频率变化率保护（dF/dt > 1Hz/s 时动作）

  合格标准：
    - 过频（51.5Hz）动作时间 0.2s±0.05s
    - 欠频（47.5Hz）动作时间 0.2s±0.05s
    - 频率变化率保护动作时间 1s±0.1s

D3. 过流与短路保护试验

  试验方法：
    1. 过流保护：逐步增加负载电流至 1.2 倍额定值，触发降功率
    2. 短路保护：在交流输出端模拟三相短路，验证速断能力
    3. 记录保护动作电流值和动作时间

  合格标准：
    - 1.2 倍额定电流过流保护：1s 内降功率
    - 2 倍额定电流短路：0.05s 内速断
    - 无损坏，不冒烟

D4. 直流过压/过流保护试验

  试验方法：
    1. 直流过压：逐步升高直流电压至 1.1 倍额定值，触发停机
    2. 直流欠压：逐步降低直流电压至 0.8 倍额定值，触发告警/停机
    3. 直流过流：注入 1.5 倍额定直流电流，触发速断

  合格标准：
    - 直流过压（1.1×Un_dc）：0.5s 内停机
    - 直流欠压（0.8×Un_dc）：1s 内告警/停机
    - 直流过流（1.5×In_dc）：0.1s 内停机

E1. 高温工作试验

  试验方法：
    1. 将 PCS 置于高温试验箱，温度设定 55℃±2℃
    2. 稳定 2h 后，以额定功率连续运行 4h
    3. 记录运行参数（效率、功率、温度）

  合格标准：
    - 期间额定功率正常运行
    - 各功率模块结温不超过器件允许值（IGBT≤150℃）
    - 无性能下降，无告警

E2. 低温工作试验

  试验方法：
    1. 将 PCS 置于低温试验箱，温度设定 -25℃±2℃
    2. 稳定 2h 后，冷启动并运行额定功率 4h

  合格标准：
    - 冷启动成功（低温启动电流不超过额定值 2 倍）
    - 低温下额定功率正常运行
    - 无凝露导致短路或告警

E3. 湿热试验

  试验方法：
    1. 温度 40℃±2℃，相对湿度 93%±3%，持续 48h
    2. 试验期间 PCS 不通电
    3. 试验后取出，恢复至常温 2h
    4. 进行绝缘电阻和耐压测试

  合格标准：
    - 绝缘电阻 ≥20MΩ（1000Vdc 兆欧表）
    - 耐压试验通过（无击穿）
    - 外观无锈蚀、变形

E4. 振动试验

  试验方法：
    1. 按 GB/T 2423.10-2019 进行
    2. 频率范围：2Hz~200Hz（扫频）
    3. 加速度：2g（包装状态）
    4. 振动方向：垂直（Z 轴）
    5. 振动时间：每个轴向 20min

  合格标准：
    - 试验后外观检查无松动
    - 功能测试正常（可正常启动运行）
    - 无结构件断裂或变形

E5. 防护等级验证

  试验方法：
    1. IP20：验证手指不能插入（防止触碰带电部分）
    2. IP54：防尘试验（8h）+ 防水试验（淋雨 10min）

  合格标准：
    - IP20：验证孔径 ≤12.5mm 的探针不能进入
    - IP54：试验后内部无灰尘进入，无水渗入

F1. 传导骚扰（电源端口）

  试验方法：
    1. 参照 CISPR 11 / GB 4824
    2. PCS 额定功率运行
    3. 使用人工电源网络（LISN）测量电源端口骚扰电压
    4. 频率范围：150kHz~30MHz

  合格标准：满足 GB 4824 Class A 或 B（根据安装环境）

F2. 辐射骚扰（机箱端口）

  试验方法：
    1. 参照 CISPR 11 / GB 4824
    2. 使用天线和接收机在 30MHz~1GHz 频段测量辐射骚扰

  合格标准：满足 Class A 或 B 限值

F3. 静电放电抗扰度（ESD）

  试验方法：
    1. 参照 IEC 61000-4-2
    2. 接触放电 ±8kV，空气放电 ±15kV
    3. 放电点：操作界面、通讯端口、外壳接缝

  合格标准：无死机、无数据丢失、自动恢复

F4. 电快速瞬变脉冲群抗扰度（EFT）

  试验方法：
    1. 参照 IEC 61000-4-4
    2. 电源端口：±2kV（峰值），5kHz/100kHz
    3. 信号端口：±1kV（峰值）

  合格标准：无告警、无数据错误、自动恢复

F5. 浪涌抗扰度（Surge）

  试验方法：
    1. 参照 IEC 61000-4-5
    2. 交流电源端口：线-地 ±2kV，线-线 ±1kV
    3. 通讯端口：±1kV

  合格标准：无损坏、自动恢复

G1. AGC 响应时间测试

  试验方法：
    1. PCS 在额定功率稳定运行
    2. 通过通讯接口发送有功功率设定指令（从 100% 降至 50%）
    3. 记录从发送指令到功率变化至目标值±2% 的时间

  合格标准：响应时间 ≤4s

G2. 通讯协议一致性测试

  试验方法：
    1. Modbus TCP：按寄存器映射表逐项读取/写入验证
    2. IEC 61850：加载 ICD 文件，验证数据模型和服务
    3. IEC 104：验证报文格式、命令类型、时间同步

  合格标准：协议实现完全一致，无丢帧、无错误

G3. 黑启动功能测试（可选）

  试验方法：
    1. 直流侧接电池 BMS，交流侧断开
    2. 发送启动指令，验证 PCS 建立交流电压
    3. 在无电网参考情况下启动离网运行

  合格标准：
    - 黑启动时间 ≤30s（从指令发出到建立电压）
    - 输出电压 THDu ≤10%（纯电阻负载）
    - 频率精度 ±0.5Hz
```
