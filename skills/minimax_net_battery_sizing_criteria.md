## 一、电芯选型矩阵

### 1.1 化学体系横向对比（LFP vs NMC vs NaS）

| 对比维度 | LFP（磷酸铁锂） | NMC（镍钴锰酸锂） | NaS（钠硫） |
|---------|----------------|------------------|------------|
| **能量密度（Wh/kg）** | 140~180 | 200~260 | 150~200 |
| **体积能量密度（Wh/L）** | 300~400 | 450~650 | 400~500 |
| **循环寿命（次）** | 4000~8000 @80%SOH | 2000~4000 @80%SOH | 2500~4500 |
| **日历寿命（年）** | 15~20 | 8~12 | 10~15 |
| **低温性能（°C）** | -20°C可用，-30°C需加热 | -20°C可用 | >300°C工作温度 |
| **高温稳定性** | 优秀（热失控点>270°C） | 一般（热失控点约210°C） | 需高温维持能耗 |
| **安全性** | 高（不起火不爆炸） | 中（热失控风险较高） | 低（高温熔融钠） |
| **回收价值** | 铁磷价值低，电池回收利润薄 | 钴镍贵金属，回收价值高 | 钠硫难以回收 |
| **成本（元/kWh）** | 500~700 | 700~1000 | 800~1200 |
| **自放电率/月** | 1~2% | 2~3% | 5~8%（保温能耗） |
| **适用场景** | 大储/工商业/安全性优先 | 乘用车/高能量密度场景 | 高温工业/特殊应用 |

**推荐结论**：储能场景（尤其电网级大储和工商业）优先选择LFP体系。电网侧项目因容量大、安全要求高、循环寿命要求高，LFP是唯一经济合理选择。

## 二、模组/簇拓扑设计

### 2.1 串联数S计算（电压窗口匹配PCS DC范围）

#### 计算公式

```
S = ceil(V_pack_max / V_cell_max)
```

其中：
- `V_pack_max` = PCS直流侧最高工作电压（常见1200V或1500V）
- `V_cell_max` = 电芯最高充电电压（磷酸铁锂典型值3.65V）
- `ceil()` = 向上取整

#### 典型场景计算

**场景1：1500V系统 + 314Ah电芯**

| 参数 | 数值 | 说明 |
|------|------|------|
| PCS直流电压范围 | 900~1500V | 1500V系统典型范围 |
| 电芯标称电压 | 3.2V | LFP 314Ah |
| 电芯最高充电电压 | 3.65V | LFP截止电压上限 |
| 电芯最低放电电压 | 2.5V | LFP截止电压下限 |
| 计算串联数 | `ceil(1500 / 3.65) = 411S` | 向上取整 |
| 实际串联数 | **411S** | 串数取整后确定 |
| PACK最高电压 | 411 × 3.65 = 1500V | 刚好匹配上限 |
| PACK标称电压 | 411 × 3.2 = 1315.2V | 系统标称电压 |
| PACK最低电压 | 411 × 2.5 = 1027.5V | 放电终止电压 |

**场景2：1000V系统 + 280Ah电芯（风冷215S方案）**

| 参数 | 数值 | 说明 |
|------|------|------|
| PCS直流电压范围 | 600~1000V | 1000V系统典型范围 |
| 电芯标称电压 | 3.2V | LFP 280Ah |
| 电芯最高充电电压 | 3.65V | LFP截止电压上限 |
| 计算串联数 | `ceil(1000 / 3.65) = 274S` | 向上取整 |
| 实际串联数 | **215S或274S** | 根据系统设计选择 |
| 215S方案说明 | 215×3.65=784.75V < 1000V | 低电压方案，适配功率受限场景 |
| 274S方案说明 | 274×3.65=1000V | 满电压方案，适配2h系统 |

**场景3：1500V系统 + 280Ah电芯（384S方案）**

| 参数 | 数值 | 说明 |
|------|------|------|
| PCS直流电压范围 | 900~1500V | 1500V系统 |
| 电芯最高充电电压 | 3.65V | LFP |
| 计算串联数 | `ceil(1500 / 3.65) = 411S` | 理论值 |
| 实际串联数 | **384S** | 工程折中方案（考虑压差余量） |
| 384S PACK最高电压 | 384 × 3.65 = 1401.6V | 低于1500V，留有10%余量 |
| 384S PACK标称电压 | 384 × 3.2 = 1228.8V | 系统标称电压 |
| 384S PACK最低电压 | 384 × 2.5 = 960V | 放电终止电压 |
| 取384S理由 | 384为16的倍数，便于模组排布 | 24个16S模组=384S |

#### 串联数快速对照表

| 电芯容量 | 1000V系统（风冷） | 1500V系统（液冷） |
|---------|------------------|------------------|
| 280Ah | 215S或274S | 384S（384S=24×16S模组） |
| 314Ah | 240S或274S | 411S（411S=27×15S模组） |
| 230Ah（功率型） | 288S | 432S（432S=24×18S模组） |

### 2.2 并联数P计算（容量+电流匹配）

#### 计算公式

```
P = ceil(I_max / I_cell_max)
单簇容量(kWh) = S × P × C_cell(Ah) × V_cell_nom(V) / 1000
单簇最大放电电流 = P × I_cell_max
```

#### 参数说明

- `S` = 串联数（来自2.1节）
- `P` = 并联数（簇内并联的电芯组数）
- `C_cell` = 电芯额定容量（Ah）
- `V_cell_nom` = 电芯标称电压（V）
- `I_max` = PCS交流侧最大功率对应的直流电流
- `I_cell_max` = 单电芯最大持续放电电流（A）

#### 计算示例：1500V / 4.4MWh液冷舱

| 参数 | 数值 | 计算过程 |
|------|------|---------|
| 系统功率目标 | 2.5MW PCS | 典型大储单舱功率 |
| 系统容量目标 | 4.4MWh | 20尺舱典型容量 |
| 放电时长 | 4.4MWh / 2.5MW = 1.76h ≈ 2h | 满足2h储能要求 |
| PCS直流电压范围 | 900~1500V | 1500V系统 |
| 选用电芯 | 314Ah LFP | 宁德时代314Ah |
| 电芯标称电压 | 3.2V | |
| 电芯最大持续放电电流 | 1C = 314A | 314Ah @ 1C放电 |
| 电芯最大持续充电电流 | 0.5C = 157A | 314Ah @ 0.5C充电 |
| 串联数S | 411S | 来自2.1节计算 |
| 计算并联数P | `P = ceil(2500000W / (411 × 3.2V × 314A))` | |
| | 逐步计算 | 直流侧最大电流 = 2500000 / (411×3.2) = 1901A |
| | 继续 | 单电芯最大电流 = 314A |
| | 结果 | P = ceil(1901 / 314) = ceil(6.05) = **6P** |
| 单簇实际容量 | 411 × 6 × 314 × 3.2 / 1000 = **2628.9 kWh** | 约2.63MWh/簇 |
| 单舱簇数量 | 4400kWh / 2628.9kWh = 1.67 → **2簇** | 每舱2簇 |
| 单舱总容量 | 2 × 2628.9 = 5257.8 kWh | 实际比目标略大 |
| 调整方案 | 可选1：降容至2簇×2.2MWh | 或选2：增加串联数至432S |

#### 计算示例：1000V / 2.76MWh风冷舱（工商业）

| 参数 | 数值 | 计算过程 |
|------|------|---------|
| 系统功率目标 | 1MW PCS | 工商业典型功率 |
| 系统容量目标 | 2.76MWh | 215S风冷方案 |
| 放电时长 | 2.76h | 满足削峰填谷 |
| PCS直流电压范围 | 600~1000V | 1000V系统 |
| 选用电芯 | 280Ah LFP | 亿纬280Ah |
| 电芯标称电压 | 3.2V | |
| 电芯最大持续放电电流 | 1C = 280A | 280Ah @ 1C |
| 串联数S | 215S | 1000V系统典型 |
| 直流侧最大电流 | 1000000W / (215 × 3.2V) = 1453A | 全功率运行 |
| 并联数P | `ceil(1453 / 280) = ceil(5.19) = 5P` | 5并联 |
| 单簇容量 | 215 × 5 × 280 × 3.2 / 1000 = **963.2 kWh** | 约1MWh/簇 |
| 单舱簇数量 | 2760 / 963.2 = 2.87 → **3簇** | 每舱3簇 |
| 单舱总容量 | 3 × 963.2 = 2889.6 kWh | 略超设计值 |
| 实际配置 | 可用2簇+1簇差异配置 | 灵活配置 |

### 2.3 簇间环流抑制设计

#### 问题描述

多簇电池并联运行时，由于各簇SOC、内阻差异，会产生簇间环流：
- 簇A SOC高、簇B SOC低 → 簇A向簇B充电
- 环流导致容量损失、发热、寿命衰减

#### 抑制方案对比

| 方案 | 原理 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|---------|
| **被动均衡（均压电阻）** | 并联电阻消耗过高中压簇电量 | 简单可靠，成本低 | 能量浪费，无法主动补偿 | 小规模≤1MW |
| **主动均衡（DC/DC模块）** | DC/DC双向变换器转移电量 | 效率高，可跨簇补偿 | 成本高，需要BMS协调 | 大规模≥1MW |
| **PCS协调控制** | PCS控制各簇充放电功率 | 无需额外硬件 | 依赖PCS功能 | 配合主动均衡使用 |

#### 推荐配置

**小规模系统（≤1MW，单舱）**
- 配置被动均衡电阻（每簇50~100Ω放电电阻）
- BMS均衡策略：SOC差值>5%时触发均压

**大规模系统（≥1MW，多舱）**
- 模组级主动均衡（每模组独立DC/DC模块）
- BMS三级均衡：模组间均衡（SMU）+ 簇间均衡（CMU）+ 系统级协调
- 均衡电流规格：模组内±2A，簇间±5A

#### 均衡参数计算

```
ΔSOC = SOC_A - SOC_B
均衡电流 I_eq = ΔSOC × C_pack / t_eq
t_eq = 均衡时间目标（通常30min~2h）
```

示例：SOC差值8%，簇容量1000Ah，均衡时间1h
```
I_eq = 8% × 1000Ah / 1h = 80mA（被动均衡）
I_eq = 8% × 1000Ah / 30min = 267mA（主动均衡）
```

### 2.4 电池簇电气参数完整计算示例

#### 项目信息
- 系统规模：20尺标准集装箱
- 目标容量：4.4MWh
- PCS功率：2.5MW
- 电压平台：1500V
- 电芯选型：宁德时代 LFP 314Ah

#### 完整计算

**Step 1：电芯基本信息**

| 参数 | 数值 |
|------|------|
| 电芯标称电压 | 3.2V |
| 电芯标称容量 | 314Ah |
| 电芯能量 | 314 × 3.2 = 1004.8Wh ≈ 1kWh |
| 电芯最大充电电压 | 3.65V |
| 电芯最小放电电压 | 2.5V |
| 电芯最大持续充电电流 | 0.5C = 157A |
| 电芯最大持续放电电流 | 1C = 314A |
| 电芯重量 | 5.6kg |
| 电芯能量密度 | 1004.8 / 5.6 = 179.4 Wh/kg |

**Step 2：串联数计算**

```
S = ceil(1500V / 3.65V) = ceil(410.96) = 411S
```

| 参数 | 数值 |
|------|------|
| 串联数S | 411S |
| 簇最高电压 | 411 × 3.65 = 1500.15V ≈ 1500V ✓ |
| 簇标称电压 | 411 × 3.2 = 1315.2V |
| 簇最低电压 | 411 × 2.5 = 1027.5V |

**Step 3：并联数计算**

```
I_DC_max = P_PCS / (S × V_cell_min) = 2500000 / (411 × 2.5) = 2432A
P = ceil(I_DC_max / I_cell_max) = ceil(2432 / 314) = ceil(7.75) = 8P
```

| 参数 | 数值 |
|------|------|
| 并联数P | 8P |
| 单簇标称容量 | 411 × 8 × 314 × 3.2 / 1000 = **3333.5 kWh** |
| 单簇最高电压电流 | 1500V × 314A = 471kW（单并） |
| 单簇最大放电功率 | 411 × 2.5V × 8 × 314A = 2580kW |
| 单簇最大放电电流 | 8 × 314 = 2512A |
| 单簇最大充电电流 | 8 × 157 = 1256A |

**Step 4：舱级配置**

| 参数 | 数值 |
|------|------|
| 单舱目标容量 | 4400kWh |
| 单簇容量 | 3333.5kWh |
| 舱内簇数 | ceil(4400 / 3333.5) = ceil(1.32) = **2簇** |
| 实际舱容量 | 2 × 3333.5 = 6667kWh ≈ 6.67MWh |
| 超配分析 | 6.67MWh / 2.5MW = 2.67h（超过2h设计） |
| 调整方案 | 选用1簇（3333.5kWh）+ 调整PCS功率曲线 |

**Step 5：电芯数量统计**

| 参数 | 数值 |
|------|------|
| 单簇电芯数 | 411 × 8 = 3288颗 |
| 单舱电芯数 | 2 × 3288 = 6576颗 |
| 每40尺集装箱 | 约8000~12000颗电芯 |
| 系统电芯总数（100MWh项目） | 1000000kWh / 3333.5kWh ≈ 300簇 |

**Step 6：重量与空间计算**

| 参数 | 数值 |
|------|------|
| 单电芯重量 | 5.6kg |
| 单模组电芯数 | 16S×1P = 16颗 |
| 单模组重量 | 16 × 5.6 = 89.6kg |
| 模组数量/簇 | 3288 / 16 = 205.5 → **206个模组** |
| 簇总重量 | 206 × 89.6 = 18.5吨（含外壳+结构） |
| 舱总重量 | 2 × 18.5 + 4吨（舱体） = 41吨 |
| 舱内空间利用率 | 需20尺舱（6.1m×2.4m×2.8m=41m³）布置2簇电池 |

### 3.1 风冷 vs 液冷设计边界

#### 选型决策树

```
项目规模?
```

#### 液冷适用条件

| 条件 | 标准值 | 说明 |

| 系统功率 | ≥2MW | 大储项目必选 |
| 环境温度 | 35~50°C | 极端气候项目 |
| 海拔 | >2000m | 高原项目强制液冷 |
| 电池放电倍率 | ≥0.5C | 高倍率放电需精确温控 |
| 系统效率要求 | >90% | 液冷能维持更优效率 |
| 舱内温差要求 | ≤5°C | 高端项目要求 |

### 3.2 液冷管路设计

#### 冷却液选型

| 冷却液类型 | 配方 | 冰点 | 沸点 | 适用场景 | 价格 |
|-----------|------|------|------|---------|------|
| 乙二醇+水 | 3:7（质量比） | -30°C | 108°C | 温和气候 | 低 |
| 乙二醇+水 | 4:6（质量比） | -40°C | 106°C | 寒冷气候 | 低 |
| 丙二醇+水 | 3:7（质量比） | -25°C | 106°C | 食品级要求 | 中 |
| 专业冷却液 | 丙二醇+防腐剂 | -45°C | 110°C | 高端项目 | 高 |

**推荐**：国内储能项目优先选用40%乙二醇+60%去离子水，防冻且经济。

#### 流量计算公式

```
Q_heat = P_loss × t_discharge
Q_cool = m × Cp × ΔT
m = Q_cool / (Cp × ΔT)
```

参数说明：
- `Q_heat` = 电池发热量（W）
- `P_loss` = 电池内阻损耗功率 = I² × R_internal
- `m` = 冷却液质量流量（kg/s）
- `Cp` = 冷却液比热容（J/(kg·°C)）
- `ΔT` = 进出口温差（°C），典型值5~8°C

#### 流量计算示例

**项目参数：2.5MW系统，2h放电**

| 参数 | 数值 | 计算过程 |
|------|------|---------|
| 电池内阻损耗 | ~1.5%（电池效率98.5%） | 典型LFP内阻损耗 |
| 发热功率 | 2500kW × 1.5% = 37.5kW | 持续发热量 |
| 冷却液比热容 | 3.5 kJ/(kg·°C) | 40%乙二醇溶液 |
| 进水温度 | 10°C | 环境温度30°C时 |
| 出水温度上限 | 18°C | ΔT=8°C设计值 |
| 所需质量流量 | m = 37.5kW / (3.5kJ·°C × 8°C) = 1.34 kg/s | |
| 体积流量 | 1.34 kg/s / 1.05 kg/L = **1.28 L/s ≈ 77 L/min** | |
| 校核 | 1.28 L/s × 8°C × 3.5 kJ/(kg·°C) = 35.8 kW ≈ 37.5kW ✓ | |

#### 管路压降计算

```
ΔP = f × (L/D) × (ρv²/2) + ΣK × (ρv²/2)
```

参数说明：
- `f` = 沿程阻力系数（Re数决定）
- `L` = 管路总长度（m）
- `D` = 管路内径（m）
- `ρ` = 冷却液密度（kg/m³）
- `v` = 流速（m/s）
- `K` = 局部阻力系数（弯头、阀门等）

**压降计算示例（2.5MW液冷系统）**

| 参数 | 数值 | 说明 |
|------|------|---------|
| 管路总长度 | 50m | 20尺舱典型管路长度 |
| 管路内径 | 25mm | DN25不锈钢管 |
| 局部元件 | 10个弯头 + 2个阀门 | 弯头K=0.3，阀门K=1.5 |
| 流速 | 1.28 L/s / (π×0.0125²) = 2606 m/h = 0.72 m/s | 合理流速<2m/s |
| 雷诺数 | Re = ρvD/μ = 1050×0.72×0.025/0.003 = 6300 | 湍流状态 |
| 沿程阻力系数 | f ≈ 0.04（湍流经验值） | |
| 沿程压降 | ΔP_f = 0.04 × (50/0.025) × (1050×0.72²/2) = 1080 Pa | |
| 局部压降 | ΔP_K = (0.3×10 + 1.5×2) × (1050×0.72²/2) = 2100 Pa | |
| 总压降 | 3180 Pa ≈ **3.2 kPa** | |
| 泵选型 | 扬程≥5m（留有余量） | 需克服管路+换热器压降 |

#### 温控算法

**模式1：定流量温控**
```
T_out_set = T_ambient + ΔT_targe
当T_battery_max > T_out_set + 2°C时 → 增加流量
当T_battery_max < T_out_set - 2°C时 → 减少流量
```

**模式2：变流量温控（推荐）**
```
PWM阀开度 = f(T_battery_avg, dT/dtime)
流量范围：30%~100%额定流量
控制目标：T_battery_avg = 25~30°C（最优工作区间）
```

**温控参数目标**

| 场景 | 进水温度 | 出水温度 | 舱内最高温度 | 温差（ΔT） |
|------|---------|---------|-------------|------------|
| 标准工况 | 10~15°C | 15~22°C | ≤35°C | 5~8°C |
| 高温工况 | 15~20°C | 22~28°C | ≤40°C | 6~10°C |
| 低温启动 | 5~10°C（加热） | 10~18°C | ≤30°C | 3~5°C |

### 3.3 热失控防护设计

#### 热失控触发条件与传播路径

LFP电芯热失控触发温度：200~270°C  
热失控阶段： 
1. SEI膜分解（80~120°C） → 
2. 负极与电解液反应（120~200°C） → 
3. 隔膜收缩短路（200°C+） → 
4. 正极分解产气（250°C+） → 
5. 电解液燃烧

#### 三级防护体系

**第一级：预防（阻止热失控发生）**

| 措施 | 规格 | 说明 |
|------|------|------|
| 电芯级 | 通过UL9540A测试 | 认证电芯不发生级联热失控 |
| BMS过温保护 | >50°C时降功率，>55°C时停机 | 主动保护 |
| 温度采样密度 | 每个模组≥2个温度点 | 快速响应 |
| 充电温控 | <0°C时限制充电电流≤0.1C | 低温防析锂 |

**第二级：控制（限制热失控扩散）**

| 措施 | 规格 | 说明 |
|------|------|------|
| 模组间防火隔板 | 耐火极限≥1h，厚度≥50mm | 阻止模组间传播 |
| 模组级气溶胶灭火器 | 触发温度70~80°C | 模组内灭火 |
| 簇级水喷淋 | 水量≥5L/min·m²，覆盖全部电池 | 簇级冷却 |
| 舱内泄压口 | 面积≥0.05m²，位于舱顶 | 防止舱体爆炸 |
| 排烟风机 | 火灾时自动启动，排烟量≥3000m³/h | 排烟 |

**第三级：灭火（处置已发生热失控）**

| 措施 | 规格 | 说明 |
|------|------|------|
| 舱内水喷淋系统 | 全淹没式，水量≥10L/min·m² | 持续降温 |
| 消防联动 | 烟感/温感→BMS→PCS→消防主机 | 全系统联动 |
| 水源保障 | 消防水池容量≥200m³（大型项目） | 持续供水 |
| 冷却时长 | 热失控后持续喷淋≥30min | 彻底降温 |

#### 消防联动逻辑

```
热失控报警条件（满足任一）：
├── 任一温度点 > 60°C（持续10s）
├── 任一模组温差 > 15°C
└── 烟感报警

联动动作（立即执行）：
1. BMS发出停机命令至PCS（断开DC接触器）
2. 启动舱内声光报警
3. 启动水喷淋（延迟30s，判断是否为误报）
4. 通知EMS监控平台
5. 启动排烟风机
6. 持续喷淋至温度<40°C
7. 通知消防部门（大型项目）
```

### 4.2 CAN菊花链 vs 星型网络选型

#### 拓扑对比

| 拓扑 | 结构 | 布线量 | 通信速率 | 故障隔离 | 成本 |
|------|------|--------|---------|---------|------|
| CAN菊花链 | 串行连接，节点首尾相连 | 少（仅2根CAN线） | 最高500kbps | 差（单点故障影响整链） | 低 |
| CAN星型 | 所有节点独立连接主控 | 多（N根CAN线） | 最高1Mbps | 好（单点故障不影响其他） | 中 |
| 高速CAN星型 | 每CMU独立CAN通道 | 最多（每个CMU独立） | 最高2Mbps | 优 | 高 |
| 以太网 | TCP/IP架构，交换机汇聚 | 中 | 100Mbps+ | 优 | 高 |

#### 菊花链适用场景

- 模组数量<20个（单CMU管理）
- 布线空间受限（电缆沟窄）
- 成本敏感项目
- 故障容忍度高（可接受单链故障）

#### 星型网络适用场景

- 模组数量>20个
- 需高可靠性（通信冗余）
- 需高速数据传输（>500kbps）
- 大储项目（>1MW）

#### 推荐配置

**工商业储能（1000V系统，≤1MW）**
```
CAN菊花链：BMU ↔ CMU
通信速率：250kbps
线缆：双绞屏蔽线，截面积≥0.75mm²
```

**电网侧大储（1500V系统，≥2MW）**
```
高速CAN星型：BMU ↔ 每个CMU独立CAN通道
通信速率：500kbps~1Mbps
线缆：双绞屏蔽线，截面积≥1.0mm²
每CMU独立告警指示灯（快速定位故障）
```

### 4.3 SOC/SOH算法选型

#### 主流算法对比

| 算法 | 原理 | 精度 | 计算量 | 适用场景 | 优缺点 |
|------|------|------|--------|---------|--------|
| **安时积分法** | ∫I·dt累积电量变化 | 1~3%（短期） | 低 | 动态工况实时计算 | 优点：简单可靠 缺点：累计误差，需定期校准 |
| **开路电压法（OCV）** | OCV与SOC查表关系 | 3~5% | 低 | 静置后校准 | 优点：无需模型 缺点：需长时静置才准确 |
| **EKF扩展卡尔曼滤波** | 状态空间模型+递推估计 | 0.5~2% | 中 | 动态工况高精度 | 优点：精度高，自适应 缺点：需精确模型参数 |
| **UKF无迹卡尔曼滤波** | 改进EKF，避免线性化 | 0.3~1% | 高 | 高精度需求 | 优点：精度最高 缺点：计算量大 |
| **神经网络法** | ANN拟合SOC-SOC关系 | 1~3% | 高 | 大数据训练场景 | 优点：可自学 缺点：需大量训练数据 |

#### 推荐选型策略

**场景1：工商业储能（成本优先）**
```
算法组合：安时积分 + OCV校准
SOC精度：约2%
实现方式：
├── 主算法：安时积分（实时计算）
├── 校准触发：静置>30min后OCV校准
└── 校准频率：每24h至少1次完整OCV校准
```

**场景2：大储项目（精度优先）**
```
算法组合：EKF + 安时积分 + OCV校准
SOC精度：约1%
实现方式：
├── 动态工况：EKF估算（状态方程基于电化学模型）
├── 稳态校准：OCV校准（每8h触发）
└── 备用机制：安时积分兜底（EKF故障时）
```

**场景3：调频场景（快速响应）**
```
算法组合：改进安时积分 + 在线参数辨识
SOC精度：约2%（响应速度优先）
实现方式：
├── 主算法：改进安时积分（带自学习修正）
├── 参数在线辨识：实时更新内阻模型
└── 辅助：5min周期OCV采样（不影响调度）
```

#### SOH估算算法

SOH（健康状态）定义：
```
SOH = C_actual / C_initial × 100%
```

| 算法 | 原理 | 精度 | 适用场景 |
|------|------|------|---------|
| **容量法** | 直接测量充放电容量 | 2~5% | 实验室/慢充场景 |
| **内阻法** | 直流内阻与SOH映射 | 5~10% | 在线估算（精度低） |
| **电化学阻抗谱（EIS）** | 交流阻抗谱分析 | 1~3% | 高精度需求（成本高） |
| **模型法** | 等效电路模型参数衰减 | 2~5% | 工程应用 |
| **机器学习法** | 大数据训练SOH预测模型 | 1~3% | 有历史数据的项目 |

**推荐**：大储项目采用"容量法+内阻法"组合：
- 定期慢充放电校准容量（每季度）
- 日常内阻监测趋势（无损失）

#### BMS关键参数整定

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| SOC显示精度 | ≤2% | 用户端显示要求 |
| SOC均衡触发阈值 | ΔSOC > 5% | 触发均衡逻辑 |

| 消防信号线 | 2.5mm²耐火线 | 烟感/温感→消防主机 | 独立线槽 |

#### 动力线与通信线分离原则

```
强制要求：动力线与通信线必须物理隔离

理由：
├── 动力线载流量大，电流波动时产生电磁干扰
├── 通信线信号弱（mV级），易受干扰
└── 干扰会导致BMS通信错误，触发保护停机

隔离方式（按优先顺序）：
1. 不同线槽：动力线在金属槽内，通信线在塑料槽内
2. 间距隔离：同路径时，保持≥100mm间距
3. 交叉垂直：必须交叉时，交叉角度≥90°
```

#### 线槽布置规范

```
舱内线槽布置（俯视图）：

        ┌─────────────────────────────┐
        │         电池架区域          │
        │  簇1    簇2    簇3    簇4   │
        └─────────┬───────────────────┘
                  │
    ┌─────────────┼─────────────────────────┐
    │             │                         │
    │  通信线槽   │    动力线槽           │
    │  (左侧)     │    (中间)              │
    │             │                         │
    └─────────────┼─────────────────────────┘
                  │
        ┌─────────┴───────────────────┐
        │         舱外接口区            │
        │   DC+/DC- 通信 接地 消防      │
        └─────────────────────────────┘
```

**关键设计点**：
- 通信线槽与动力线槽间距≥150mm
- 通信线槽位于舱左侧（远离动力线）
- 动力线槽中间贯通（最短路径）
- 舱内接口集中于舱体一端（便于外部接入）

## 附录B：常用公式速查

### B.1 电芯参数计算

```
电芯能量(Wh) = 额定容量(Ah) × 标称电压(V)
电芯功率(kW) = 额定容量(Ah) × 最大放电倍率(C) × 标称电压(V) / 1000
```

### B.2 簇参数计算

```
串联数S = ceil(PCS最高直流电压 / 电芯最高充电电压)
并联数P = ceil(PCS最大直流电流 / 电芯最大持续放电电流)
单簇容量(kWh) = S × P × 电芯额定容量(Ah) × 电芯标称电压(V) / 1000
单簇最大放电功率(kW) = S × 电芯最低电压(V) × P × 电芯最大放电电流(A) / 1000
```

### B.3 热管理参数计算

```
电池发热功率(W) = 放电功率(kW) × (1/效率 - 1)
冷却液质量流量(kg/s) = 发热功率(W) / [比热容(J/kg·°C) × 温差(°C)]
冷却液体积流量(L/s) = 质量流量(kg/s) / 密度(kg/L)
```

### B.4 BMS参数计算

```
SOC变化量(%) = ∫电流(A)·dt / 额定容量(Ah) × 100
ΔSOC均衡 = SOC_max - SOC_min
均衡电流(A) = ΔSOC(%) × 额定容量(Ah) / 均衡时间(h)
SOH(%) = 当前最大容量(Ah) / 初始额定容量(Ah) × 100
```
