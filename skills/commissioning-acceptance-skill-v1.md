# 调试验收 Skill v1.0

> 设计人：卢继雄 | 基于5年微网项目实战训练

## 概述
自动生成FAT/SAT测试项、遥测遥信核对表、告警联动测试表、充放电测试记录模板。

## 输入
- 设备清单 + 点表
- 策略配置
- 保护配置

## 输出
- FAT测试项Excel
- SAT测试项Excel
- 遥测遥信核对表
- 告警联动测试表
- 充放电测试记录模板

## 规则

### R1: FAT测试项
- 遥测精度测试: 每测点对比标准表, 误差≤0.5%
- 遥信动作测试: 每信号实际触发验证
- 遥控执行测试: 充放模式切换/功率设定
- 保护动作测试: 过流/过压/欠压/频偏/防孤岛

### R2: SAT测试项
- 72h连续运行无故障
- 并离网切换3次, 每次成功, 切换时间<20ms
- 充放电循环5次, SOC从20%→90%→20%
- 告警联动: 消防触发→PCS停机→断路器跳闸

### R3: 遥测遥信核对表
- 每测点: 标准值/实测值/误差/结论
- 每信号: 触发方式/预期状态/实测状态/结论

### R4: 充放电测试记录
- 记录项: 时间/SOC/功率/电压/电流/温度/告警
- 采样间隔: 1s
- 持续: 完整充放循环

## 验证Checklist
- [ ] FAT项覆盖所有设备
- [ ] SAT项含72h连续运行
- [ ] 核对表覆盖所有测点
- [ ] 充放电记录模板完整
