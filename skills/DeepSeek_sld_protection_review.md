
## 四、电缆选型三原则计算

### 4.1 原则一:温升校验(按持续允许电流)

**核心公式**:
```
I_z ≥ I_max
```

**最大持续工作电流计算**:
```
交流三相回路: I_max = P_max / (√3 × U_n × cosφ × η)
直流回路:     I_max = P_max / (U_n × η)
```

**计算示例(PCS交流侧, 500kW, 10kV)**:

```
I_max = 500 / (1.732 × 10 × 0.95 × 0.98) ≈ 31.0A

修正系数:
  K_1 = 0.91 (40°C, XLPE绝缘)
  K_2 = 0.85 (6根电缆同层敷设)
  K = 0.91 × 0.85 = 0.774

修正后所需载流量: I_z_needed = 31.0 / 0.774 ≈ 40.1A

查表: 3×25mm² → I_z = 105A → 修正后 105×0.774=81.3A ≥ 40.1A ✓
推荐: 3×25mm² YJV22-8.7/15kV
```

### 4.2 原则二:电压降校验(ΔU ≤ 5%)

**核心公式(交流三相)**:
```
ΔU = √3 × I × L × (r × cosφ + x × sinφ) / U_n
ΔU% = ΔU / U_n × 100%
```

**低压侧特殊要求**:
- 储能电池簇至PCS DC侧: ΔU ≤ 1%(电压低, 电流大)
- 低压动力回路(400V): ΔU ≤ 5%, 长距离(>100m)逐段校验

### 4.3 原则三:经济电流密度选截面积

**核心公式**:
```
S_e = I_max / J
```

**经济电流密度表(铜导体)**:

| 年最大负荷利用小时数 T(h) | 经济电流密度 J(A/mm²) |
|------------------------|---------------------|
| <3000(峰谷调节) | 2.50 |
| 3000~5000(工商业储能) | 2.00 |
| 5000~7000(削峰填谷) | 1.50 |
| >7000(长期调频/备用) | 1.00 |

## 六、接地系统设计

### 6.1 接地电阻要求

| 接地类型 | 接地电阻要求 | 适用条件 |
|---------|------------|---------|
| 防雷接地 | ≤ 10Ω | 一般地区 |
| 防雷接地 | ≤ 30Ω | 高土壤电阻率地区(ρ>500Ω·m) |
| 工作接地 | ≤ 4Ω | 弱电系统共用接地 |
| 保护接地 | ≤ 10Ω | 低压配电系统TN |
| 联合接地 | ≤ 1Ω | 特殊要求(通信机房) |

### 6.2 接地电阻计算公式

```
单支垂直接地极: R ≈ ρ/(2πL) × ln(4L/d)
水平接地网:     R ≈ ρ/(2πL) × ln(L²/(h·d))
并联多支:       R_total = R/(n × η)
```

**计算示例**(ρ=100Ω·m, 角钢L50×50×5×2500mm, n=5根):

```
R_single = 100/(2π×2.5) × ln(4×2.5/0.05) = 33.7Ω
R_total = 33.7/(5×0.75) = 8.99Ω ≤ 10Ω ✓
```

### 6.3 设备等电位联结要求

| 设备类型 | 联结线规格 |
|---------|----------|
| PCS/逆变器外壳 | ≥ 16mm² 黄绿双色PE线 |
| 电池舱/电池簇金属外壳 | ≥ 16mm² |
| 变压器外壳 | ≥ 25mm² |
| 低压配电柜/开关柜 | ≥ 16mm² |
| 桥架/电缆支架 | ≥ 10mm²(每隔5m) |

## 七、储能系统短路电流计算

### 7.1 电网侧最大/最小短路电流

**最大短路电流(用于校验保护灵敏度)**:
$$I_{k.max} = \frac{U_n}{\sqrt{3} \times Z_{eq.min}}$$

**最小短路电流(用于校验保护选择性)**:
$$I_{k.min} = \frac{C_{min} \times U_n}{\sqrt{3} \times Z_{eq.max}}$$

| 故障类型 | $C_{max}$ | $C_{min}$ |
|---------|----------|----------|
| 三相短路 | 1.0 | 0.95 |
| 两相短路 | 0.87 | 0.95×0.87 |
| 单相接地 | 1.0 | 0.95 |

**计算示例(10kV, S_k.min=250MVA)**:

```
Z_grid.min = 10²/250 = 0.4Ω
I_k.max = 10/(√3×0.4) = 14.43kA
PCC点短路电流范围: 10.06~14.43kA
```

### 7.2 GFM/GFL PCS短路贡献分析

| 参数 | GFM模式 | GFL模式 |
|------|---------|---------|
| 稳态短路电流 | 3~5倍In | 1.1~1.2倍In |
| 保护动作时间 | 需快速切除 | 可适当延长 |
| 反向时限流 | 必须考虑 | 可忽略 |
| LVRT支撑能力 | 强 | 弱 |
| 对电网稳定性 | 好 | 一般 |

**保护整定建议**:
- GFM模式: 出线保护需考虑储能反向短路电流, 时间定值+0.2s裕度
- GFL模式: 直接按常规电网整定, 考虑PCS低电压脱网特性

### 7.3 电池侧短路电流

**电池内阻模型**:
$$I_{bat.sc} = \frac{U_{bat}}{R_{bat}}$$

**计算示例(1簇电池, 400串, 280Ah)**:
- 簇开路电压: U_oc = 3.2V×400 = 1280V
- 簇内阻: R_bat = 0.001Ω×400/10 = 0.04Ω
- 簇短路电流: I_bat.sc = 1280/0.04 = 32000A(32kA)

| 短路电流等级 | 耐受时间 | 风险等级 |
|------------|---------|---------|
| >5倍In | <1s | 极高 |
| 3~5倍In | 1~10s | 高 |
| 1.5~3倍In | 10~60s | 中 |

### 7.4 多PCS并联短路电流叠加

**矢量叠加法(推荐)**:
$$I_{total} = \sqrt{I_1^2 + I_2^2 + 2 \times I_1 \times I_2 \times \cos(\Delta\theta)}$$

**简化算术法(保守)**:
$$I_{total} = K_c \times \sum I_{i.sc}$$
K_c = 0.85~0.95

**计算示例(5×500kW PCS, GFM模式)**:
- 单台额定电流: I_rated=721A(400V侧)
- 单台I_sc.GFM=4.0×721=2884A
- 总短路电流: I_total=0.9×5×2884=25974A≈26.0kA
- **PCS主导**, 短路电流约为额定电流的18倍

## 八、变压器保护整定

### 8.1 变压器差动保护

**比率制动原理动作方程**:
$$|I_d| > I_{op.min} \quad (|I_r| ≤ I_{r.set})$$
$$|I_d| > K_{slope} \times (|I_r| - I_{r.set}) + I_{op.min} \quad (|I_r| > I_{r.set})$$

**计算示例(6.3MVA, 10/0.4kV, u_k=6.5%)**:

| 参数 | 数值 |
|------|------|
| 高压侧额定电流 | 363.4A |
| 低压侧额定电流 | 9093.4A |
| 高压侧CT变比 | 400/5 |

| 保护段 | 作用 | 动作时间 | 整定原则 |

## 九、出线保护配合

### 9.1 三段式电流保护整定

| 保护段 | 保护范围 | 动作时间 | 主要作用 |
|--------|----------|----------|----------|
| I段 | 本线路全长80~85% | 瞬时(0s) | 快速切除近端故障 |
| II段 | 本线路全长100% | t_II | 切除本线路末端故障 |
| III段 | 本线路及相邻线路 | t_III | 后备保护 |

**I段整定**(躲过末端最大短路电流):
$$I_{op.I} = K_{rel} \times I_{k.max.line}$$

**II段整定**(与下级配合):
$$I_{op.II} = K_{rel} \times I_{k.min.peer}$$

**III段整定**(躲过最大负荷):
$$I_{op.III} = \frac{K_{rel} \times I_{load.max}}{K_{re}}$$

### 9.2 反向潮流渗透配合策略

当储能反向送电时:
$$I_{op} = K_{rel} \times I_{load.max.rev}$$
$$t_{adj} = t_{base} + \Delta t_{ESS}$$

**逆功率保护配置**:
$$P_{op.rev} = 0.02 \times S_{rated}$$

### 13.1 CT配置原则

K_s = 1.2~1.4

**准确级选择**:

| 保护类型 | 推荐准确级 | 适用场景 |
|---------|-----------|----------|
| 差动保护 | 5P20/10P20 | 主保护 |
| 过流保护 | 5P20 | 后备保护 |
| 测量/计量 | 0.5/0.2S | 监控计量 |

**CT二次负载校验**:
$$S_{burden} = I_{sec}^2 \times (R_{CT} + R_{lead} + R_{relay})$$
要求: S_burden < S_CT.rated

## 十四、常见设计缺陷与纠正

### 14.1 安全类致命缺陷(最高优先级)

| 缺陷 | 后果 | 正确做法 |
|------|------|---------|
| **无消防联动硬接线回路** | 通信故障火灾扩大, 人员伤亡 | FSS→安全继电器→硬线分断 |
| **直流侧无熔断器保护** | 电池短路无法切断, 热失控 | DC正极必须配熔断器 |
| **无ESD紧急停机回路** | 紧急情况无法一键断电 | 独立硬接线ESD回路 |
| **防孤岛未配置** | 线路检修人员触电 | PCC必须配独立防孤岛保护 |
| **DC断路器用AC型号** | 直流电弧无法熄灭 | 必须用DC专用断路器 |
| **保护定值无级差配合** | 越级跳闸/全站停电 | 上下级时间差≥200ms |

### 14.2 符号类缺陷

| 缺陷 | 正确做法 |
|------|---------|
| 断路器画成隔离开关符号 | 断路器方框必须有"/"斜线 |
| 变压器无接线组别标注 | 必须标注Dyn11/Yyn0 |
| 避雷器不画接地 | 下端必须连接接地符号 |
| CT不画二次引出线 | 必须圆外引虚线表示二次回路 |

### 14.3 标注类缺陷

| 缺陷 | 正确做法 |
|------|---------|
| 断路器无编号 | 唯一编号QF1,QF2... |
| CT无变比 | 标注如500/5A, 5P20 |
| 电缆无截面积 | 标注型号+芯数×截面积 |
| 变压器无阻抗百分比 | 标注Uk%=6 |
| 直流侧仅标额定值 | 标注Vmin~Vmax范围 |
