# 一次图审图 Skill v1.0

> 设计人：卢继雄 | 基于5年微网项目实战训练

## 概述
自动检查一次接线图中关键设备是否缺失：PCC、CT/PT、断路器、接地、防孤岛、防逆流、保护分区。

## 输入
- 一次接线图 (SVG/JSON描述)
- 设备清单

## 输出
- 审图结果: PASS/WARN/FAIL
- 缺失项列表
- 修改建议

## 规则

### R1: PCC并网开关柜必须存在
- 检查: 并网运行或双模时,必须有PCC断路器(QF01)
- 缺失级别: FAIL

### R2: CT/PT配置
- 检查: PCC处需配置计量CT(0.2S级)+保护CT(0.5级)
- 检查: 10kV侧需PT(10kV/100V)
- 缺失级别: WARN

### R3: 断路器配置
- 检查: 每条馈线需有断路器+综保
- 检查: 储能PCS进线需专用断路器
- 缺失级别: FAIL

### R4: 接地系统
- 检查: TN-S系统(PE+N分开)或TN-C-S
- 检查: 防雷接地+保护接地+工作接地
- 缺失级别: WARN

### R5: 防孤岛保护
- 检查: 并网系统必须有防孤岛保护(AFD+频偏+压偏)
- 缺失级别: FAIL

### R6: 防逆流保护
- 检查: 自发自用余电不上网模式需防逆流(策略+硬件)
- 缺失级别: FAIL

### R7: 保护分区
- 检查: 变压器差动保护
- 检查: 储能PCS过流过压欠压
- 检查: BMS三级保护(1级降功率→2级停机→3级急停)
- 缺失级别: FAIL

### R8: 消防联动
- 检查: FSS→PCS/断路器硬接线联跳
- 检查: 不能仅靠通信,必须有硬接线
- 缺失级别: FAIL

## 验证Checklist
- [ ] PCC存在且含断路器
- [ ] CT/PT标注完整
- [ ] 每条馈线有断路器
- [ ] 接地系统标注
- [ ] 防孤岛保护标注
- [ ] 防逆流措施标注
- [ ] 保护分区完整
- [ ] 消防硬接线联跳
