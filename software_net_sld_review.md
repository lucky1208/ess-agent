## 一、审图总则与标准依据

### 1.1 审图分级

| 等级 | 名称 | 深度要求 | 适用阶段 | 输出 |
|------|------|---------|---------|------|
| L1 | 概念审查 | 拓扑正确性、主要设备选型 | 可研/方案 | 意见备忘录 |
| L2 | 初步设计审查 | 间隔结构、保护配置、参数合理 | 初步设计 | 审查意见函 |
| L3 | 施工图审查 | 全部参数、电缆清册、端子排 | 施工图 | 正式审图报告 |
| L4 | 竣工图审查 | 与实际设备一致 | 竣工 | 竣工审图报告 |

### 1.2 适用标准速查

| 标准 | 审图引用内容 |
|------|------------|
| **GB/T 4728.1~13** | 图形符号正确性检查 |
| **GB 51048** | 电化学储能电站设计规范的SLD要求 |
| GB/T 36547 | 储能接入电网的PCC参数要求 |
| GB/T 36558 | 储能系统通用技术条件的参数范围 |
| GB 50052 | 供配电系统设计规范——负荷分级 |
| GB 50054 | 低压配电设计规范——0.4kV侧保护要求 |
| GB/T 50062 | 继电保护和自动装置设计规范——保护配置 |
| GB 50065 | 交流电气装置的接地设计规范 |
| **DL/T 2246** | 储能并网运行控制——防孤岛/低穿要求 |
| **IEC 60617** | 国际项目符号验证 |
| **NFPA 855** | 储能安全距离/消防标注(出海项目) |
| **UL 9540/9540A** | 储能系统安全标注(北美项目) |

### 1.3 审图流程

```
Step 1: 整体结构检查
  ├── 图框/标题栏/图例是否完整
  ├── 图名/图号/版本/日期是否标注
  └── 电压等级是否全图标注

Step 2: 逐间隔逐Bay检查
  ├── 10kV进线间隔 → 母线 → 出线间隔 → 变压器间隔
  ├── 0.4kV/0.69kV侧间隔
  └── 直流储能间隔 → 电池簇内部

Step 3: 保护配置检查
  ├── 保护分区是否合理
  ├── 保护定值配合是否可行
  └── 防孤岛/防逆流/消防联动是否完整

Step 4: 参数标注完整性检查
  ├── 每条线路电缆型号/截面积
  ├── 每个设备额定参数
  └── CT/PT变比

Step 5: 标准符号验证
  └── 所有符号是否符合GB/T 4728

Step 6: 出具审图报告
  ├── 问题清单(严重/一般/建议)
  ├── 通过/不通过判断
  └── 整改建议
```

## 二、SLD整体结构完整性检查

### 2.1 图面基本信息检查 (14项)

```
□ 1.  图框完整(图纸幅面A3/A2/A1明确)
□ 2.  标题栏包含: 项目名称、图名、图号、版本、日期
□ 3.  设计单位信息标注
□ 4.  图例(Legend)完整，覆盖所有使用符号
□ 5.  绘图比例标注(如适用)
□ 6.  电压等级标注贯穿全图(至少1处/电压等级)
□ 7.  电流方向标注(充电/放电方向标记)
□ 8.  高压/低压/直流分区边界清晰(虚线分界)
□ 9.  母线颜色/线宽正确(母线≥3倍普通线宽)
□ 10. 能量流和信息流线型区分(能量=实线, 信息=虚线)
□ 11. 导线交叉点正确处理(连接=实心圆, 不连接=跨弧)
□ 12. 元件编号唯一且符合GB/T 6988命名规范
□ 13. 文字标注字体大小一致(设备13px/参数11px/备注10px)
□ 14. 是否有"施工图"或"方案图"声明
```

### 2.2 系统架构完整性检查

```
□ 15. 从PCC到电池簇的完整路径是否画出
      10kV PCC → 开关柜 → 变压器 → 低压侧 → PCS → DC侧 → 高压箱 → 电池簇
□ 16. 是否有缺失支路(站用电、SVG、充电桩等)
□ 17. 并网点(PCC)是否明确标注
□ 18. 计量点(关口表/并网表)是否标注
□ 19. 保护区域划分是否标注(进线区/母线区/变压器区/馈线区/PCS区/电池区)
□ 20. 分区之间保护边界标注
```

## 三、按电气间隔(Bay)逐项检查

### 3.1 10kV进线间隔 (Incoming Feeder Bay)

```
进线间隔元件顺序(电源侧→母线侧):
  ┌──────────────────────────────────────────────────┐
  │ 电缆终端 → QS1(隔离开关) → F1(避雷器)            │
  │          → TV1(PT/计量) → TA1(CT/保护+测量)      │
  │          → QF1(断路器,进线总) → 10kV母线          │
  └──────────────────────────────────────────────────┘

检查项:
□ 21. 进线隔离开关QS1是否存在? 是否有接地刀?
□ 22. 避雷器F1是否配置? 位于进线CT之前(保护CT)?
□ 23. 电压互感器TV1是否配置? 用于计量还是保护? 变比标注?
□ 24. 电流互感器TA1是否配置? 保护/测量绕组分别标注?
□ 25. TA1的变比和准确级标注? (如: 500/5A, 5P20/0.5S)
□ 26. 断路器QF1是否配置? 额定参数标注?
□ 27. QF1是否有保护装置? 保护功能标注?
□ 28. 进线电缆型号/截面积标注?
□ 29. PT一次侧是否有熔断器保护?
□ 30. 是否有带电显示器(施工图要求)?

常见缺失:
  ❌ 漏装避雷器 → 雷击过电压损坏设备
  ❌ PT无熔断器 → PT故障时无法隔离,可能引发短路
  ❌ CT无准确级 → 无法判断是测量级还是保护级
```

### 3.2 10kV母线间隔 (Busbar Bay)

```
10kV母线:
  ┌──────────────────────────────────────────────────┐
  │ ==== 10kV Busbar (标注: TMY-xxx) ====             │
  │   ├── PT柜: TV2(母线PT, 10/0.1kV, 0.5级)         │
  │   └── 母联/分段(如有双母线)                        │
  └──────────────────────────────────────────────────┘

检查项:
□ 31. 母线材质/规格标注? (如: TMY-100×10 铜排)
□ 32. 母线额定电流标注?
□ 33. 母线PT(TV2)是否配置? 变比标注?
□ 34. 母线避雷器是否配置?
□ 35. 母线接地闸刀是否配置? (施工/检修用)
□ 36. 母联开关(如有双母线)是否配置?
```

### 3.3 出线/变压器馈线间隔 (Outgoing/Transformer Feeder Bay)

```
出线到变压器T1:
  ┌──────────────────────────────────────────────────┐
  │ 10kV母线 → QF2(断路器) → TA2(CT) → 电缆 → T1     │
  └──────────────────────────────────────────────────┘

检查项:
□ 37. 出线断路器QF2是否配置? 额定参数?
□ 38. TA2变比/准确级标注? (需与变压器容量匹配)
□ 39. 出线电缆型号/截面积?
□ 40. 出线避雷器是否配置?
□ 41. 变压器T1完整参数标注?
      容量(kVA) + 电压比 + 接线组别(Dyn11/Yyn0) + 阻抗% + 冷却方式
□ 42. 变压器中性点接地方式标注? (直接接地/经小电阻/消弧线圈)
□ 43. 变压器保护配置是否标注? (差动87T/过流50/51/零序50N/51N)

变压器参数验证公式:
  I_rated_primary = S(kVA) / (√3 × U1(kV))
  例: 1250kVA, 10kV → I1 = 1250/(1.732×10) = 72.2A
  CT一次电流 ≥ 1.3 × I_rated = 94A → 选用100/5A
```

### 3.4 0.4kV/0.69kV低压侧间隔

```
变压器二次侧:
  ┌──────────────────────────────────────────────────┐
  │ T1 二次侧(0.4kV) → QF-LV1(低压总断路器)            │
  │                   → TA-LV(低压CT)                  │
  │                   → 0.4kV母线                      │
  └──────────────────────────────────────────────────┘

0.4kV母线分支:
  ├── PCS支路1: QF-AC1 → CT1 → PCS1
  ├── PCS支路2: QF-AC2 → CT2 → PCS2
  ├── SVG支路:  QF-SVG → SVG
  ├── 站用电:   QF-Aux → 站用变压器 → 220V/380V
  └── APF支路:  QF-APF → APF(有源滤波, 如需)

检查项:
□ 44. 低压总断路器QF-LV1是否配置? 额定电流与变压器容量匹配?
□ 45. 低压总CT是否配置? 变比与母线电流匹配?
□ 46. 低压PT(如需)是否配置?
□ 47. 各出线回路断路器齐全? QF-AC1, QF-AC2...
□ 48. 低压侧避雷器/浪涌保护器(SPD)配置?
□ 49. 低压母线规格标注?
□ 50. 站用电回路是否画出? (控制电源、照明、空调等)
□ 51. 无功补偿回路是否画出? (按GB 50052要求, 功率因数需≥0.9)
□ 52. 有源滤波回路是否画出? (如PCS谐波含量超标)
□ 53. 低压侧电缆型号/截面积标注?
□ 54. 是否有发电机接口(柴发)标注? (如有)
```

### 3.5 PCS/储能直流间隔

```
PCS支路(交流侧→直流侧):
  ┌──────────────────────────────────────────────────┐
  │ 0.4kV AC → QF-AC → CT → PCS → QF-DC → FU-DC     │
  │                                   → KM-DC+       │
  │                                   → KM-DC-       │
  │                                   → 高压箱BCP    │
  └──────────────────────────────────────────────────┘

检查项:
□ 55. PCS交流侧断路器QF-AC是否配置? 额定参数?
□ 56. PCS额定功率/电压范围标注?
□ 57. PCS拓扑类型标注(single-stage/two-stage/cascaded-H)?
□ 58. 直流侧断路器QF-DC是否配置? (通常正极1个,负极可选)
□ 59. 直流侧熔断器FU-DC是否配置? (正极必须有,负极建议有)
□ 60. 直流接触器KM-DC+/KM-DC-是否配置?
□ 61. 预充回路: KM-pre + R-pre 是否画出?
□ 62. 直流侧电缆型号/截面积标注?

直流侧断路器验证:
  I_dc_rated = P_pcs(W) / V_dc_min(V)
  例: PCS=630kW, Vdc_min=600V → I_dc=1050A
  选用DC断路器 ≥ 1250A
```

### 3.6 高压箱(BCP)内部检查

```
高压箱内部:
  ┌──────────────────────────────────────────────┐
  │ 电池簇正极                                    │
  │   ├── FU+(正极熔断器) xxxA                     │
  │   ├── KM+(主正接触器) xxxV DC, xxxA           │
  │   ├── Hall电流传感器 (xxxA)                   │
  │   └── (预充回路: KM-pre + R-pre 并接KM+两端)   │
  │                                                │
  │ 电池簇负极                                    │
  │   ├── Shunt分流器或Hall (电流检测)             │
  │   ├── KM-(主负接触器)                          │
  │   └── FU-(负极熔断器,可选)                     │
  │                                                │
  │ 绝缘监测模块 (不平衡电桥)                      │
  │ BCMU控制板                                    │
  └──────────────────────────────────────────────┘

检查项:
□ 63. 高压箱BCP内部元件是否完整画出?
□ 64. 主正接触器KM+是否标注? 额定参数?
□ 65. 主负接触器KM-是否标注?
□ 66. 预充接触器KM-pre和预充电阻R-pre是否标注?
□ 67. 熔断器FU+/FU-是否标注? 额定电流?
□ 68. 电流传感器(Hall/Shunt)是否标注? 量程?
□ 69. 绝缘监测是否标注?
□ 70. BCMU控制板是否标注?
```

## 四、断路器参数验证规则

### 4.1 断路器额定参数验证表

| 参数 | 符号 | 要求 | 验证公式 |
|------|------|------|---------|
| 额定电压 | Un | ≥ 系统最高电压 | Un_breaker ≥ Un_system × 1.1 |
| 额定电流 | In | ≥ 1.3 × 回路最大工作电流 | In ≥ 1.3 × P_max/(√3×U) |
| 短路分断能力 | Icu/Ics | ≥ 安装点预期短路电流 | Icu ≥ Ik"(三相) × 1.1 |
| 短路接通能力 | Icm | ≥ 2.5 × Icu (AC) | Icm ≥ 2.5 × Icu |
| 热稳定电流 | Ith | 短时耐受 ≥ 短路电流持续时间 | Ith²×t ≥ Ik²×t_fault |
| 操作循环 | O-CO-t-CO | 满足标准操作序列 | 按GB 1984 |

### 4.2 断路器选型合理性检查

```
AC断路器选型检查:
□ 额定电压 = 系统标称电压 × 1.1 (12kV用于10kV系统)
□ 额定电流 ≥ 实际可能最大工作电流 × 1.3  ← 至少30%裕量
□ 分断能力 ≥ 安装点预期最大短路电流
□ 操作机构类型标注(弹簧/永磁/液压)
□ 是否有欠压脱扣器/分励脱扣器标注(二次图配合)
□ 是否有合闸闭锁/防跳功能标注

DC断路器选型特殊检查:
□ 直流电压额定值 ≥ 电池系统最高电压 × 1.1
□ 直流分断能力 ≥ 电池系统预期最大短路电流(电池短路电流大!)
□ 极性标注(+/-)
□ 是否有直流电弧熄灭措施(磁吹/真空)
□ 是否标注DC专用(不能混用AC断路器)
```

### 4.3 断路器定值合理性快速检查

| 保护类型 | 整定范围 | 典型倍数 | 验证 |
|---------|---------|---------|------|
| 长延时过载(Ir) | 0.4~1.0×In | 0.9×In | ≤电缆载流量 |
| 短延时短路(Isd) | 1.5~10×Ir | 3~5×Ir | ≤上级Isd×0.8 |
| 瞬时速断(Ii) | 2~15×In | 8~12×In | 躲过最大启动电流 |
| 接地故障(Ig) | 0.2~1.0×In | 0.3×In | ≤上级Ig×0.8 |

## 五、CT/PT配置与参数验证

### 5.1 电流互感器(CT)验证

```
CT二次绕组配置要求(每个CT通常有多个二次绕组):

绕组用途         | 准确级要求     | 额定二次电流 | 容量(VA)
────────────────┼───────────────┼─────────────┼──────────
计量(Metering)   | 0.2S 或 0.5S  | 5A 或 1A     | ≥10VA
测量(Measurement)| 0.5 或 1.0    | 5A 或 1A     | ≥15VA
保护(Protection) | 5P20 或 10P20 | 5A 或 1A     | ≥20VA
差动(Differential)| 5P20 或 PX   | 5A 或 1A     | ≥20VA

CT一次电流选择验证:
  I_ct_primary ≥ I_rated_max × 1.3
  I_ct_primary ≤ I_rated_max × 2.0  (计量CT不能太大,否则小电流精度差)

CT变比检验公式:
  Ratio = I_primary / I_secondary
  例: 400/5A → Ratio=80

  CT二次侧负载验证:
  S_load(VA) = I_secondary² × (R_ct + R_wire + R_relay)
  要求: S_load ≤ S_rated_ct
```

### 5.2 电压互感器(PT)验证

```
PT配置要求:

PT类型         | 准确级  | 变比示例        | 用途
──────────────┼────────┼───────────────┼──────────
母线PT         | 0.5/3P | 10/0.1kV      | 测量+保护
计量PT         | 0.2    | 10/0.1kV      | 关口计量
零序PT(开口三角)| 3P     | (10/√3)/(0.1/3)| 零序电压检测

PT验证项:
□ 变比是否标注? (如10/0.1kV)
□ 准确级是否标注?
□ 额定容量是否标注? (VA)
□ 一次侧是否有熔断器保护?
□ 二次侧是否有空开保护(防止短路)?
□ 计量PT与保护PT是否独立?  ← 不能共用!
```

## 六、保护分区与配合校验

### 6.2 保护时间配合校验

```
时间级差原则:
  Zone 1保护动作时间 < Zone 2 < Zone 3 < Zone 4

  最内层(BMS/PCS):  t ≤ 100ms (电子保护, 瞬间)
  馈线级(Zone 2):   t = 300ms (定时限)
  变压器级(Zone 3): t = 500ms (定时限)
  PCC级(Zone 4):    t = 800ms~1s (定时限)

  Δt_min = t_breaker_open + t_relay_reset + t_margin
          = 60ms + 50ms + 90ms = 200ms

电流级差原则:
  上级动作电流 < 下级动作电流 × 1.2 (确保选择性)
  上级动作电流 > 下级负荷电流 × 1.5 (确保可靠性)

检查方法:
  对每个相邻保护层级:
  □ 上级动作时间 - 下级动作时间 ≥ 200ms?
  □ 上级动作电流 ≥ 下级动作电流 × 1.2?
  □ 上级返回电流 ≤ 下级动作电流? (防止误动后无法恢复)
```

### 6.3 保护死区检查

```
常见保护死区:
  ❌ CT与断路器之间 → 需配置断路器失灵保护(BFP)
  ❌ 母线PT与母线之间 → 需配置备自投/母线残压检测
  ❌ 变压器差动保护CT之间 → 需配置非电量保护(瓦斯/温度)
  ❌ 直流侧接触器与PCS之间 → 需PCS自身保护+熔断器

检查项:
□ 83. 断路器失灵保护(BFP)是否配置?
□ 84. 主保护与后备保护是否有重叠区?
□ 85. 是否存在仅有单一保护的区域?

```

### 8.1 防逆流保护配置条件

```
是否需要防逆流保护?

  │   PCC处多功能电表 → EMS → PCS降功率控制        │
  │   (注意: 这种方案响应较慢, 需可靠的通信)        │
  │                                                │
  │ 方案C: 逆功率保护 + EMS联合                     │
  │   逆功率继电器做最后保护(>设定值跳闸)           │
```

### 8.3 防逆流与防孤岛的关系区分

```
防孤岛(Anti-Islanding): 电网停电后,防止DG继续向局部电网供电
  - 检测对象: 电压/频率异常
  - 动作对象: PCC断路器
  - 目的: 安全(防触电)

防逆流(Anti-Reverse-Power): 正常并网时,防止DG向电网反送电
  - 检测对象: 功率方向/大小
  - 动作对象: PCS降功率 or PCC断路器
  - 目的: 合同/政策(不反送)

⚠️ 两者不能互相替代! 各有独立的功能要求。
```

## 十五、常见设计缺陷与纠正

### 15.1 符号类缺陷

| 缺陷 | 影响 | 正确做法 |
|------|------|---------|
| 断路器画成隔离开关符号(无斜线) | 施工无法区分, 选型错误 | 断路器方框必须有"/"斜线 |
| 隔离开关画了斜线成断路器 | 误判有分断能力, 安全事故 | 隔离开关方框+刀片, 无斜线 |
| 变压器无接线组别标注 | 施工单位无法确定接线方式 | 必须标注 Dyn11/Yyn0 |
| 避雷器不画接地 | 保护功能表达不完整 | 避雷器下端必须连接接地符号 |
| CT不画二次引出线 | CT连接不明确 | 必须圆外引虚线表示二次回路 |
| 熔断器用矩形代替(不画S线) | 与电阻/断路器混淆 | 矩形内必须有S形熔丝 |

### 15.2 线路类缺陷

| 缺陷 | 影响 | 正确做法 |
|------|------|---------|
| 母线画成普通线宽 | 无法区分母线与支线 | 母线宽度≥3倍支线宽(≥6px) |
| 交叉点不处理(不画点也不画弧) | 无法判断是否连接 | 必须画实心圆或跨弧 |
| 能量线和信号线相同线型 | 混淆主回路和二次回路 | 能量=实线, 信号/通信=虚线 |
| 直流电缆型用交流型号 | 直流电缆有特殊要求 | DC侧用YJVR(软)等专用型号 |
| 光纤和铜缆同一符号 | 安装方式不同 | 光纤用分路符号+标注单/多模 |

### 15.3 标注类缺陷

| 缺陷 | 影响 | 正确做法 |
|------|------|---------|
| 断路器无编号 | 无法对应设备清单 | 每个断路器唯一编号 QF1,QF2... |
| CT无变比 | 施工单位无法选型 | 必须标注如 500/5A, 5P20 |
| 电缆无截面积 | 无法采购/施工 | 必须标注 型号+芯数×截面积 |
| 变压器无阻抗百分比 | 短路计算缺参数 | 必须标注 Uk%=6 等 |
| 直流侧电压仅标额定值 | 选型忽略电压范围 | 标注 Vmin~Vmax 范围 |

### 15.4 安全类致命缺陷(最高优先级)

| 缺陷 | 后果 | 正确做法 |
|------|------|---------|
| **无消防联动硬接线回路** | 通信故障火灾扩大, 人员伤亡 | FSS→安全继电器→硬线分断所有断路器+PCS ESD |
| **直流侧无熔断器保护** | 电池短路无法切断, 热失控起🔥 | DC正极必须配熔断器, 负极建议配 |
| **无ESD紧急停机回路** | 紧急情况无法一键断电 | 独立硬接线ESD回路, 红色蘑菇头按钮 |
| **防孤岛未配置** | 线路检修人员触电⚡ | PCC必须配独立防孤岛保护装置 |
| **直流绝缘监测缺失** | 漏电/接地故障无法检测 | BMS必须监测正对地和负对地绝缘电阻 |
| **PCC断路器无分断能力** | 短路无法断开 | 验证断路器短路分断能力Icu≥Isc |
| **DC断路器用AC型号** | 直流电弧无法熄灭 | 必须用DC专用断路器/接触器 |
| **保护定值无级差配合** | 越级跳闸/全站停电 | 上下级时间差≥200ms |
